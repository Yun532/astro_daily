周末经典专题课

周末 arXiv 通常不更新；本期改为一讲讲透的高能天体物理经典专题课，重点补公式推导、经典拟合和关键图表读法。

1. 高能天体物理零基础课（二）：同步辐射、逆康普顿、热辐射和吸收

  • 课程系列：高能天体物理零基础桥梁课（第 2/5 讲）
  • 本讲边界：本讲讲三条主线都会用到的辐射机制：同步辐射、逆康普顿、黑体/光球辐射、冷却和吸收。不讲具体 GRB prompt、TDE disk 或 PWN 动力学细节。
  • 上下文承接：上一讲建立了观测量语言；本讲把这些观测量和电子、磁场、光子场、温度等物理量连接起来。
  • 主题：radiation mechanisms for high-energy transients and non-thermal nebulae
  • 经典工作：从单个电子辐射、幂律电子分布和磁场/光子场出发，建立 synchrotron、inverse Compton、thermal emission、absorption 的共同图像。
  • 为什么经典：GRB afterglow、PWN、pulsar halo 和许多 TDE radio/X-ray 论文都在反复使用同一组辐射机制。先把这组语言学会，后续读模型会轻很多。

经典工作详细讲解

第一段：本讲承接第 1 讲的观测量语言。上一讲我们学习了望远镜和论文如何报告 \(F_{\nu}\)、\(\nu F_{\nu}\)、luminosity、light curve、photon index、redshift 与距离等量，也强调了 \(dF/d\ln\nu=\nu F_{\nu}\)，因此在 \(\nu F_{\nu}\) 图上峰值代表每个对数频率区间能量输出最集中的位置。本讲要向前推进一步：不再只问“观测到什么光子”，而是问“什么电子、磁场、光子场、温度和吸收介质产生了这些光子”。所以当论文写出 \(F_{\nu}\propto \nu^{-\beta}\) 时，我们要能判断这个斜率可能对应电子谱、冷却过程、吸收过程，还是多个区域叠加后的有效结果。

第二段：本讲的科学问题可以概括为一条翻译链：从光子谱形反推粒子和环境。若一个暂现源或非热星云在 radio、optical、X-ray 之间呈现宽带幂律，第一候选通常是光学薄同步辐射；若同一电子群还处在强种子光子场中，它们可以通过逆康普顿散射产生 GeV 或 TeV 辐射；若谱中出现窄而近似 Planck 形状的峰，则温度 \(T\) 和有效面积可能是主要诊断量；若低频 radio 被压低或高能 gamma-ray 出现 cutoff，则必须检查光学深度 \(\tau_{\nu}\) 和逃逸概率。这里的重点不是给每种源类贴标签，而是建立一套跨源类通用的辐射语法。

第三段：历史动机来自几个经典事实。Crab Nebula、超新星遗迹、AGN jet、GRB afterglow、PWN、pulsar halo 和许多 TDE radio 源都显示出非热谱，而不是单一黑体谱。Crab PWN 说明相对论粒子、磁场、同步辐射冷却和空间形态可以组成一个统一实验室；GRB afterglow 把 \(\nu_{a}\)、\(\nu_{m}\)、\(\nu_{c}\) 和峰通量变成暂现源的标准谱语言；pulsar halo 和 PWN 的 TeV 辐射则提醒我们，同一批电子可能既产生同步辐射，也通过逆康普顿辐射照亮高能天空。TDE radio 研究中常用同步自吸收峰反演半径、磁场和能量，也是在使用同一组基础关系。

第四段：方法上，本讲从单个相对论电子开始，而不是从某个具体源的动力学开始。电子在磁场 \(B\) 中弯曲运动会产生同步辐射，特征频率随 \(\gamma^{2}B\) 增长；电子在种子光子场中会把低能光子提升到更高能量，在 Thomson 极限下有 \(h\nu_{\rm IC}\simeq (4/3)\gamma^{2}h\nu_{0}\)；热辐射用 \(B_{\nu}(T)\) 和 \(L_{\rm BB}=4\pi R^{2}\sigma_{\rm SB}T^{4}\) 描述；吸收用 \(I_{\nu}=I_{\nu,0}e^{-\tau_{\nu}}\) 和 \(\tau_{\nu}=\int\alpha_{\nu}ds\) 描述。这样组织课程的好处是，GRB 外激波、PWN 终止激波、pulsar halo 扩散电子和 TDE 外流虽然动力学不同，但谱诊断的骨架高度相似。

第五段：本讲最核心的结果有三类。第一，同步辐射把观测频率和电子能量联系起来，近似可理解为给定 \(B\) 时，频率 \(\nu\) 主要选择 \(γ\sim(\nu/B)^{1/2}\) 的电子。第二，若电子分布为 \(N(\gamma)d\gamma=K\gamma^{-p}d\gamma\)，在光学薄且未冷却的同步谱段中有 \(F_{\nu}\propto\nu^{-(p-1)/2}\)，因此 \(\beta=(p-1)/2\)；若电子经历辐射冷却，持续注入模型中高能端电子谱通常增陡，光子谱可变为 \(F_{\nu}\propto\nu^{-p/2}\)。第三，逆康普顿与同步辐射的相对强度在 Thomson 区域近似由 \(Y\equiv P_{\rm IC}/P_{\rm syn}\simeq U_{\rm rad}/U_{B}\) 衡量，但当 \(4\gamma h\nu_{0}\ll m_{e}c^{2}\) 不再成立时，Klein-Nishina 抑制会改变高能谱和冷却效率。

第六段：这些公式必须和观测诊断连在一起。看到谱峰时，不能立刻称它为温度峰，而要区分它可能是同步注入峰、冷却相关峰、同步自吸收峰、热峰、逆康普顿峰或吸收造成的转折。看到宽带幂律时，也不能直接把斜率等同于粒子加速谱，而要检查该段是否光学薄、是否已冷却、是否受前景吸收或仪器响应影响。看到高能 cutoff 时，要同时列出最大加速能量、冷却、逃逸、Klein-Nishina 抑制、光子-光子吸收和背景扣除误差等候选解释。

第七段：本讲还要训练读论文图表的习惯。读宽带 SED 时，先确认横轴是频率还是光子能量，纵轴是 \(F_{\nu}\)、\(\nu F_{\nu}\) 还是 \(E^{2}dN/dE\)；读同步谱段图时，标出 \(\nu_{a}\)、\(\nu_{m}\)、\(\nu_{c}\) 和峰通量，并判断慢冷或快冷；读冷却时间图时，比较 \(t_{\rm syn}\)、\(t_{\rm IC}\)、\(t_{\rm age}\) 与 \(t_{\rm esc}\) 的交点；读残差图时，关注低频、软 X-ray、高能 gamma-ray 和仪器交界处是否有系统结构；读角图时，检查 \(B\)、电子能量、\(p\)、\(Y\)、\(\nu_{c}\) 和吸收参数之间的相关性。

第八段：局限性需要从一开始讲清楚。真实源通常不是单区、均匀磁场和单一幂律电子分布；多区叠加会抹平断裂，时间依赖注入会改变冷却谱，粒子输运会让不同能段显示不同空间尺度，吸收可以发生在源内、宿主星系、银河系前景或宇宙背景光中。拟合中也常有强简并，例如较强 \(B\) 和较少电子、较弱 \(B\) 和较多电子都可能给出类似同步通量，必须借助逆康普顿峰、冷却时间、源大小或时间演化来打破部分简并。本讲故意不展开 GRB prompt、TDE 盘结构、PWN MHD 动力学，也不深入仪器响应和显著性；这些边界保证我们专注于“物理谱”本身，下一讲再进入真实探测器计数、背景和响应折叠。

背景知识

第一段：第 1 讲解决的是观测语言问题：望远镜论文中出现的 flux、flux density、luminosity、spectrum、SED、light curve、photon index 和 redshift 到底是什么意思。本讲解决下一层问题：这些观测量由哪些基本辐射机制产生。高能天体物理的困难在于，我们几乎不能直接拿到粒子分布、磁场强度或源区大小，只能通过光子谱和光变间接推断。因此，学习同步辐射、逆康普顿、热辐射和吸收，就是学习如何把 \(F_{\nu}\)、\(\nu F_{\nu}\)、谱峰、斜率、断裂和 cutoff 翻译成物理量。

第二段：现代高能天体物理中最反复出现的背景事实，是非热幂律谱的普遍性。普通恒星的光谱可以在很大程度上用温度和黑体近似理解，但超新星遗迹、Crab Nebula、AGN jet、GRB afterglow、PWN、pulsar halo 和许多 radio transient 往往在多个数量级频率范围内呈现近似幂律。若观测到 \(F_{\nu}\propto\nu^{-\beta}\)，这通常提示源中存在非热相对论粒子；但这只是起点，不是结论。我们还要问这个斜率是否来自未冷却电子、冷却电子、吸收修正、多区叠加或有限仪器带宽。

第三段：同步辐射成为经典工具，是因为它自然连接了三个量：电子 Lorentz 因子 \(\gamma\)、磁场 \(B\) 和观测频率 \(\nu\)。Crab Nebula 是最有代表性的天体实验室之一，它在 radio、optical、X-ray 和 gamma-ray 都发亮，说明脉冲星风能量可以转化为相对论粒子和磁场。Kennel 与 Coroniti 关于 Crab PWN 的经典工作把风、终止激波、磁化和同步辐射放入统一框架。即使本讲不展开 PWN 的 MHD 细节，我们仍要借这个历史案例理解：不同能段的光子常常来自不同能量、不同冷却状态的电子。

第四段：GRB afterglow 把同步辐射谱段语言推向了暂现源研究中心。Sari、Piran 与 Narayan 的经典 afterglow 框架用几个特征频率和一个峰通量描述宽带谱，其中 \(\nu_{a}\) 表示自吸收频率，\(\nu_{m}\) 与注入电子能量有关，\(\nu_{c}\) 与冷却有关。虽然本讲不讲外激波动力学，也不推导这些频率如何随爆炸能量、外介质密度和时间演化，但会讲清这些谱断裂为什么有辐射物理意义。也就是说，本讲提供的是之后读 GRB afterglow、TDE radio 和 PWN 宽带谱时共用的底层语法。

第五段：逆康普顿的重要性来自“同一批电子可以产生两个能段”的事实。在 Thomson 区域中，电子可把种子光子能量提升约 \(\gamma^{2}\) 倍，因此 CMB、红外光、星光或同步光子都可能被提升到 X-ray、GeV 或 TeV 能段。若同步辐射告诉我们电子与 \(B\) 的组合，逆康普顿则进一步告诉我们电子与种子光子场 \(U_{\rm rad}\) 的组合。两者相对强度近似给出 \(U_{\rm rad}/U_{B}\)，这就是为什么 PWN、pulsar halo、AGN jet 和 GRB afterglow 的高能成分常常要同步与逆康普顿一起拟合。

第六段：热辐射和吸收提醒我们，高能源并不只有非热过程。若光子和物质接近平衡，谱形可能接近 Planck 函数 \(B_{\nu}(T)\)，峰的位置主要测温度 \(T\)，归一化结合距离可估计有效面积。但高能暂现源的热峰常常被相对论运动、多温度叠加、Compton 化或几何效应展宽，不能简单等同于单温黑体。另一方面，吸收会决定产生的光子是否能逃逸；同步自吸收可制造 radio 峰，自由-自由吸收可压低低频辐射，光子-光子吸收可削弱高能 gamma-ray。

第七段：本讲特别要避免三类常见误区。第一，把所有峰都称作温度，这是错误的；同步峰、逆康普顿峰、自吸收峰和热峰有不同物理含义。第二，把所有幂律都当作加速谱，这也不对；冷却、吸收、传播和仪器选择效应都能改变斜率。第三，把最佳拟合的 \(B\)、电子能量或 \(p\) 当成唯一答案很危险；这些参数常与源区大小、填充因子、距离、吸收柱密度、种子光子场和电子低能截断强相关。正确读法是同时看模型假设、残差、置信区间、后验相关和系统误差。

第八段：本讲在系列中的边界也很重要。上一讲已经建立观测量语言，本讲只讨论物理谱层面的辐射与吸收机制；下一讲才系统讨论真实探测器如何把物理光谱变成 counts，包括有效面积、能量响应、背景、显著性、上限和 Poisson 似然。换句话说，本讲先让学生知道“如果物理谱是这样，它说明了什么粒子和场”；下一讲再回答“真实仪器能不能可靠地测出这个谱形”。

基础理论 / 方法脉络

第一段：本讲的基本物理图像从一个相对论电子开始。电子带电量为 \(e\)，质量为 \(m_{e}\)，在磁场 \(B\) 中受到 Lorentz 力而弯曲运动；只要运动方向改变，就存在加速度，而加速电荷会辐射。非相对论回旋角频率为 \(\omega_{B}=eB/(m_{e}c)\)，但高能天体中的电子常有 Lorentz 因子 \(\gamma\gg1\)，辐射会被相对论 beaming 和时间压缩推到更高频率。同步辐射的典型频率可写为 \(\nu_{\rm syn}=\frac{3}{4\pi}\gamma^{2}\frac{eB}{m_{e}c}\sin\alpha_{p}\)，其中 \(\alpha_{p}\) 是 pitch angle。

第二段：同步辐射最重要的直觉是“观测频率选择电子能量”。由 \(\nu_{\rm syn}\propto\gamma^{2}B\) 可知，在给定 \(B\) 时，更高频率来自更高 \(\gamma\) 的电子；在给定 \(\nu\) 时，若 \(B\) 更强，所需 \(\gamma\) 更低。因此，当论文说 X-ray 同步辐射追踪新近加速的高能电子时，背后不仅是频率高，还因为这些高 \(\gamma\) 电子通常冷却很快。读图时若只看到 X-ray 变暗，不能立刻说粒子注入停止，也可能是 \(B\)、冷却、逃逸或吸收发生变化。

第三段：单个电子的同步辐射功率由磁场能量密度控制。磁场能量密度为 \(U_{B}=B^{2}/(8\pi)\)，相对论电子的同步功率可写作 \(P_{\rm syn}=\frac{4}{3}\sigma_{T}c\gamma^{2}\beta^{2}U_{B}\)。电子能量为 \(E_{e}=\gamma m_{e}c^{2}\)，因此同步冷却时间为 \(t_{\rm syn}=E_{e}/P_{\rm syn}=6\pi m_{e}c/(\sigma_{T}B^{2}\gamma)\)。这一步把谱和时间联系起来：高频光子对应高 \(\gamma\) 电子，而高 \(\gamma\) 电子有更短 \(t_{\rm syn}\)。

第四段：天体源中常把非热电子分布近似写成 \(N(\gamma)d\gamma=K\gamma^{-p}d\gamma\)，适用范围为 \(\gamma_{m}<\gamma<\gamma_{\rm max}\)。这个幂律可能来自激波加速、磁重联或湍流加速的近似结果，但它不是不可改变的定理。若源处于光学薄、未冷却、单区近似，并用 \(\nu\propto\gamma^{2}B\) 的映射，可以得到同步发射率 \(j_{\nu}\propto KB^{(p+1)/2}\nu^{-(p-1)/2}\)。对应到观测通量，若 \(F_{\nu}\propto\nu^{-\beta}\)，则未冷却同步谱段给出 \(\beta=(p-1)/2\)，也就是 \(p=2\beta+1\)。

第五段：冷却会改变上述简单映射。定义冷却 Lorentz 因子时，令冷却时间等于源年龄或动力学时间 \(t\)，得到 \(\gamma_{c}=6\pi m_{e}c/(\sigma_{T}B^{2}t)\)，对应冷却频率 \(\nu_{c}=\frac{3}{4\pi}\gamma_{c}^{2}\frac{eB}{m_{e}c}\)。在持续注入且辐射损失满足 \(\dot{\gamma}\propto-\gamma^{2}\) 的情形下，\(\gamma>\gamma_{c}\) 的电子谱常由 \(N(\gamma)\propto\gamma^{-p}\) 增陡为 \(N(\gamma)\propto\gamma^{-(p+1)}\)，同步谱则由 \(F_{\nu}\propto\nu^{-(p-1)/2}\) 变为 \(F_{\nu}\propto\nu^{-p/2}\)。因此，冷却断裂的典型诊断是光谱指数增加 \(1/2\)，但真实数据中多区叠加和吸收会使断裂变平滑。

第六段：逆康普顿辐射与同步辐射平行理解。同步辐射是电子在磁场中损失能量，逆康普顿则是电子把低能种子光子提升到高能。在 Thomson 极限中，若种子光子频率为 \(\nu_{0}\)，散射后频率满足 \(h\nu_{\rm IC}\simeq\frac{4}{3}\gamma^{2}h\nu_{0}\)，功率为 \(P_{\rm IC}=\frac{4}{3}\sigma_{T}c\gamma^{2}\beta^{2}U_{\rm rad}\)。因此，Compton 参数可近似写为 \(Y\equiv P_{\rm IC}/P_{\rm syn}\simeq U_{\rm rad}/U_{B}\)。这个关系是读 PWN、pulsar halo 和高能 afterglow 论文时的第一层诊断：同步峰与逆康普顿峰的相对强度在告诉我们磁场能量密度和光子能量密度的竞争。

第七段：逆康普顿公式有一个关键边界，即 Klein-Nishina 效应。Thomson 条件可写为 \(4\gamma h\nu_{0}\ll m_{e}c^{2}\)；若不满足，电子静止系中入射光子能量接近 \(m_{e}c^{2}\)，散射截面下降，能量提升不再服从简单的 \(\gamma^{2}\) 规律。观测上，这会让高能 IC 通量低于 Thomson 预言，也会改变电子冷却。读 TeV 谱时，不能把高能端偏低自动解释为电子数不足或最大能量太低，必须同时检查 Klein-Nishina 抑制、EBL 吸收和背景建模。

第八段：热辐射提供另一套诊断语言。若光子和物质接近热平衡，谱亮度为 \(B_{\nu}(T)=\frac{2h\nu^{3}}{c^{2}}\frac{1}{\exp(h\nu/kT)-1}\)，有效半径为 \(R\) 的近似黑体光度为 \(L_{\rm BB}=4\pi R^{2}\sigma_{\rm SB}T^{4}\)。低频 Rayleigh-Jeans 端、高频 Wien 端和峰位置可以帮助估计 \(T\)，归一化结合距离可估计有效面积。但在高能暂现源中，所谓光球辐射常受相对论 Doppler 因子、多温度叠加和 Compton 化影响，因而会比单温黑体更宽。本讲只建立这些基础关系，不进入 GRB prompt 光球或 TDE 盘结构细节。

第九段：最后，吸收和逃逸决定“产生的光子是否能被看见”。若吸收系数为 \(\alpha_{\nu}\)，光学深度为 \(\tau_{\nu}=\int\alpha_{\nu}ds\)，沿路径传播的强度可写为 \(I_{\nu}=I_{\nu,0}e^{-\tau_{\nu}}\)。同步自吸收频率由 \(\tau_{\nu_{a}}\simeq1\) 定义，低于该频率的 radio 辐射会被显著压低；自由-自由吸收来自热等离子体；光子-光子吸收会影响高能 gamma-ray。实际观测诊断中，逃逸条件 \(\tau_{\nu}\lesssim1\) 和冷却条件 \(t_{\rm cool}(\gamma_{c})=t_{\rm age}\) 一样重要。到这里，本讲完成了从电子、磁场、光子场、温度和光学深度到观测谱峰、斜率、断裂和 cutoff 的基础闭环；下一讲将把这些物理谱放入真实仪器响应和统计推断中。

公式与推导

第一步：先从单个电子在磁场中的运动定标开始。假设电子电荷量为 \(e\)、质量为 \(m_e\)，处在局域近似均匀的磁场 \(B\) 中；非相对论回旋角频率给出最基本的磁场时间尺度：

\[\omega_B=\frac{eB}{m_ec}.\]

这一步解出的是电子被磁场弯转的基础频率。高能天体中的电子通常满足 \(\gamma\gg1\)，辐射被相对论准直和到达时间压缩推到更高频率，因此下一步要把 \(\omega_B\) 转换成可观测的同步辐射特征频率。

第二步：加入相对论电子的 Lorentz 因子 \(\gamma\) 和 pitch angle \(\alpha_p\)。在单区、局域均匀磁场近似下，同步辐射的典型频率为

\[\nu_{\rm syn}=\frac{3}{4\pi}\gamma^2\frac{eB}{m_ec}\sin\alpha_p.\]

这个公式解出的是“哪个电子主要发出哪个频率”的映射：给定 \(B\) 和 \(\alpha_p\) 时，观测频率 \(\nu\) 主要选择 \(\gamma\sim(\nu/B)^{1/2}\) 的电子。下一步要问这些被选中的电子每秒损失多少能量。

第三步：假设辐射区中磁场能量密度为 \(U_B\)，电子速度为 \(v=\beta c\)，且 pitch angle 已做各向同性平均或吸收到数量级因子中。磁场能量密度与单电子同步功率为

\[U_B=\frac{B^2}{8\pi},\qquad P_{\rm syn}=\frac{4}{3}\sigma_Tc\gamma^2\beta^2U_B.\]

这一步解出的是单个电子的同步辐射损失率。它说明同步冷却随 \(\gamma^2\) 和 \(B^2\) 增强；因此高频同步光子不仅要求较高 \(\gamma\)，也通常对应较短寿命。下一步用电子能量除以功率得到冷却时间。

第四步：假设电子能量为 \(E_e=\gamma m_ec^2\)，且暂时忽略逆康普顿、绝热膨胀和逃逸损失，则同步冷却时间为

\[t_{\rm syn}=\frac{E_e}{P_{\rm syn}}=\frac{6\pi m_ec}{\sigma_TB^2\gamma}.\]

这一步把谱诊断和时间诊断连接起来：较大的 \(B\) 或较高的 \(\gamma\) 都使 \(t_{\rm syn}\) 变短。读 X-ray 同步辐射时，这个关系提醒我们高能电子可能只记录近期加速或持续注入。下一步从单个电子推广到一群非热电子。

第五步：假设加速过程产生幂律电子分布，并只在 \(\gamma_m<\gamma<\gamma_{\rm max}\) 的范围内使用这个近似：

\[N(\gamma)d\gamma=K\gamma^{-p}d\gamma.\]

这里 \(K\) 是归一化，\(p\) 是电子谱指数。这一步不是从第一性原理证明加速谱，而是建立读非热源论文时最常用的最低阶模型。下一步利用 \(\nu_{\rm syn}\propto\gamma^2B\) 把电子谱映射成光子谱。

第六步：在光学薄、单区、未冷却的限制下，用 \(\delta\)-function 近似认为单个电子的辐射集中在 \(\nu\sim\nu_{\rm syn}\)。于是同步体发射率满足

\[j_\nu\propto K B^{(p+1)/2}\nu^{-(p-1)/2}.\]

这一步解出的是电子幂律如何变成光子幂律。它的物理含义是：谱斜率主要由 \(p\) 决定，归一化则同时依赖电子数和磁场。因此仅凭一个同步谱段通常难以唯一确定 \(K\) 与 \(B\)。下一步把发射率斜率转成观测通量斜率。

第七步：若观测谱段确实是光学薄、未冷却同步辐射，并写作

\[F_\nu\propto\nu^{-\beta},\qquad \beta=\frac{p-1}{2},\qquad p=2\beta+1.\]

这就是从观测量反推电子谱的第一条闭合关系。使用它时必须先检查该谱段是否避开了自吸收、冷却断裂、热成分和仪器边界；否则由 \(\beta\) 反推的 \(p\) 只是有效斜率。下一步处理冷却会如何改变这个闭合关系。

第八步：令同步冷却时间等于源年龄或动力学时间 \(t\)，先忽略其他损失项，可定义冷却 Lorentz 因子及其对应的冷却频率：

\[\gamma_c=\frac{6\pi m_ec}{\sigma_TB^2t},\qquad \nu_c=\frac{3}{4\pi}\gamma_c^2\frac{eB}{m_ec}.\]

这一步解出的是冷却断裂的位置。若 \(\nu<\nu_c\)，对应电子尚未显著冷却；若 \(\nu>\nu_c\)，对应电子在源年龄内已经损失大量能量。下一步写出持续注入模型中冷却后谱斜率的标准极限。

第九步：假设电子持续注入，且高能端辐射损失满足 \(\dot{\gamma}\propto-\gamma^2\)。在 \(\gamma>\gamma_c\) 上，稳态电子谱相对于注入谱增陡 \(1\)，因此

\[N(\gamma)\propto\gamma^{-(p+1)},\qquad F_\nu\propto\nu^{-p/2}.\]

这一步给出冷却后同步谱的标准限制情形：从未冷却到冷却后，光谱指数通常增加 \(1/2\)。观测上若看到类似增陡，需要同时检查是否真是冷却断裂，而不是多区叠加、吸收或最大能量截止。下一步加入另一个常见冷却和辐射通道：逆康普顿散射。

第十步：假设电子 Lorentz 因子为 \(\gamma\)，种子光子频率为 \(\nu_0\)，散射处在 Thomson 区域且近似各向同性。散射后光子能量的典型提升为

\[h\nu_{\rm IC}\simeq\frac{4}{3}\gamma^2h\nu_0.\]

这一步解出的是逆康普顿峰的位置诊断：同一批电子可把 CMB、红外或星光提升到 X-ray、GeV 或 TeV 能段。下一步比较逆康普顿功率和同步功率，得到高能峰相对强度的诊断。

第十一步：若种子光子场能量密度为 \(U_{\rm rad}\)，且仍处于 Thomson 极限，单电子逆康普顿功率和 Compton 参数为

\[P_{\rm IC}=\frac{4}{3}\sigma_Tc\gamma^2\beta^2U_{\rm rad},\qquad Y\equiv\frac{P_{\rm IC}}{P_{\rm syn}}\simeq\frac{U_{\rm rad}}{U_B}.\]

这一步解出的是同步与逆康普顿相对强度的最低阶关系。若 \(Y\) 较大，总冷却时间应由同步和逆康普顿共同决定，而不能只用 \(t_{\rm syn}\)。下一步必须标出这个简洁关系的适用边界。

第十二步：检查入射光子在电子静止系中的能量。Thomson 近似要求

\[4\gamma h\nu_0\ll m_ec^2.\]

若该条件不满足，就进入 Klein-Nishina 区域，散射截面下降，\(h\nu_{\rm IC}\propto\gamma^2h\nu_0\) 和 \(Y\simeq U_{\rm rad}/U_B\) 都不再是可靠近似。观测上，高能逆康普顿通量偏低可能来自 Klein-Nishina 抑制，而不一定意味着电子数不足。下一步补上非热机制之外的热辐射诊断。

第十三步：若光子和物质接近热平衡，并用温度 \(T\) 与有效半径 \(R\) 描述发射区，黑体谱亮度和总光度为

\[B_\nu(T)=\frac{2h\nu^3}{c^2}\frac{1}{\exp(h\nu/kT)-1},\qquad L_{\rm BB}=4\pi R^2\sigma_{\rm SB}T^4.\]

这一步把热峰位置、温度和有效面积连接起来。谱峰测 \(T\)，归一化结合距离约束 \(R\)，但相对论光球、多温度叠加和 Compton 化会使真实热峰比单温黑体更宽。最后一步说明即使辐射产生了，也必须能逃逸才会被观测到。

第十四步：假设辐射沿路径 \(s\) 传播时遇到吸收系数 \(\alpha_\nu\)，光学深度与逃逸强度满足

\[\tau_\nu=\int\alpha_\nu ds,\qquad I_\nu=I_{\nu,0}\exp(-\tau_\nu),\qquad \tau_{\nu_a}\simeq1.\]

这一步统一描述吸收和同步自吸收频率 \(\nu_a\)。当 \(\tau_\nu\ll1\) 时，观测谱接近发射谱；当 \(\tau_\nu\gtrsim1\) 时，低频转折或高能 cutoff 可能主要来自逃逸概率，而不是电子谱本身截止。至此，推导链从单电子、电子分布、冷却、逆康普顿、热辐射到吸收闭合到观测诊断：读论文时应把谱峰、斜率、断裂和残差分别对应到 \(B\)、\(p\)、\(\gamma_c\)、\(Y\)、\(T\) 与 \(\tau_\nu\)，同时检查每个关系的适用条件。

经典拟合 / 应用方法

第一段：本讲的拟合要从“经验谱形”开始，而不是一上来就宣布某个物理模型正确。最稳妥的第一步，是用 broken power law 或 smoothly broken power law 描述观测到的 \(F_{\nu}\)、\(\nu F_{\nu}\) 或 \(E^{2}dN/dE\)。这一层模型的参数包括归一化、一个或多个断裂频率 \(\nu_{\rm br}\)、低能与高能谱指数、平滑参数，以及可能的低频吸收或高能 cutoff。它回答的问题是“谱形在哪里变斜、峰在哪里、斜率是多少”，而不是“磁场一定是多少”。例如在同步辐射语境下，经验断裂可能对应 \(\nu_a\)、\(\nu_m\)、\(\nu_c\) 或 \(\nu_{\rm max}\)，但在没有额外约束前，不能只凭一个折线就把它唯一命名为冷却断裂。

第二段：第二步才进入物理单区模型。单区同步模型通常把电子分布写成 \(N(\gamma)d\gamma=K\gamma^{-p}d\gamma\)，并加入 \(\gamma_m\)、\(\gamma_{\rm max}\)、磁场 \(B\)、源区半径 \(R\)、年龄或动力学时间 \(t\)、距离、填充因子和吸收参数。若加入逆康普顿成分，还需要种子光子能量密度 \(U_{\rm rad}\)、种子光子谱形以及 Klein-Nishina 修正；若加入热成分，则要拟合温度 \(T\) 和有效半径 \(R_{\rm BB}\) 或等价归一化；若存在吸收，则要拟合光学深度 \(\tau_{\nu}\)、氢柱密度或自由-自由吸收参数。物理模型的价值在于把 \(\nu_a\)、\(\nu_m\)、\(\nu_c\)、峰通量、IC 峰和热峰连接到 \(B\)、电子能量、冷却时间、光子场和有效面积，但前提是模型假设必须明示。

第三段：似然函数必须和数据类型匹配。若论文给出的是已经校准并可近似独立的 flux points，可以先用 Gaussian 似然：

\[\ln\mathcal{L}=-\frac{1}{2}\sum_i\left[\frac{F_i^{\rm obs}-F_i^{\rm mod}}{\sigma_i}\right]^2+\mathrm{const}.\]

这里的 \(F_i^{\rm mod}\) 是模型在对应频率或能段上的预测，\(\sigma_i\) 包含统计误差，有时还应加入系统误差项。若数据来自 X-ray 或 gamma-ray 计数，则不应把 counts 直接当成物理 \(F_{\nu}\)。更合适的是 Poisson 似然：

\[\ln\mathcal{L}=\sum_i\left[n_i\ln\mu_i-\mu_i-\ln(n_i!)\right],\]

其中期望计数可概念性写为

\[\mu_i=\int R_i(E)F_E^{\rm mod}dE+b_i.\]

这里 \(R_i(E)\) 代表响应、有效面积、曝光和能量重分布的综合作用，\(b_i\) 是背景期望。完整响应细节留到第 3 讲，本讲只要求学生知道物理谱和观测计数之间隔着仪器。

第四段：先验和参数边界会强烈影响后验，尤其在宽带数据不完整时。常见先验包括 \(p\) 的合理范围、\(B>0\)、\(R>0\)、\(\gamma_m<\gamma_{\rm max}\)、总能量不超过合理能量预算，以及冷却时间不与假设年龄严重矛盾。若模型声称某段为未冷却同步辐射，就要检查拟合出的 \(\nu_c\) 是否确实高于该观测段；若模型声称低频峰是同步自吸收，就要检查 \(\tau_{\nu_a}\simeq1\) 是否在峰附近成立。先验不是装饰，它会排除一些数学上可拟合、物理上不自洽的解。

第五段：后验解释要关注相关性，而不是只报最佳拟合值。同步通量大致由电子数和磁场共同决定，所以 \(B\) 与电子总能量常强反相关；谱斜率可由 \(p\) 控制，也可能被吸收、非同时观测或多区叠加改变，因此 \(p\) 与吸收参数可能相关；IC 强度与 \(U_{\rm rad}\)、电子数和 \(B\) 相关，因此 \(Y\equiv P_{\rm IC}/P_{\rm syn}\) 的不确定度不应只由高能峰的统计误差决定。读论文时要看边缘置信区间、二维置信轮廓、角图和 posterior predictive band。若角图中 \(B\) 与电子能量形成长条形退化方向，单个“最佳 \(B\)”并不能代表模型知道了真实磁场。

第六段：残差诊断是判断模型缺失物理的关键。若 radio 低频点系统性低于光学薄同步模型，候选解释包括同步自吸收、自由-自由吸收、源混叠或低频通量标定问题；若 optical 到 X-ray 的斜率比未冷却同步关系 \(\beta=(p-1)/2\) 更陡，可能是冷却断裂，也可能是红化、宿主吸收或非同时观测；若 TeV 通量低于 Thomson IC 预言，要检查 Klein-Nishina 抑制、EBL 吸收、背景扣除和扩展源模板；若残差在仪器能段交界处突然翻转，优先怀疑交叉定标或响应问题，而不是马上发明新成分。

第七段：系统误差要和统计误差一起进入解释。距离误差会影响光度和能量；红化与氢柱密度会改变 optical、UV 和 soft X-ray 斜率；绝对通量标定会移动整条 SED；非同时多波段数据会在暂现源中制造假峰；扩展源的空间模板会影响 gamma-ray 总通量；种子光子场模型会改变 IC 预测；电子低能截断会影响总能量但未必明显改变观测谱。一个实用流程是：先用最少参数模型拟合主要斜率和峰，再逐项加入吸收、IC、热成分、多区结构或 cutoff；每加入一个成分，都要说明新增参数由哪一段数据约束，并用残差、信息准则、后验预测或交叉验证判断它是否真的必要。

重点章节 / 结果段落怎么读

阅读经典同步辐射或余辉谱段论文时，第一遍不要急着追所有源动力学细节，而要抓住“粒子分布如何映射到光子谱”的主线。凡是论文定义 \(N(\gamma)\)、\(p\)、\(\gamma_m\)、\(\gamma_c\)、\(\nu_a\)、\(\nu_m\)、\(\nu_c\) 和峰通量的段落，都是本讲的核心延伸。读这些段落时应逐句标出：作者假设电子是单幂律还是 broken power law？冷却是否只由同步辐射控制？源是否光学薄？自吸收是内部同步自吸收还是外部吸收？这些假设决定了能否使用 \(\beta=(p-1)/2\) 或冷却后 \(\beta=p/2\) 的闭合关系。

读 Sari、Piran 与 Narayan 关于 GRB afterglow 谱段的经典结果时，建议把注意力放在谱构造逻辑上，而不是把本讲提前变成外激波动力学课。他们的谱由几个特征频率和一个峰通量组织起来，快冷与慢冷的区别来自 \(\nu_m\) 与 \(\nu_c\) 的排序。课堂读法是：先画出 \(F_{\nu}\) 对 \(\nu\) 的 log-log 图，再标出 \(\nu_a\)、\(\nu_m\)、\(\nu_c\)，最后检查每个谱段斜率是来自单电子低频尾、未冷却电子、冷却电子还是吸收。这样读会发现，经典余辉谱段其实是在反复使用本讲的同步辐射、冷却和吸收语言。

读 Kennel 与 Coroniti 的 Crab PWN 经典模型时，重点不是一开始掌握所有 MHD 方程，而是看他们怎样把脉冲星风、终止激波、磁场、相对论电子和同步辐射亮度放入同一图像。Crab 的教学价值在于不同能段电子寿命不同：radio 电子可保存较长时间，X-ray 同步电子冷却很快，因此 X-ray 形态和谱软化更接近近期加速与输运。读相关段落时要问：作者如何估计 \(B\)？如何由谱斜率推断电子分布？如何解释从中心到外侧的谱变软？这些问题比单纯记住 Crab 的某个参数值更重要。

读现代 PWN、pulsar halo 或 TeV 扩展源论文时，IC 相关段落要按三件事拆开：电子注入谱、种子光子场和输运。TeV gamma-ray 不是电子分布的直接照片，而是电子分布与 CMB、红外、星光等 \(U_{\rm rad}\) 卷积后的结果。若论文写到 Klein-Nishina 修正，应检查 \(4\gamma h\nu_0\ll m_ec^2\) 是否仍成立；若不成立，高能 IC 谱变软并不一定说明 \(\gamma_{\rm max}\) 很低。读图和读表时还要看作者是否同时拟合同步 X-ray 与 IC gamma-ray，因为两者相对强度才真正约束 \(U_B\) 与 \(U_{\rm rad}\) 的竞争。

读 TDE radio、超新星 radio 或其他 radio transient 的同步自吸收反演论文时，关键段落通常围绕峰频、峰通量、距离和观测时间展开。作者常用 \(\tau_{\nu_a}\simeq1\) 以及 equipartition 或最小能量假设来估计 \(R\)、\(B\) 和总能量。课堂读法是先分清观测量与假设量：峰频和峰通量是图上读出的，距离来自红移或宿主，几何、填充因子、等能量假设和球对称性则是模型假设。然后追问：如果偏离 equipartition，能量会增加多少？如果源是喷流或非球对称，\(R\) 和 \(B\) 会如何改变？如果低频衰减来自自由-自由吸收而非同步自吸收，整个反演是否仍成立？

读热辐射或光球相关段落时，要特别避免把所有峰都叫温度。真正的单温黑体应接近 \(B_{\nu}(T)\)，低频和高频端形状都有约束；高能暂现源中的热峰可能被相对论运动、多温度叠加、Compton 化和几何效应展宽。读这些段落时，先看作者是否给出 \(T\)、\(R_{\rm BB}\) 和 \(L_{\rm BB}=4\pi R^2\sigma_{\rm SB}T^4\) 的自洽关系，再看是否比较了非热模型、吸收模型和多温度模型。若只在窄能段中看到一个弯曲谱形，就声称发现热峰，通常是不够的。

最后，阅读任何宽带拟合论文时都要把“结论段”拆回“模型段、数据段、残差段和系统误差段”。如果结论说 \(B\) 很弱，要检查是否有 IC 峰约束；如果结论说 \(p\) 很硬，要检查吸收和仪器响应是否会改变斜率；如果结论说存在第二成分，要检查残差是否连续支持它，而不是只由一个离群点驱动。这样的读法把经典工作从公式库变成训练诊断能力的课程材料。

建议重点查看的图表

一，宽带 SED 图。先确认横轴是频率 \(\nu\)、光子能量 \(E\) 还是波长，纵轴是 \(F_{\nu}\)、\(\nu F_{\nu}\)、\(dN/dE\) 还是 \(E^{2}dN/dE\)。在 \(\nu F_{\nu}\) 或 \(E^{2}dN/dE\) 图上，峰值更接近每个对数频率或对数能量区间的能量输出。重点看同步峰、IC 峰、热峰、低频转折和高能 cutoff 的位置，并检查每个峰是否有足够波段覆盖。

二，同步谱段与断裂频率图。检查 \(\nu_a\)、\(\nu_m\)、\(\nu_c\) 和峰通量是否被清楚标出，并判断源处在快冷还是慢冷。若纵轴是 \(F_{\nu}\)，未冷却段可对应 \(F_{\nu}\propto\nu^{-(p-1)/2}\)，冷却段可对应 \(F_{\nu}\propto\nu^{-p/2}\)；若纵轴换成 \(\nu F_{\nu}\)，视觉斜率会整体改变。不要在一个很窄的频段上过度声称测得 \(p\)。

三，冷却时间比较图。横轴可为 \(\gamma\)、电子能量 \(E_e\) 或对应光子频率，纵轴为 \(t_{\rm syn}\)、\(t_{\rm IC}\)、\(t_{\rm age}\) 和 \(t_{\rm esc}\)。重点看冷却曲线与年龄或逃逸时间的交点，从而读出 \(\gamma_c\)、\(\nu_c\) 或最大可存活能量。若 IC 冷却曲线在高能端不再按 Thomson 预期下降，要检查 Klein-Nishina 抑制。

四，电子能谱与能量分布图。横轴通常是 \(\gamma\) 或 \(E_e\)，纵轴可能是 \(N(\gamma)\)、\(\gamma^2N(\gamma)\) 或能量密度。重点看注入谱指数 \(p\)、低能截断 \(\gamma_m\)、冷却断裂 \(\gamma_c\)、高能 cutoff \(\gamma_{\rm max}\)，以及电子能量主要集中在哪一段。注意 \(\gamma^2N(\gamma)\) 图强调每个对数 \(\gamma\) 区间的能量，不等同于粒子数最多的位置。

五，自吸收 radio 峰图。若横轴是频率，检查低频斜率是否接近同步自吸收预期，还是更像自由-自由吸收造成的指数式压低；若横轴是时间，检查峰频是否随时间降低、峰通量如何演化。关键诊断是 \(\tau_{\nu_a}\simeq1\) 附近是否与模型一致，并且低频数据是否足够密集。

六，多成分模型分解图。看 synchrotron、SSC、external IC、blackbody、absorbed power law、thermal plasma 等成分是否分别画出，合成模型是否确实优于单成分模型。重点问每个成分由哪一段数据约束：热成分是否有峰两侧数据，IC 成分是否有 GeV 或 TeV 点，自吸收是否有低频 radio 点。若一个成分只靠一个数据点支撑，要非常谨慎。

七，残差与比值图。横轴应与主谱图一致，纵轴可能是 \((D-M)/\sigma\)、data/model 或 \(\chi\) residual。重点看残差是否随机围绕零分布，还是在低频、soft X-ray、高能 gamma-ray 或仪器交界处出现连续正负结构。连续残差通常比单个 \(\chi^2\) 数值更能提示缺失的吸收、冷却、KN、背景或校准问题。

八，参数后验角图。重点读 \(B\)、电子总能量、\(p\)、\(\nu_c\)、\(Y\)、\(U_{\rm rad}\)、吸收柱密度和归一化之间的相关性。若二维轮廓很长，说明参数之间存在强简并；若后验贴着先验边界，说明数据没有真正约束该参数。好的论文应同时给出边缘区间、相关性和模型预测带，而不是只给最佳拟合表。

关键图表逐图导读

图 1：宽带 SED 图。横轴若从 radio 延伸到 TeV，通常跨越十几到二十个数量级；纵轴若为 \(\nu F_{\nu}\) 或 \(E^{2}dN/dE\)，峰值表示能量输出最集中的对数频率或对数能量区间。读图时先分辨低能宽峰、高能宽峰和可能的热峰：低能宽峰常由同步辐射解释，高能宽峰可能来自 IC，较窄且接近 Planck 形状的峰才可能支持热辐射。模型线要看总模型和各分量是否同时给出。误读风险是看到双峰就自动称为 SSC；实际上第二峰可能是外部 IC、hadronic 成分、背景残留或非同时数据拼接出来的假峰。

图 2：同步谱段示意图。横轴为 \(\nu\)，纵轴为 \(F_{\nu}\)，模型线在 \(\nu_a\)、\(\nu_m\)、\(\nu_c\) 附近改变斜率。读图时先判断断裂顺序，从而区分快冷和慢冷；再检查观测点覆盖的是自吸收段、低频单电子尾、未冷却幂律段还是冷却段。若模型声称由斜率得到 \(p\)，要核对使用的是 \(F_{\nu}\propto\nu^{-(p-1)/2}\) 还是 \(F_{\nu}\propto\nu^{-p/2}\)。常见误读是把任何变陡都叫冷却断裂，但最大能量 cutoff、多区叠加、前景吸收和仪器带宽边缘也会制造类似弯曲。

图 3：冷却时间比较图。横轴可以是 \(\gamma\)、\(E_e\) 或对应的同步频率，纵轴是时间。同步冷却曲线通常随 \(\gamma\) 增大而下降，IC 冷却在 Thomson 区域也类似下降，\(t_{\rm age}\) 常画成水平线，\(t_{\rm esc}\) 可能随能量变化。交点给出 \(\gamma_c\) 或最大可存活能量，并进一步映射到 \(\nu_c\)。读图时要看模型是否把 \(t_{\rm cool}^{-1}=t_{\rm syn}^{-1}+t_{\rm IC}^{-1}+\cdots\) 的总冷却考虑进去。误读风险是只看 \(t_{\rm syn}\) 而忽略强辐射场下的 IC 冷却，或在 KN 区域仍使用 Thomson 近似。

图 4：电子谱与光子谱对照图。上图可能画 \(N(\gamma)\) 或 \(\gamma^2N(\gamma)\)，下图画同步和 IC 光谱。读图时要把电子谱中的 \(\gamma_m\)、\(\gamma_c\)、\(\gamma_{\rm max}\) 分别映射到光子谱中的 \(\nu_m\)、\(\nu_c\) 和高能 cutoff，并检查模型线是否经过数据点而非只匹配峰位。残差若在光子谱 cutoff 附近呈系统结构，可能说明电子 cutoff 形状、辐射核函数、吸收或仪器响应处理不当。误读风险是把光子 cutoff 的形状一比一等同于电子 cutoff；实际映射会被同步核函数、IC 核函数和吸收平滑。

图 5：自吸收 radio 峰图。若横轴为频率、纵轴为 \(F_{\nu}\)，低频侧的上升斜率和峰位可检验 \(\tau_{\nu_a}\simeq1\)；若横轴为时间、纵轴为某一频率下的 flux density，则峰出现的时间反映该频率从光学厚变为光学薄的过程。模型线要同时解释峰频随时间移动和峰通量演化，不能只拟合某一时刻的峰值。误读风险是把所有低频压低都归因于同步自吸收；外部自由-自由吸收、闪烁、非同时观测和低频校准也会改变 radio 峰。

图 6：多成分模型分解图。横轴通常为 \(\nu\) 或 \(E\)，纵轴为 \(\nu F_{\nu}\) 或 \(E^{2}dN/dE\)，不同线型代表 synchrotron、SSC、external IC、blackbody、absorbed model 或背景成分。读图时先看总模型是否穿过数据，再看每个分量是否有独立波段支撑。例如黑体成分应在峰两侧有约束，IC 成分应由 GeV 或 TeV 数据限制，吸收成分应改善低频或 soft X-ray 残差。误读风险是把“多画几条线后看起来更好”当作物理发现；自由参数过多时，模型可能只是过拟合。

图 7：残差图。横轴与主谱图一致，纵轴可能是 data/model、\((D-M)/\sigma\) 或贡献到 \(\chi^2\) 的标准化残差。读图时要寻找连续结构：radio 端连续为负可能提示吸收或非同时数据，soft X-ray 连续为负可能提示氢柱密度不足，GeV 到 TeV 的弯曲残差可能提示 KN、EBL 或背景建模问题。单个离群点可以是统计涨落或交叉定标问题，连续多个点同向偏离才更像缺失物理。误读风险是只看总 \(\chi^2\) 可接受就忽略局部系统残差。

图 8：后验角图。对初学者，最重要的是看二维轮廓的方向和宽度。若 \(B\) 与电子总能量强反相关，说明同步通量不能单独确定二者；若 \(p\) 与吸收柱密度相关，说明低能吸收会污染谱斜率；若 \(Y\) 与 \(U_{\rm rad}\) 或 \(B\) 相关，说明 IC 峰强度的解释并不唯一。模型预测带应覆盖数据及其不确定度，而不是只给一条最佳拟合线。误读风险是把边缘后验的中位数当作唯一物理值，忽略参数简并和先验边界。

后续阅读路径

第一条路径是从本讲走向 GRB afterglow。建议先读 Sari、Piran 与 Narayan 关于同步余辉谱段的经典工作，读法是把每个特征频率都对应回本讲的 \(\gamma_m\)、\(\gamma_c\)、\(\nu_a\)、\(\nu_m\)、\(\nu_c\)、\(B\) 和 \(t\)。第一遍只关注辐射谱段：哪些斜率来自未冷却电子，哪些来自冷却电子，哪些来自自吸收。第二遍再读外激波动力学如何决定这些频率随时间演化。这样可以避免把辐射机制、动力学演化和观测拟合三件事混在一起。

第二条路径是从本讲走向 PWN、pulsar halo 和扩展 TeV 源。先用 Crab PWN 理解“注入、磁场、同步冷却、空间输运”的全波段图像，再读 H.E.S.S.、HAWC、LHAASO 等关于 TeV halo 或 PWN 的论文。阅读时可以做三个估算：给定 TeV 光子能量，估计需要多高的电子 \(\gamma\)；给定种子光子场，检查 IC 是否仍在 Thomson 区域；给定年龄和扩展角尺度，判断电子能否在冷却前扩散到观测范围。这条路径会把本讲的 \(t_{\rm syn}\)、\(t_{\rm IC}\)、\(Y\) 和 Klein-Nishina 边界真正用起来。

第三条路径是走向 TDE radio、超新星 radio 和其他 radio transient。这里最有用的桥梁是 synchrotron self-absorption 和 equipartition 反演。先学习峰频、峰通量、距离和时间如何给出 \(R\)、\(B\) 和能量，再比较不同几何、填充因子、非等能量、外部自由-自由吸收和非同时观测对结果的影响。读这类论文时，不要只记住反演公式，而要追问 \(\tau_{\nu_a}\simeq1\) 是否被数据支持，以及低频侧是否真的测到足够斜率。

第四条路径是走向热辐射、光球和 Compton 化，但要保持本讲边界。可以先读黑体、modified blackbody 和 Comptonized spectrum 的基础材料，理解 \(B_{\nu}(T)\)、\(L_{\rm BB}=4\pi R^2\sigma_{\rm SB}T^4\) 与真实宽峰之间的差别。之后再进入 GRB prompt 光球、TDE 盘或超软 X-ray 成分等专题。这样安排的好处是，学生不会把本讲的热辐射公式误用成“看到峰就量温度”，而会先检查峰宽、低频端、高频端和非热替代模型。

第五条路径是为第 3 讲准备统计和仪器语言。把本讲的每个物理谱想象成进入真实探测器之前的 \(F_E^{\rm mod}\)，下一步它会被有效面积、能量响应、曝光、点扩散函数和背景折叠成 \(\mu_i\)。建议复习 Poisson 似然、上限、背景区、显著性和响应矩阵的基本概念。这样进入下一讲时，学生会清楚地区分两个问题：物理模型预测了什么谱，以及仪器是否有能力可靠地区分这些谱。

下一讲建议

第 3 讲最自然的方向是“从计数到发现：仪器响应、背景、显著性和多波段联合拟合”。本讲停在物理谱层面，讨论 \(F_{\nu}\)、\(\nu F_{\nu}\)、同步辐射、逆康普顿、热辐射、冷却和吸收；下一讲应转向真实数据层面，说明探测器测到的是 counts 而不是直接的 \(F_{\nu}\)，并系统介绍有效面积、能量响应、曝光、背景、点扩散函数、上限和 Poisson 似然。

第二个可作为第 3 讲主线的方向，是用同一个 toy source 贯穿练习：先设定一个包含 synchrotron、IC 和 absorption 的物理 \(F_E\)，再通过概念响应 \(\mu_i=\int R_i(E)F_E^{\rm mod}dE+b_i\) 生成计数，最后用 likelihood 恢复参数并比较残差。这样能自然连接第 1 讲的观测量语言、第 2 讲的辐射物理和第 3 讲的统计推断。

第三个可选方向是“显著性与模型比较”。本讲已经强调残差和后验不能只看最佳拟合；下一讲可以进一步讲 on/off 背景、Li 与 Ma 显著性、likelihood ratio、trial factor、upper limit 和模型选择。边界要明确：下一讲不重新推导同步辐射或 IC 公式，而是回答这些物理成分在有限统计、背景和仪器响应下是否真的被数据支持。
