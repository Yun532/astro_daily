经典工作详细讲解

本节是“GRB prompt emission”系列的第 1 讲，目标不是先解释辐射微物理的全部细节，而是建立 prompt emission 这门课的两个地基：第一，为什么毫秒到数十秒的伽马射线爆发会立即逼出一个相对论性、光学薄的外流图像；第二，为什么 BATSE 时代的经验谱——尤其是 Band 函数——至今仍是读 prompt 论文时的共同语言。上一轮或相邻课程中如果已经学习过 GRB afterglow，重点通常放在外流与外部介质相互作用后的 Blandford–McKee 自相似减速、同步辐射谱和多波段光变曲线；而本讲刻意回到更早的时刻：中心引擎刚释放能量、辐射主要在 keV–MeV 能段出现、时间结构高度不规则、观测者还看不到典型 afterglow 的阶段。第 2、3 讲才会更系统进入 internal shock、磁耗散、photosphere、synchrotron line-of-death、时间分辨谱演化和辐射机制判别。

历史动机可以从一个表面上很简单的问题开始：如果 GRB 在宇宙学距离释放巨大能量，为什么我们能直接看到 MeV 光子？在非相对论、静止源的直觉里，一个在短时间内发出巨大伽马射线流量的紧凑区域应当充满高能光子，高能光子之间通过 γγ→e± 产生大量正负电子对，使源区对伽马射线高度不透明。这就是所谓 compactness problem。Goodman、Paczynski、Shemi 与 Piran 等经典工作的重要性在于，他们将这个矛盾转化为对源区运动状态的定量要求：辐射区必须以很大的 bulk Lorentz factor 向观测者运动，典型 Γ 至少为 10^2，很多明亮高能事件甚至要求更高。这个结论不是某个具体辐射模型的附属品，而是 prompt emission 的几何和相对论运动底线。
BATSE 的革命性贡献在于样本和能段覆盖。BATSE 不只是发现 GRB 各向同性分布、推动宇宙学距离图像，也给出了大量 prompt 光谱，使人们可以用统一经验函数描述 keV–MeV 谱形。Band 等人在 1993 年提出的 Band 函数本质上是一个平滑拼接的 broken power law：低能端用指数截断幂律，高能端用幂律延伸，并保证函数值和斜率在拼接处连续。它并不声称自己是唯一物理辐射机制，但它把观测中最重要的几个量压缩成低能谱指数 \(\alpha\)、高能谱指数 \(\beta\)、特征能量 \(E_0\) 或 \(\nu F_\nu\) 峰值能量 \(E_{\rm pk}\) 以及归一化。直到 Fermi/GBM、Konus-Wind、Swift/BAT 时代，许多 prompt 论文仍以 Band 参数作为第一层比较语言。

本讲的关键科学问题因此有三层。第一层是“透明性问题”：在观测到高能光子的情况下，源区的大小、变时标和 Lorentz factor 如何相互约束？第二层是“经验谱问题”：给定探测器计数谱，如何通过响应矩阵折叠模型，得到 α、β、\(E_pk\) 等 photon spectrum 参数，而不是把计数峰误读成物理峰？第三层是“诊断问题”：这些参数如何帮助我们识别辐射机制的候选范围，例如非热同步辐射、准热 photosphere、Compton 化或多成分叠加。第 1 讲只建立这些量的意义和基本检验方式，不在这里过早宣布某个机制胜出。
Band 函数的关键结果之一是让 prompt 光谱从“很硬的一团伽马射线”变成可比较的参数空间。通常人们关心 \(\nu F_\nu\) 或 \(E^2N(E)\) 中的峰值 \(E_{\rm pk}\)，因为它接近每对数能段能量输出最大的地方；\(\alpha\) 控制低于峰值的光子数谱斜率；\(\beta\) 控制高能尾部是否陡降、能量积分是否收敛。经验上许多 burst 的 \(E_{\rm pk}\) 落在几十 keV 到数 MeV，\(\alpha\) 常在 -1 附近，\(\beta\) 常在 -2 到 -3 一带，但分布很宽，并且强烈依赖时间分辨、能段覆盖和选择效应。特别要注意：时间积分谱会把不同瞬时谱叠加在一起，可能人为拓宽谱形、软化或硬化某些指数，因此不能把一次 time-integrated Band 拟合直接等同于瞬时辐射机制。

这套经典框架也有明显局限。compactness 给出的 Γ 下限依赖最高能光子能量、变时标、谱外推和辐射半径假设；如果光子场各向异性、发射区分层、或高能光子来自不同区域，简单的一区估计会改变。Band 函数则只是经验参数化，拟合好并不代表物理解释唯一；一个窄的 photospheric 成分加一个非热尾部、多个 cutoff power law 的时间叠加，或者探测器能段有限造成的退化，都可能被 Band 函数近似吸收。读论文时应特别检查残差、参数置信区间、模型比较准则和能段边界，而不是只看表格中的 α、β、\(E_pk\)。

后续影响非常深远。相对论火球图像把 GRB 从“天体瞬变事件”推进到“超相对论外流实验室”；Band 谱把 prompt emission 的数据分析标准化，使不同卫星、不同样本、不同模型可以围绕同一组谱参数展开争论。更重要的是，这两条线共同塑造了现代问题：如果 Γ 很大，辐射区半径在哪里？如果 \(E_pk\) 与光度或能量有经验相关，是否反映辐射机制、喷流结构还是选择效应？如果 α 经常比简单快冷同步辐射预期更硬，是同步模型需要修改，还是存在 photosphere/磁重联/持续加热？这些问题会自然导向本系列后两讲。

背景知识

GRB prompt emission 的历史背景可以追溯到一个先验几乎不存在的观测事实：天空中会突然出现持续毫秒到数百秒的伽马射线闪光，而且时间结构极其复杂。在早期，距离尺度并不清楚，银河系、中子星磁层、太阳系边缘甚至宇宙学模型都曾被讨论。真正改变局面的，是大样本各向同性分布和 logN–logP 偏离欧氏均匀分布的证据，以及后来 afterglow 红移测量确认的宇宙学距离。但即使在红移确认之前，prompt 的高亮度、短变时标和高能光子已经提出了极端物理条件问题。

本讲的核心问题是：我们如何从 prompt 伽马射线本身推出源区必须相对论性运动，并且如何用一个稳健的经验谱描述大量爆发？这两个问题看似一个偏理论、一个偏数据，却在课程中必须连在一起。因为 compactness 估计需要观测光度、变时标和高能谱尾；而这些量又来自光谱拟合与时间分析。换句话说，Lorentz factor 下限不是凭空写出的，它依赖我们如何从探测器计数恢复 photon spectrum，如何定义 \(E_pk\)，如何判断高能尾是否真实存在。

为什么这件事重要？第一，它给出 GRB 相对论喷流的最早、最直接证据。Afterglow 的 Blandford–McKee 减速解告诉我们外流后来如何扫起外部介质，但 prompt compactness 约束告诉我们在发射 prompt 时外流已经拥有很大的 Γ。第二，它决定了辐射区尺度：观测变时标 δt 与相对论运动共同给出典型半径 R≈2Γ^2cδt 的量级估计，这会影响 internal shock、photosphere、磁耗散等模型的可行空间。第三，它是高能天体物理中少见的“用透明性反推动力学”的经典案例。
Band 函数之所以成为经典，并不是因为它是最深刻的理论模型，而是因为它足够灵活、足够稳定，并且与探测器数据分析流程兼容。对于 BATSE、Fermi/GBM 这类宽能段伽马射线仪器，观测到的是不同能道中的计数，而不是直接的真实光子谱。研究者需要假设一个 photon model，将其通过仪器响应矩阵折叠为预测计数，再与实际计数比较。Band 函数提供了一个低能斜率、峰值位置和高能斜率都可调的基准模型，因此可以作为不同 burst、不同时间片、不同卫星之间的共同参照。

常见误区之一，是把 Band 函数看成“GRB 的物理定律”。事实上 Band 参数是经验描述量，必须结合模型残差和能段覆盖解释。另一个误区，是把 \(E_pk\) 当成探测器计数谱峰值；严格地说，\(E_pk\) 是能量通量谱 νFν 或 \(E^2N(E)\) 的峰值，只有在完成响应修正和模型拟合后才有物理意义。第三个误区，是忽略时间分辨：一个 burst 的 time-integrated spectrum 往往混合了 hard-to-soft 演化、强度追踪演化和多个脉冲，因此其 α、β 和 \(E_pk\) 未必代表任何单一瞬时辐射区。

还要警惕“只用一个数字理解 prompt”的倾向。例如只报告 Γ_min、\(E_iso\) 或 \(E_pk\) 都不足以说明物理图像。Γ_min 是下限，不是直接测量；\(E_iso\) 假设各向同性，不等于喷流真实能量；\(E_pk\) 的分布受触发阈值、能段选择和红移缺失影响。一个好的 prompt 读法应当同时看光变、时间分辨谱、最高能光子、谱滞后、残差结构和样本选择。第 1 讲把这些基本读法先建立起来，后面才有可能严肃讨论辐射机制。

与 afterglow 课程相比，本讲的边界也要清楚。Afterglow 中常用的外激波同步辐射具有相对平滑的多波段光变和较可控的动力学演化；prompt 则发生在更靠近中心引擎或喷流内部的区域，时间结构直接携带引擎活动、壳层碰撞、磁耗散或 photosphere 调制的信息。本讲不会展开 Blandford–McKee 推导，也不会把 internal shock 当作默认正确模型；它只说明所有这些模型在进入舞台前必须满足的观测和相对论约束。

基础理论 / 方法脉络

理解 prompt emission 的第一步，是区分三个层次的量：观测者直接得到的是探测器计数随时间和能道的变化；经过仪器响应和背景建模后，才拟合出光子数谱 N(E)，通常单位可写成 photons cm^-2 s^-1 keV^-1；再由红移和宇宙学距离可推得源静止系的光度、能量和峰值能量。课程中最容易混淆的是 photon spectrum 与 energy spectrum：如果 N(E) 是每单位能量的光子数谱，则能量通量谱是 E N(E)，而每对数能段的能量分布是 \(E^2\) N(E)，Band 谱中常说的 \(E_pk\) 指的正是 \(E^2N(E)\) 的峰值位置。

第二步是建立紧凑源的直觉。GRB prompt 光变常有很短的变时标 δt，从毫秒到秒量级不等。若源区不相对论运动，因果性会给出 R≲cδt 的大小估计；同时宇宙学距离下的观测流量意味着极高光度。于是源区内光子数密度巨大，高能光子很容易与低能靶光子发生 γγ→e^+e^-。如果光学深度 τ_{γγ}≫1，MeV 甚至 GeV 光子就不应逃逸，观测谱也应趋向热化，这与非热高能尾和高能光子探测相矛盾。

第三步引入相对论外流。若辐射区以 Lorentz factor Γ 朝向观测者运动，至少有三件事同时改变：观测到的光子能量相对共动系被蓝移，源区允许的半径因角延迟关系变为 R∼2Γ^2cδt，光子场在共动系中的能量密度下降，并且高能光子在共动系中未必超过 pair-production 阈值。这些效应合在一起，使 γγ 光学深度对 Γ 有很强依赖，通常可近似理解为 Γ 越大，源越透明。由 τ_{γγ}<1 得到的 Γ_min 是 prompt 理论中最基本的诊断量之一。

第四步是把时间结构和发射半径联系起来。对于一个相对论薄壳，如果壳面上不同角度的光子同时发出，到达观测者的时间差约为 R/(2Γ^2c)。因此，当我们看到一个宽度为 δt 的脉冲时，常用 R≈2Γ^2cδt 作为数量级估计。但这不是定理，而是依赖发射几何和时间结构来源的估计：δt 可能来自中心引擎开关、壳层厚度、磁耗散斑块或曲率效应。读论文时要看作者把 δt 用作最短变时标、脉冲宽度还是整个 T90 的一部分，因为这会改变半径和 compactness 约束。

第五步进入经验谱。Band 函数可以写成低能段 \(N(E)=A(E/E_{\rm piv})^\alpha \exp(-E/E_0)\)，高能段为与之平滑连接的 \(N(E)=A'(E/E_{\rm piv})^\beta\)；其中拼接能量由 \(\alpha\)、\(\beta\)、\(E_0\) 决定，而 \(\nu F_\nu\) 峰值在低能指数满足 \(\alpha>-2\) 时为 \(E_{\rm pk}=(2+\alpha)E_0\)。这里的 \(\alpha\)、\(\beta\) 是 photon index，不是能量谱指数；如果 \(\alpha=-1\)，则 \(E^2N(E)\) 在低能侧随 \(E^1\) 上升。高能端若 \(\beta<-2\)，\(\nu F_\nu\) 能量积分在高能侧收敛；若 \(\beta\) 接近或大于 -2，则总能量和最高能端观测会变得特别敏感。

第六步是理解为什么 Band 谱只是第一层语言。实际 prompt 谱常用 Band、cutoff power law、blackbody+power law、multi-color blackbody、synchrotron-inspired model 等比较。若高能端计数不足，Band 的 β 可能不受约束，此时 cutoff power law 可能统计上更稳定；若低能端能段太窄，α 与 \(E_pk\) 会强相关；若存在热成分，单一 Band 可能给出看似异常的 α 或残差凸起。好的分析不应只问“Band 拟合是否收敛”，还要问残差是否系统偏离、参数后验是否贴边、不同模型是否在物理量上给出一致诊断。

第七步把理论量连接到观测诊断。一个 prompt 事件通常要同时检查：T90 和更短的脉冲宽度给出的时间尺度；fluence 和 peak flux 给出的能量与光度尺度；Band 参数给出的 \(E_pk\)、α、β；最高能光子及其到达时间给出的 γγ 透明性约束；若有红移，还可计算 \(E_pk,z=(1+z)E_pk\)、\(E_iso\) 和 \(L_iso\)。由这些量可以初步判断事件是否需要很高 Γ、是否有异常硬低能谱、是否存在额外高能成分、是否适合与 Amati/Yonetoku 类经验关系比较。

第八步说明本讲与后续的分工。到这里我们已经具备读 prompt 经典论文的最低工具：知道 compactness problem 为什么要求 relativistic outflow，知道 Band 函数如何描述 BATSE/GBM 谱，知道 \(E_pk\) 与 α、β 的观测含义，也知道这些量的常见陷阱。下一讲可以在此基础上讨论 internal shock、photosphere 和磁耗散如何产生不同谱形与效率；再下一讲再进入时间分辨谱、模型比较、同步辐射限制和多信使/高能延迟等前沿问题。

公式与推导

第一步：先从观测量定义各向同性等效光度。假设 burst 有红移 \(z\)、光度距离 \(d_L\)，观测到能量通量 \(F\) 或峰值通量 \(F_{\rm pk}\)，则

\[L_{\rm iso}=4\pi d_L^2 F_{\rm bol},\qquad E_{\rm iso}=\frac{4\pi d_L^2 S_{\rm bol}}{1+z}.\]

这里 \(F_{\rm bol}\) 和 \(S_{\rm bol}\) 是经过能段外推与 K-correction 后的 bolometric flux 与 fluence。这个量不是喷流真实能量，而是把源当作各向同性时的等效强度；它将进入 compactness 估计，因为光度越大，同一体积内的光子密度越高，\(\gamma\gamma\) 吸收越严重。

第二步：用最短变时标给出非相对论紧凑源的大小估计。若先暂时假设源区静止或低速运动，观测到的快速起伏 \(\delta t\) 受因果性限制为

\[R_{\rm nr}\lesssim \frac{c\,\delta t}{1+z}.\]

这一步解出的是源区半径的上限。对于毫秒到秒的 prompt 结构，\(R_{\rm nr}\) 很小；与第一步的巨大 \(L_{\rm iso}\) 结合，就会推出极高的光子密度。这正是 compactness problem 的入口。

第三步：估计静止源中的光子数密度。若典型光子能量为 \(\langle E\rangle\)，在半径 \(R\) 内近似各向同性辐射，则

\[n_\gamma \sim \frac{L_{\rm iso}}{4\pi R^2 c\,\langle E\rangle}.\]

这里求得的是实验室系或源静止系的光子数密度数量级。它不是精确谱积分，但足以显示矛盾：\(R\) 越小、\(L_{\rm iso}\) 越大，\(n_\gamma\) 越高；下一步将用它计算高能光子的 pair-production 光学深度。

第四步：写出 \(\gamma\gamma\to e^+e^-\) 阈值条件。两光子能量分别为 \(E_1,E_2\)，夹角为 \(\theta\)，则产生电子对要求

\[E_1E_2(1-\cos\theta)\ge 2(m_ec^2)^2.\]

这个式子说明：一个观测到的高能光子 \(E_1\) 只要遇到足够多的靶光子 \(E_2\)，就会被吸收。若光子场近似各向同性，\((1-\cos\theta)\sim 1\)，MeV–GeV 光子在 GRB 源区中很容易满足阈值；因此是否能逃逸取决于下一步的光学深度。

第五步：给出非相对论 compactness 的光学深度估计。pair-production 截面在峰值附近为 \(\sim 0.1\sigma_T\)，数量级上可写为

\[\tau_{\gamma\gamma}^{\rm nr}\sim n_\gamma\sigma_{\gamma\gamma}R \sim \frac{\sigma_{\gamma\gamma}L_{\rm iso}}{4\pi R c\,\langle E\rangle}.\]

代入 \(R\lesssim c\delta t/(1+z)\) 后，典型 cosmological GRB 会得到 \(\tau_{\gamma\gamma}^{\rm nr}\gg 1\)。这意味着若源不相对论运动，MeV 以上非热光子不应直接逃出，谱也会趋向热化；与 BATSE/Fermi 看到的非热 Band 高能尾冲突。因此下一步必须引入 bulk Lorentz factor。

第六步：相对论运动首先改变时间尺度与发射半径。假设辐射区以 Lorentz factor \(\Gamma\) 朝向观测者运动，脉冲宽度主要由角延迟或壳层时间尺度给出，则

\[R\sim \frac{2\Gamma^2 c\delta t}{1+z}.\]

这一步解出的是 prompt 发射半径的常用数量级，而不是严格定理。它的物理含义是：相对论 beaming 使观测者只看到角度 \(\sim\Gamma^{-1}\) 的区域，因此同样的 \(\delta t\) 可以对应比非相对论估计大 \(\sim2\Gamma^2\) 的半径。半径变大直接降低光子密度。

第七步：相对论运动还改变共动系中的能量密度和光子能量。多普勒因子 \(\mathcal{D}\simeq\Gamma\) 时，观测能量与共动能量满足

\[E'\simeq \frac{(1+z)E}{\Gamma},\qquad U'_\gamma \sim \frac{L_{\rm iso}}{4\pi R^2 c\Gamma^2}.\]

这里撇号表示外流共动系。第一项降低了 \(\gamma\gamma\) 阈值问题中的光子能量，第二项降低了共动辐射能量密度。也就是说，大 \(\Gamma\) 同时使靶光子更稀、更软，使源更透明。

第八步：把高能谱形放入透明性估计。若在 Band 高能端可近似写成光子数谱

\[N(E)=N_0\left(\frac{E}{E_0}\right)^\beta,\qquad \beta<-1,\]

则对一个观测高能光子 \(E_{\max}\)，能与之成对产生的靶光子阈值约为

\[E_{\rm th}\sim \frac{\Gamma^2(m_ec^2)^2}{(1+z)^2E_{\max}}.\]

这一步把谱拟合参数 \(\beta\) 与最高能光子 \(E_{\max}\) 接上：\(\beta\) 决定高于 \(E_{\rm th}\) 的靶光子有多少，\(E_{\max}\) 越高则透明性约束越强。

第九步：写出含 \(\Gamma\) 的 \(\gamma\gamma\) 光学深度数量级。把半径、共动能量密度和高能谱积分合并，可得到常用形式

\[\tau_{\gamma\gamma}(E_{\max})\sim \frac{\sigma_T L_{>E_{\rm th}}}{16\pi R c\Gamma^2 E_{\rm th}} \lesssim 1.\]

这里 \(L_{>E_{\rm th}}\) 是靶光子阈值以上的等效光度，实际读论文时通常由 Band 或 cutoff power-law 谱积分得到。这个不等式给出的不是 \(\Gamma\) 的测量值，而是透明性下限 \(\Gamma_{\min}\)。

第十步：把不等式化为常见的 Lorentz factor 下限形式。对于高能光子谱 \(N(E)\propto E^\beta\)，透明性约束通常可整理为

\[\Gamma_{\min}\propto \left[ \frac{\sigma_T d_L^2 F_{E}}{c^2\delta t} \left(\frac{E_{\max}}{m_ec^2}\right)^{-1-\beta} (1+z)^{-2\beta-2} \right]^{\frac{1}{2(1-\beta)}}.\]

比例系数依赖谱归一化、积分上下限和精确截面。这个式子最重要的课程信息是依赖关系：更短的 \(\delta t\)、更强的高能通量、更高的 \(E_{\max}\) 都会提高 \(\Gamma_{\min}\)；而谱指数 \(\beta\) 的不确定会显著影响结果。

第十一步：转入 Band 函数本身，建立经验谱参数与 \(E_{\rm pk}\) 的关系。Band 光子谱可写为

\[N(E)=A\left(\frac{E}{E_{\rm piv}}\right)^\alpha e^{-E/E_0},\quad E
\[N(E)=A\left[\frac{(\alpha-\beta)E_0}{E_{\rm piv}}\right]^{\alpha-\beta}e^{\beta-\alpha}\left(\frac{E}{E_{\rm piv}}\right)^\beta,\quad E\ge E_b,\]

其中拼接能量为

\[E_b=(\alpha-\beta)E_0.\]

低能段是指数截断幂律，高能段是幂律，归一化因子保证函数值和斜率连续。\(\alpha,\beta\) 是 photon index，不是 \(\nu F_\nu\) 指数。

第十二步：求 \(\nu F_\nu\) 峰值并连接到观测诊断。因为每对数能段能量输出为

\[\nu F_\nu \propto E^2N(E),\]

在低能 Band 段对 \(E^2N(E)\) 求导，得到

\[\frac{d}{dE}\left(E^{\alpha+2}e^{-E/E_0}\right)=0 \quad\Rightarrow\quad E_{\rm pk}=(2+\alpha)E_0,\qquad \alpha>-2.\]

若同时 \(\beta<-2\)，则高能端能量积分收敛，\(E_{\rm pk}\) 是清晰的能量输出峰；若 \(\beta\ge -2\)，高能能量对探测器上限和谱外推高度敏感。实际读 prompt 论文时，\(E_{\rm pk}\)、\(\alpha\)、\(\beta\)、\(\delta t\)、\(E_{\max}\) 和红移共同决定两类诊断：一类是经验谱比较，另一类是 \(\tau_{\gamma\gamma}<1\) 给出的 \(\Gamma_{\min}\)。这就是本讲从 BATSE Band 谱走向相对论火球透明性约束的完整推导链。

经典拟合 / 应用方法

第一步要把“拟合 prompt 光谱”理解成一个前向建模问题，而不是把探测器计数直接画成物理谱。BATSE、GBM、Konus-Wind 或 Swift/BAT 记录的是能道中的计数 \(C_i\)，真实光子谱 \(N(E)\) 先要乘以有效面积、能量再分布和探测效率，经过响应矩阵 \(R_i(E)\) 折叠成预测计数：\(\mu_i(\theta)=t_{\rm exp}\int R_i(E)N(E;\theta)\,dE+B_i\)。这里 \(\theta\) 可以是 Band 模型的 \(A,\alpha,\beta,E_0\) 或 \(E_{\rm pk}\)，也可以是 cutoff power law 的 \(A,\alpha,E_{\rm cut}\)，\(B_i\) 是背景模型在该时间片的预测计数。课程中最重要的观念是：拟合对象通常是计数空间的 \(\mu_i\) 与 \(C_i\)，而不是已经“反卷积”后的图上点；图上 unfolded spectrum 只是可视化，不应作为最终统计推断的依据。

第二步是明确模型参数的物理含义和退化关系。Band 函数的核心参数是归一化 \(A\)、低能 photon index \(\alpha\)、高能 photon index \(\beta\)、特征能量 \(E_0\)，常报告的 \(\nu F_\nu\) 峰值为 \(E_{\rm pk}=(2+\alpha)E_0\)。若高能端计数少，\(\beta\) 会变得很不稳定，甚至只给出上限或贴近参数边界；这时 cutoff power law 或 Comptonized model 可能给出更可靠的 \(E_{\rm pk}\)，但无法约束真正的高能幂律尾。若探测器低能阈值接近 \(E_{\rm pk}\)，\(\alpha\) 与 \(E_{\rm pk}\) 会强相关；若 \(E_{\rm pk}\) 超出能段，Band 参数会更多反映仪器覆盖而非源本身。读表格时不要只看 best-fit 数字，而要看参数协方差、置信区间是否非对称、\(E_{\rm pk}\) 是否处在有效能段内部、\(\beta\) 是否由真实高能数据约束。

第三步是选择合适的似然或统计量。对于高计数、背景近似高斯的能道，可以使用 \(\chi^2\) 或 PG-stat 的近似；对于低计数伽马射线数据，更常见的是 Poisson 计数似然，例如每个能道 \(C_i\) 服从 \({\rm Poisson}(\mu_i)\)，对数似然为 \(\ln L=\sum_i[C_i\ln\mu_i-\mu_i-\ln(C_i!)]\)。如果背景不是已知常数，而是由 burst 前后时间段拟合出来，应该把背景不确定性传播到总似然中，或使用适合源区 Poisson、背景高斯/Poisson 的统计量。很多 prompt 论文报告 C-stat、PG-stat、BIC、AIC 或 likelihood ratio；读者要检查它们是否用于嵌套模型、参数是否在边界上、额外成分的显著性是否经过模拟校准。

第四步是从频率学置信区间或贝叶斯后验中理解“不确定性”。频率学做法常给出 ΔC-stat、Δχ² 对应的参数区间；贝叶斯做法则给出 p(θ|data)∝L(data|θ)p(θ)，并通过 MCMC 或 nested sampling 获得后验。对 Band 拟合而言，后验常常不是椭圆形：α–\(E_pk\)、β–归一化、背景–高能尾之间会有强相关。若论文只报告对称误差 ±σ，读者应怀疑是否掩盖了非高斯后验；若 β 的后验很宽，任何依赖高能外推的能量、γγ opacity 或 Γ_min 都会被系统性放大。一个好的分析应展示 corner plot、二维置信轮廓，至少说明参数是否被能段边界截断。

第五步是残差诊断。看 prompt 光谱拟合不能只看 reduced χ² 或 C-stat/dof，因为系统残差可能在总体统计量看似可接受时仍然明显。需要检查 residuals 随能量是否呈随机散布：低能端连续偏正可能暗示额外吸收、低能校准或模型低能斜率不对；\(E_pk\) 附近的波浪状残差可能暗示单一 Band 太宽或太窄；高能端连续正残差可能意味着额外 power-law 成分或 LAT 相关高能尾；某个能道附近尖锐结构则更可能是响应矩阵、背景线或坏道问题。残差还要按时间片检查，因为 time-integrated residual 可能被多个演化谱平均掉。

第六步是时间分辨拟合与时间积分拟合的区别。Band 1993 经典工作建立了总体经验谱语言，但现代读法必须把一个 burst 拆成多个时间片或脉冲相位：上升段、峰值段、衰减段的 \(E_pk\)、α、通量可能有 hard-to-soft 或 intensity-tracking 演化。若把这些谱直接积分，一个瞬时窄谱会被混合成宽谱，α、β 也可能被人为改变。因此读论文时要检查时间片如何定义：固定信噪比、Bayesian blocks、脉冲分解还是固定时间宽度；不同划分是否给出一致的 \(E_pk\) 演化；最短变时标 δt 是否与用于 Γ_min 的时间片一致。

第七步是系统误差。prompt 谱常见系统误差包括：背景多项式阶数选择、地球遮挡和粒子背景变化、探测器入射角响应、能量标定、不同探测器交叉归一化、触发阈值导致的样本选择、红移缺失造成的亮源偏置，以及只在有限能段内拟合却外推到 bolometric 能段。Swift/BAT 的窄能段尤其容易把 \(E_pk\) 推到能段外；GBM NaI 与 BGO 联合拟合要注意交叉校准；LAT 高能光子若到达时间晚于 MeV prompt 峰，其是否与同一区域发射会影响 γγ 透明性解释。凡是论文把 Band 参数用于 Amati/Yonetoku 关系、\(E_iso\)、\(L_iso\) 或 Γ_min，都应检查这些系统误差是否进入误差预算。

第八步是应用层面的判断。Band 拟合本身不给出唯一辐射机制，但它提供了机制筛选的共同坐标：α 是否硬于简单 optically thin synchrotron 的 -2/3 “line of death”；β 是否足够陡使能量收敛；\(E_pk\) 是否随通量演化；谱宽是否比单温黑体宽、比标准同步谱窄；是否需要黑体加非热成分；高能光子和 β 是否支持强 Γ_min。课程中要把拟合结果当作诊断工具，而不是最终答案：先问模型是否统计上足够、参数是否被数据真正约束、残差是否显示多成分，再讨论 photosphere、synchrotron、Compton 化、磁耗散或 internal shock 的物理解释。

重点章节 / 结果段落怎么读

读 Band et al. 1993 这类经典 prompt 光谱论文时，第一段应先读“数据与样本如何进入问题”。BATSE 的价值不仅是单个 burst 的漂亮谱，而是用相对统一的仪器和能段处理一批 GRB，使人们第一次能系统比较 α、β、\(E_pk\) 的分布。读者要关注样本选择：是否只包含足够亮、足够高信噪比的事件；时间积分谱还是峰值谱；能段覆盖到哪里；背景怎样选取。很多后来关于 \(E_pk\) 分布或谱指数分布的结论，都受这些入口条件影响。

第二段读“模型定义”，尤其是 Band 函数为什么这样写。它不是任意拼接两条线，而是低能指数截断幂律与高能幂律的平滑连接，函数值和斜率在 break 处连续。课程读法是先确认 N(E) 是 photon spectrum，再把它转换成 \(E^2N(E)\) 看能量峰；随后检查论文中使用的是 E0、break energy 还是 \(E_pk\)。不同论文记号不同，但物理对应关系必须理清，否则会把谱断点、指数截断能量和 νFν 峰值混为一谈。

第三段读“拟合方法”。经典 BATSE 论文已经强调经验模型对计数数据的适配，现代读者要主动补上响应矩阵折叠、背景、统计量和参数误差的概念。重点不是背诵某个 best-fit α，而是理解：为什么一个 burst 可以被 Band 函数描述，为什么某些 burst 只需要 cutoff power law，为什么 β 在高能计数不足时难以确定。读论文的表格时要把参数误差、自由度、拟合统计量和能段一起看，不能把不同仪器、不同时间片、不同模型下的 α 直接横向比较。

第四段读“结果分布”。Band 参数分布的经典意义在于把 prompt 光谱变成可统计讨论的参数空间：α 常在 -1 左右，β 常在 -2 到 -3，\(E_pk\) 从几十 keV 到 MeV 量级不等。课程中应把这些数字当作经验事实的起点，而非辐射机制的终点。要问：分布宽度是真实物理差异、时间演化混合、还是选择效应？β<-2 的事件能量积分较稳定，β 接近 -2 的事件对高能外推非常敏感；α 的分布若经常硬于同步辐射简单预期，就会引出后续“line-of-death”和 photosphere 讨论。

第五段读 compactness 与相对论火球经典工作的“问题设置”。Goodman、Paczynski、Shemi 与 Piran 相关工作的核心不是某个细节公式，而是把短变时标、高光度、高能非热光子三者放在一起，说明静止紧凑源会因 γγ→e± 而不透明。读这些段落时要抓住逻辑链：观测变时标给半径，流量与距离给光度，光度与半径给光子密度，光子密度给 pair opacity，τγγ≫1 与观测透明性矛盾，于是需要 Γ≫1。这个推理早于许多具体 prompt 辐射模型，因此是系列课程的底层边界条件。

第六段读“限制与假设”。经典 compactness 估计通常假设一区、近似各向同性、同一时间同一区域产生 MeV 与更高能光子，并用观测谱外推估计靶光子数。现代论文可能考虑角向结构、多区发射、高能延迟、磁化喷流或 photosphere，导致 Γ_min 改变。课程读法不是否定经典结果，而是区分稳健结论与模型依赖：稳健的是 GRB prompt 需要相对论外流；较依赖模型的是某个事件精确的 Γ_min 数值、发射半径和高能光子是否同源。

第七段读“经典工作如何连接后续”。Band 谱提供经验坐标，compactness 提供动力学底线，两者合起来规定了后续模型的舞台：任何 internal shock、magnetic reconnection、photospheric emission 或 synchrotron 模型，都必须在高 Γ、快速变时标、keV–MeV 峰值、非热尾和复杂时间演化中自洽。第 1 讲读经典工作的目标不是立即选择机制，而是训练学生在每篇 prompt 论文中先问：数据如何变成谱参数？谱参数如何进入透明性和能量估计？哪些结论来自观测，哪些来自模型假设？

建议重点查看的图表

  1. 光变曲线与 T90/脉冲结构图：看不同能段的计数率随时间变化，确认 burst 的总持续时间、最短可辨变时标 δt、峰值时间、硬软能段是否同步。不要把 T90 直接当作 compactness 中的 δt；Γ_min 往往更敏感于最短脉冲或最高能光子所在时间片。

  2. 计数谱与响应折叠模型图：看横轴是探测器能道还是能量，纵轴是计数率、counts s^-1 keV^-1 还是 unfolded flux。关键是模型线是否在计数空间与数据比较，残差是否随机。不要把 unfolded 图上的点当成独立观测的真实 photon spectrum。

  3. νFν 或 \(E^2N(E)\) 光谱图：看 \(E_pk\) 是否处于探测器有效能段内部，峰形是窄、宽还是多峰，Band、cutoff power law、blackbody+power law 等模型在峰附近有何差异。若峰靠近能段边界，\(E_pk\) 和 α/β 的可信度要降低。

  4. Band 参数分布图：包括 α、β、\(E_pk\) 的直方图或散点图。看分布中心、宽度、截断、离群点以及选择效应；尤其检查 α>-2/3 或 β>-2 的事件是否真实受约束，还是由低信噪比、能段不足或模型退化造成。

  5. 参数置信区间或 posterior corner plot：看 α–\(E_pk\)、β–归一化、背景–高能尾之间的相关性。单个 best-fit 点意义有限，二维轮廓能告诉我们 \(E_pk\) 是否被测到、β 是否只是下限、额外成分是否与 Band 参数退化。

  6. 时间分辨谱演化图：看 \(E_pk\)、α、β、通量随时间或脉冲相位变化。重点区分 hard-to-soft 演化和 intensity-tracking 演化；检查时间积分谱是否可能由多个瞬时谱叠加形成。

  7. γγ opacity / Γ_min 诊断图：通常画 Γ_min 随 δt、\(E_max\)、β 或光度变化，或给出最高能光子的到达时间与透明性约束。看高能光子是否与 MeV 发射同一时间片、同一区域假设是否合理，Γ_min 是下限而不是直接测量值。

  8. 经验关系图，如 \(E_pk,z\)–\(E_iso\) 或 \(E_pk,z\)–\(L_iso\)：看事件是否落在 Amati/Yonetoku 类关系附近，误差条是否包括红移、K-correction 和谱参数不确定性。不要把相关关系自动解释为辐射机制；触发阈值、样本截断和红移不完备可能塑造斜率与散布。

关键图表逐图导读

图 1：多能段 prompt 光变曲线。横轴通常是触发后时间，纵轴是不同能段的计数率或背景扣除计数率。先看 T90，再看更短的脉冲宽度、上升/下降时间和能量依赖：硬能段是否更早、更窄，软能段是否拖尾。模型线若出现，常是脉冲分解或背景模型，不是辐射物理模型。残差要看背景扣除后是否有系统起伏，尤其在 burst 前后。常见误读是把整个 T90 当作发射半径估计中的 δt；实际 compactness 约束常由最短显著变时标和最高能光子所在时间片控制。

图 2：Band 函数在 photon spectrum 或 unfolded spectrum 上的拟合。横轴是能量，纵轴可能是 photons cm^-2 s^-1 keV^-1，也可能是能量通量或 unfolded νFν。要先确认图中点是否经过模型依赖的反卷积；如果是 unfolded spectrum，它主要用于视觉展示。模型线通常包括 Band、cutoff power law 或 power law，重点看 \(E_pk\) 附近曲率和高能尾是否被数据覆盖。残差面板若在低能端连续偏离，说明 α 或校准可能有问题；若高能端连续正残差，可能需要额外成分。误读风险是把图上最高点当作 \(E_pk\)，而不区分 photon spectrum 峰与 \(E^2N(E)\) 峰。

图 3：νFν 谱峰图。横轴为能量，纵轴为 \(E^2N(E)\) 或 νFν，Band 模型线的最高点就是 \(E_pk\)。数据点应在峰两侧都有覆盖，才能说峰值被测定；若只看到上升段或下降段，\(E_pk\) 可能只是模型外推。比较 Band 与 cutoff power law 时，要看高能侧是否确有幂律尾，还是指数截断已足够。残差若在峰附近呈“窄凸起”，可能暗示 photospheric 成分；若单一 Band 太宽或太窄，说明多成分或时间叠加值得考虑。误读风险是把一个统计上可接受的 Band 拟合解释为唯一物理机制。

图 4：α–\(E_pk\) 或 α 分布图。横轴可为 \(E_pk\)，纵轴为 α，或直接画 α 的直方图。要看 α 是否集中在 -1 附近，有多少事件超过同步辐射简单慢冷极限 -2/3 或快冷预期 -3/2。模型线可能标出 synchrotron line-of-death 或样本中位数。检查误差条很关键：低信噪比或 \(E_pk\) 接近低能阈值时，α 可能系统偏硬或偏软。误读风险是把所有 α>-2/3 的点都当成同步辐射失败的铁证；实际还要看时间分辨、能段、黑体成分和参数后验。

图 5：β 分布或高能尾诊断图。横轴可能为 β，或 β 与 fluence/\(E_pk\) 的散点。要看 β 是否小于 -2，因为这决定 νFν 高能积分是否收敛；β 接近 -2 的事件对高能外推和 bolometric flux 特别敏感。模型线若给出 Band 与 cutoff power law 的比较，要看加入 β 是否真正改善似然，还是高能端计数不足导致参数漂移。残差若在 BGO/LAT 能段持续正偏，说明单一 Band 可能漏掉高能成分。误读风险是把一个未约束的 β 当作真实高能谱指数，再用于 Γ_min 或总能量估计。

图 6：时间分辨 \(E_pk\) 与通量演化图。横轴为时间或脉冲相位，纵轴为 \(E_pk\)、α 或能量通量，常见画法是多面板同步展示。要判断 \(E_pk\) 是随时间单调变软，还是随光度上升下降而追踪强度。模型线可能是经验相关 \(E_pk\)∝F^η 或脉冲模型。残差与误差条能显示是否每个时间片都真正约束 \(E_pk\)。误读风险是把 time-integrated \(E_pk\) 当作瞬时物理峰；若存在强烈谱演化，积分谱的宽度和指数可能只是混合结果。

图 7：Γ_min 或 γγ opacity 约束图。横轴可能是 Lorentz factor Γ、最高能光子能量 \(E_max\)、变时标 δt 或时间，纵轴为 τγγ 或允许/排除区域。要看 τγγ=1 的交点如何给出 Γ_min，以及输入的 Band 参数、红移、最高能光子时间片是否明确。模型线通常对应不同 δt、β 或发射半径假设。误读风险是把 Γ_min 当作 Γ 的测量值；它只是透明性下限，而且若高能光子来自不同区域或延迟成分，一区同源假设会改变结论。

后续阅读路径

从本讲的经典工作继续读，第一条路径是沿着“经验谱目录”向现代样本扩展。Band et al. 1993 给出共同语言后，可以读 BATSE spectral catalog，再读 Fermi/GBM 的 time-integrated 与 time-resolved spectral catalogs，比较 α、β、\(E_pk\) 在不同仪器和样本中的变化。阅读重点不是记住目录中位数，而是学习现代目录如何处理响应、背景、模型选择、参数截断和 selection effects。这样可以看清哪些所谓典型 prompt 性质是稳健的，哪些依赖探测器能段和触发阈值。

第二条路径是沿着 compactness problem 走向高能 GRB。Goodman、Paczynski、Shemi 与 Piran 的透明性论证可以自然连接到 EGRET、AGILE、Fermi/LAT 对 GeV 光子的探测。现代问题包括：最高能光子给出的 Γ_min 有多强；GeV 发射为何常延迟于 MeV prompt；高能成分是否与 Band 成分同源；如果不是同源，γγ opacity 约束如何修改。读这条线时应特别关注每篇文章对 δt、发射半径、光子场角分布和时间片同源性的假设。

第三条路径是从 Band 参数走向辐射机制判别。先读 optically thin synchrotron 对 α、谱宽和冷却斜率的预测，再读 photospheric emission、subphotospheric dissipation、Comptonization、magnetic reconnection 和 hybrid models。这里要保持本讲建立的纪律：不要因为某个模型能拟合一个 νFν 峰就宣布胜利，而要看它是否同时解释 \(E_pk\) 分布、α 的硬度、β 的高能尾、时间演化、辐射效率和 Γ/R 约束。

第四条路径是读时间分辨与多成分分析。现代 prompt 研究越来越少满足于单个 time-integrated Band 拟合，而是用 Bayesian blocks、脉冲分解、层级贝叶斯样本建模、physical synchrotron templates 或 blackbody+nonthermal decompositions。沿这条线读论文时，要检查额外成分显著性是否经过模拟，参数是否因时间分箱改变，残差改善是否局限于某个能段，以及物理解释是否与 light curve morphology 一致。

第五条路径是把 prompt 与 afterglow 重新接起来，但不要混淆阶段。afterglow 给出的初始 Lorentz factor、喷流开角、外部密度和动能，可以与 prompt 的 Γ_min、\(E_iso\)、\(L_iso\)、\(E_pk\) 比较，形成能量预算和喷流结构图像。阅读时要区分：prompt compactness 给的是早期透明性下限，afterglow onset 或 deceleration time 给的是外激波动力学估计；二者若不一致，可能提示多区发射、能量注入、磁化或观测选择效应。

下一讲建议

  1. 下一讲最自然的方向是“prompt 辐射区与能量耗散机制：internal shock、photosphere 与磁耗散的比较”。本讲已经建立了 Γ_min、R≈2Γ²cδt、Band 参数和 \(E_pk\) 的基础；下一讲可以问在这些约束下，壳层碰撞、光球释放和磁重联分别在哪里发生、效率如何、能否产生观测到的谱宽和时间结构。边界是：下一讲开始讨论物理模型，但仍不应把某一机制预设为正确。

  2. 另一个方向是“同步辐射 line-of-death 与 prompt 谱形问题”。本讲只说明 α、β、\(E_pk\) 如何测量，以及 α 的异常硬度为什么重要；下一讲或第三讲可以系统推导同步辐射低能斜率、冷却谱、衰减磁场、持续加热和边缘效应，并与 Band 参数分布比较。边界是：这属于辐射微物理判别，不再只是经验谱语言。

  3. 第三讲可进入“时间分辨谱、GeV 延迟与多成分 prompt”。本讲已提醒 time-integrated spectrum 的陷阱和 γγ opacity 的一区假设；后续可以专门讨论 Bayesian blocks、脉冲相位谱演化、LAT 高能光子、额外 power-law 成分、photospheric bump 和多区发射。它与本讲的关系是从基础诊断走向前沿模型比较，从单一 Band 拟合走向动态、多成分、跨能段的 prompt 图像。
