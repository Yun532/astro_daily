经典工作详细讲解

本讲是“GRB 瞬时辐射经典课程：谱、紧致性与喷流物理”的第一讲，没有上一讲需要承接；它承担的是为后两讲建立共同语言的任务。我们暂时不问 Band 谱究竟来自同步辐射、光球还是磁化耗散，也不讨论内部激波、磁重联、偏振和喷流组成，而是先回答一个更基础的问题：观测者为什么会相信 GRB 是宇宙学距离上的高光度事件？如果它们确实在宇宙学距离，为什么短到毫秒—秒级的变时和 MeV 光子并没有让源被自身的 γγ→e^+e^- 湮灭完全遮蔽？这一讲的核心结论是：相对论性膨胀不是 GRB 模型的装饰性假设，而是由透明性要求直接推出的门槛条件。

历史动机首先来自 BATSE。早期 GRB 的距离尺度并不清楚，太阳系、银河盘、银河晕和宇宙学模型都曾被认真讨论。BATSE 的经典贡献不是给单个暴提供精确红移，而是用全天大样本改变了距离尺度判断：GRB 在天空上近似各向同性，不追踪银河盘；同时累计亮度分布 logN-logP 或 \(V/V_{\max}\) 又偏离均匀欧几里得空间中 \(N(>P)\propto P^{-3/2}\) 的简单预期，表现为弱源相对亏缺。各向同性本身不能唯一证明宇宙学起源，因为巨大银河晕也能给出近似各向同性；但“各向同性 + 非欧几里得亮度分布”的组合使宇宙学尺度成为越来越自然的解释。后来光学余辉和红移测量坐实了这一点，但 compactness problem 在红移确认前已经显示出强烈物理约束意义。

一旦接受宇宙学距离，观测到的通量就变成巨大各向同性等效光度。给定能量通量 \(F\) 和 luminosity distance \(d_L\)，常写 \(L_{\rm iso}=4\pi d_L^2F\,k\)，其中 \(k\) 表示把观测能段换算到源静止系指定能段的修正。典型 GRB prompt emission 的 \(L_{\rm iso}\) 可达 \(10^{51}-10^{53}\,{\rm erg\,s^{-1}}\)，而光变时间尺度 \(\delta t\) 可短到毫秒到秒。如果用非相对论因果直觉估计发射区大小，\(R_{\rm nr}\lesssim c\delta t/(1+z)\)，那么巨大光度被压在极小区域内，源内 MeV 光子数密度极高。对一个高能光子而言，它会穿过充满目标光子的辐射场，并通过 \(\gamma\gamma\to e^+e^-\) 产生电子正电子对；数量级上 \(\tau_{\gamma\gamma}\sim n_\gamma\sigma_{\gamma\gamma}R\)，常远大于 1。这就是 compactness disaster：静止或非相对论源应当高度不透明，观测者不应看到明亮的非热 MeV 谱。

为了把这个矛盾讲清楚，本讲先建立 Band function 这套经验观测语言。Band 谱定义在光子数谱 \(N(E)=dN/(dE\,dA\,dt)\) 上，而不是直接定义在 \(\nu F_\nu\) 上。低能段是带指数截断的幂律 \(N(E)=A(E/100\,{\rm keV})^\alpha \exp(-E/E_0)\)，高能段是与低能段平滑连接的幂律，断裂能量 \(E_b=(\alpha-\beta)E_0\)；\(\nu F_\nu=E^2N(E)\) 的峰值为 \(E_{\rm peak}=(2+\alpha)E_0\)，条件是峰位落在低能截断段且 \(\alpha>-2\)。这里必须反复强调：\(\alpha,\beta\) 是 photon spectrum 的指数，不是 \(\nu F_\nu\) 斜率；\(\nu F_\nu\) 的低能和高能渐近斜率分别是 \(\alpha+2\) 和 \(\beta+2\)。Band function 的作用是给观测建立坐标系，而不是提前承诺某一种辐射机制。

有了谱和光度，紧致性推导就成为一条清晰的物理链。第一，短变时在非相对论源中给出小半径；第二，巨大 \(L_{\rm iso}\) 和小 \(R\) 给出高光子数密度 \(n_\gamma\sim L_{\rm iso}/(4\pi R^2c\langle E\rangle)\)；第三，\(\gamma\gamma\) 阈值 \(E_1E_2(1-\cos\theta)\ge2(m_ec^2)^2\) 说明 MeV 光子之间可以有效产生对；第四，取 \(\sigma_{\gamma\gamma}\sim0.1\sigma_T\) 得到 \(\tau_{\gamma\gamma}\sim L_{\rm iso}\sigma_{\gamma\gamma}/(4\pi Rc\langle E\rangle)\)，代入 \(R_{\rm nr}\) 后通常 \(\tau_{\gamma\gamma}\gg1\)。若源真是静止紧致源，它更可能形成热化 pair fireball，而不是让非热 Band 谱逃逸。这一矛盾不依赖于是否看到 GeV 光子；GeV 光子会显著加强约束，但 MeV 光子本身已经足够严重。

相对论性膨胀解除矛盾的方式不是单一修正，而是半径、能量和角度的三重修正。若发射区以 Lorentz factor \(\Gamma\) 指向观测者运动，同样的观测变时对应 \(R\simeq2\Gamma^2c\delta t/(1+z)\)，比静止估计大 \(\Gamma^2\) 量级；观测能量在共动系降低为 \(E'\simeq E(1+z)/\Gamma\)；pair-production 阈值在观测系随 \(\Gamma^2\) 提高，\(E_t\sim\Gamma^2(m_ec^2)^2/[E_{\max}(1+z)^2]\)，使最高能光子需要寻找更高能、更稀少的目标光子。若高能尾近似 \(N(E)\propto E^\beta\)，则阈值以上目标光子数对 \(\Gamma\) 很敏感，常得到 \(\tau_{\gamma\gamma}\propto\Gamma^{-2(1-\beta)}\) 的陡降关系；当 \(\beta\simeq-2\) 时，\(\tau_{\gamma\gamma}\) 近似按 \(\Gamma^{-6}\) 下降。最终令 \(\tau_{\gamma\gamma}(E_{\max},\Gamma_{\min})=1\)，就得到 Lorentz factor 的透明性下限。

这里的 \(\Gamma_{\min}\) 必须被正确解释。它不是对真实 \(\Gamma\) 的精确测量，而是在特定模型假设下得到的下限：最高能光子和低能 Band 光子同区、共时、共空间产生；光子场在共动系近似各向同性；谱可以从观测能段合理外推到 \(\gamma\gamma\) 阈值所需能量；红移、\(\delta t\)、\(E_{\max}\) 的关联判断可靠。更严格的表达应写成 \(\tau_{\gamma\gamma}=\int dl'\int d\Omega'\int d\epsilon'\,n'(\epsilon',\Omega')(1-\cos\theta')\sigma_{\gamma\gamma}(s)\)，而课堂上的幂律公式只是帮助学生理解参数依赖和数量级。若 GeV 光子来自外激波或更大半径区域，单区 \(\Gamma_{\min}\) 的解释就必须修改。

本讲还要求学生把理论推导连接到真实数据分析。Band 参数不是从“已经去响应”的谱点上直接读出来的，而是通过 forward-folding 得到：入射光子谱 \(N(E;\theta)\) 经探测器响应矩阵 \(R_i(E)\) 折叠为预测计数 \(\mu_i=\int R_i(E)N(E;\theta)dE+B_i\)，再用 Poisson likelihood 或 Cash statistic 与观测计数 \(C_i\) 比较。读论文时要检查 Band、CPL、单幂律、黑体加幂律等模型之间的残差差异，检查 \(\beta\) 是否真正被约束，检查 \(E_{\rm peak}\) 是否落在灵敏能段内，并确认 compactness 分析是否把 \(A,\alpha,\beta,E_{\rm peak},z,\delta t,E_{\max}\) 的不确定性传播到 \(\Gamma_{\min}\) 后验中。尤其对于最高能单光子，还要问它是否与低能 prompt emission 共时、是否有显著背景概率、是否存在高能延迟。

关键图像阅读贯穿本讲。BATSE 天空分布图训练学生看各向同性和曝光选择，而不是把局部空洞误读为物理结构；logN-logP 图训练学生比较强源端 \(-3/2\) 斜率、弱源端变平和完整性阈值；\(\nu F_\nu\) Band 谱图训练学生分清 photon index 与能量谱斜率；时间分辨谱演化图训练学生区分 hard-to-soft 与 intensity tracking；\(\tau_{\gamma\gamma}(\Gamma)\) 曲线训练学生读出 \(\tau=1\) 交点并观察 \(\delta t,\beta,E_{\max},z\) 如何移动 \(\Gamma_{\min}\)；残差图则提醒学生，经验模型是否足够不能只看最佳拟合参数，必须看折叠计数谱和系统性残差。

本讲的后续影响在于，它把 GRB prompt emission 的研究从“看到一个短暂高能爆发”推进到“必须存在 \(\Gamma\simeq10^2-10^3\) 的相对论外流”。后两讲将沿用这里建立的 \(\alpha,\beta,E_{\rm peak},L_{\rm iso},\delta t,E_{\max},\Gamma_{\min}\) 语言，进一步追问微观辐射机制：下一讲将讨论 optically thin synchrotron、快冷问题和 line-of-death；第三讲再进入光球辐射、耗散变宽、多成分 prompt 谱和多区模型。本讲到透明性约束为止，故意不把 Band function 解释成同步辐射，也不把 \(\Gamma_{\min}\) 解释成喷流动力学的完整答案。

背景知识

GRB 的发现史本身就带有距离尺度之谜。20 世纪 60–70 年代，Vela 卫星最初为监测核试验而发现来自太空的短暂伽马射线闪光。早期观测没有 arcsecond 定位，没有宿主星系，没有光学余辉，也没有红移。因此研究者首先争论的不是“哪种辐射机制产生 Band 谱”，而是“这些爆发离我们有多远”。如果它们在太阳系或银河系近邻，能量需求相对温和；如果它们在宇宙学距离，释放能量和光子密度将极端巨大，物理图像会完全不同。
BATSE 搭载在 Compton Gamma Ray Observatory 上，以全天监测和大样本统计改变了这个问题的走向。若 GRB 是银河盘内源，天空分布应明显集中在银道面；BATSE 看到的却是近似各向同性分布。这个结果首先排除了简单薄盘模型，但还没有单独证明宇宙学距离，因为一个足够大的银河晕也可以在天空上近似各向同性。真正关键的是亮度统计：在均匀欧几里得空间中，如果源的空间密度和光度函数足够简单，累计亮度分布应近似满足 \(N(>P)\propto P^{-3/2}\)；BATSE 的弱源端相对这一预期变平，\(V/V_{\max}\) 统计也显示偏离。这提示样本正在接近某种尺度边界、宇宙学几何效应或源演化，而不只是本地均匀分布。

理解这一历史背景很重要，因为 compactness problem 的强度直接取决于距离尺度。通量是观测量，光度不是；只有当 GRB 被放到宇宙学距离上，同样的观测通量才意味着 \(L_{\rm iso}=4\pi d_L^2F\,k\) 这样巨大的各向同性等效光度。BATSE 时代尚未大规模获得红移，但统计证据已经把研究者推向宇宙学解释。1997 年以后，BeppoSAX 定位、余辉发现和宿主星系红移测量确认许多长暴位于宇宙学距离；随后短暴也被定位到外星系环境。于是，BATSE 统计给出的距离尺度线索和后来的红移证据共同使紧致性矛盾成为 GRB 物理的基本约束。

本讲为什么要把 Band function 放在 compactness 推导之前？因为 \(\gamma\gamma\) 光深不只依赖总光度，也依赖光子能量分布。Band function 提供了一套统一描述 prompt 谱的经验语言：归一化 \(A\)、低能 photon index \(\alpha\)、高能 photon index \(\beta\)、特征能量 \(E_0\) 或 \(E_{\rm peak}\)。它使我们能从观测计数出发估计能量通量、平均光子能量、高能尾目标光子数，以及最高能光子遇到的 pair-production 靶场。没有这套谱语言，compactness 推导就会停留在粗略光度估算，无法进入实际论文中的 \(\Gamma_{\min}\) 诊断。

一个常见误区是把 Band function 当成某种确定的物理机制。实际上 Band 谱是经验函数，不等同于同步辐射、光球辐射、Compton 化或磁重联辐射。一个 GRB 谱被 Band function 拟合得很好，只说明它在给定能段内可以由平滑断裂幂律近似描述；它不自动说明低能指数来自同步辐射，也不自动排除热成分或多成分叠加。后续课程会讨论不同辐射模型如何解释 \(\alpha,\beta,E_{\rm peak}\) 和谱宽，但第一讲必须保持边界：Band 是观测坐标系，不是终局答案。

另一个常见误区是认为 compactness problem 只在探测到 GeV 光子时才存在。事实上，\(\gamma\gamma\to e^+e^-\) 的阈值由 \(E_1E_2(1-\cos\theta)\ge2(m_ec^2)^2\) 决定；宇宙学 GRB 的 MeV 光子数密度极高，即使最高能光子只在 MeV 量级，非相对论紧致源也可能极端不透明。GeV 光子的重要性在于它们把目标光子阈值推到更清楚的 Band 高能尾区域，并使 \(\Gamma_{\min}\) 约束更强；但没有 GeV 探测并不等于没有透明性问题。

第三个误区是把观测到的短变时直接理解成静止源大小 R≈cδt，然后再把相对论运动看成可选补丁。正确的课程逻辑相反：如果源是非相对论的，短变时和宇宙学光度会导致 τ_{γγ}≫1，与观测到的非热透明谱矛盾；因此我们被迫引入相对论性外流。相对论性运动同时改变发射半径、共动系能量和光子碰撞角，这三者共同把光深压低。它不是在已有模型上加一个小修正，而是 GRB prompt emission 基本图像的入口。

还要警惕把 \(\Gamma_{\min}\) 当作“喷流速度的精确测量”。\(\Gamma_{\min}\) 来自透明性条件 \(\tau_{\gamma\gamma}(E_{\max},\Gamma_{\min})=1\)，它依赖单区、共时、谱外推、红移、\(\delta t\)、最高能光子关联概率和探测器响应。真实 \(\Gamma\) 可以大于 \(\Gamma_{\min}\)；如果最高能光子来自更大半径或外激波区域，单区 \(\Gamma_{\min}\) 还会改变。读论文时应该把 \(\Gamma_{\min}\) 看成模型相关但非常有力的下限，而不是像谱线红移那样直接测得的物理量。

本系列的连续性也在这里确定。第一讲只建立 BATSE 统计事实、Band 谱语言、非相对论 compactness disaster 和相对论透明性下限。第二讲才会从同一套 \(\alpha,\beta,E_{\rm peak}\) 出发讨论 optically thin synchrotron、快冷问题和 line-of-death；第三讲再讨论光球变宽、耗散、多成分和多区 prompt 模型。因此，本讲的背景不是为了穷尽 GRB 理论，而是为后续所有辐射机制比较设置共同的观测和动力学约束。

基础理论 / 方法脉络

第一层基础是观测量。GRB prompt emission 的原始数据不是一条“物理谱线”，而是不同探测器、不同能道中随时间变化的计数 \(C_i(t)\)。从这些计数中，我们关心光变曲线、峰值流量 \(P\)、总 fluence、最短显著变时 \(\delta t\)、时间积分和时间分辨能谱、最高能光子 \(E_{\max}\)，以及若能获得的红移 \(z\)。BATSE 主要在几十 keV 到数 MeV 能段建立了大样本统计；Fermi/GBM 与 LAT 则把低能 Band 光子场和高能 GeV 光子联系起来，使透明性约束更强也更复杂。

第二层基础是距离尺度和光度。BATSE 的天空各向同性与 logN-logP 非欧几里得偏离说明 GRB 不像简单银河盘源；后来的红移测量确认许多 GRB 位于宇宙学距离。给定观测能量通量 \(F\) 和 luminosity distance \(d_L(z)\)，各向同性等效光度写作 \(L_{\rm iso}=4\pi d_L^2F\,k\)。这里的“各向同性等效”并不表示真实喷流一定各向同性，而是观测者在不知道喷流开角或为了便于比较时使用的标准化量。compactness 问题正是从 \(L_{\rm iso}\) 很大、\(\delta t\) 很短这两个观测事实的组合中生长出来的。

第三层基础是光子数谱与能量谱的区分。观测 prompt 谱通常用光子数谱 \(N(E)=dN/(dE\,dA\,dt)\) 表示，单位可写为 \({\rm ph\,cm^{-2}\,s^{-1}\,keV^{-1}}\)；能量通量谱为 \(F_E=E\,N(E)\)，而常见的 \(\nu F_\nu\) 表示等价于 \(E^2N(E)\)。因此，如果 \(N(E)\propto E^\alpha\)，那么 \(\nu F_\nu\) 的斜率不是 \(\alpha\)，而是 \(\alpha+2\)。这个区分非常重要，因为 Band function 中的 \(\alpha\) 和 \(\beta\) 是 photon index；把它们误当成 \(\nu F_\nu\) 指数会导致对谱峰、能量输出和理论模型比较的系统性误读。

第四层基础是 Band function。低能段通常写为 \(N(E)=A(E/100\,{\rm keV})^\alpha \exp(-E/E_0)\)，高能段为与其平滑连接的幂律 \(N(E)\propto E^\beta\)，连接能量为 \(E_b=(\alpha-\beta)E_0\)。在峰值落在低能指数截断段且 \(\alpha>-2\) 时，\(\nu F_\nu\) 峰值能量为 \(E_{\rm peak}=(2+\alpha)E_0\)；若同时 \(\beta<-2\)，高能侧 \(\nu F_\nu\) 向下，\(E_{\rm peak}\) 可清楚解释为能量输出最集中的尺度。若 \(E_{\rm peak}\) 落在探测器能段之外，或 \(\beta\) 无法被约束，就不能把最佳拟合参数过度解释为稳固物理结论。

第五层基础是非相对论紧致性估算。假设源静止或非相对论运动，最短变时 \(\delta t\) 对应因果半径 \(R_{\rm nr}\lesssim c\delta t/(1+z)\)。在半径 \(R\) 内，若光度为 \(L_{\rm iso}\)、平均光子能量为 \(\langle E\rangle\)，源内光子数密度数量级为 \(n_\gamma\sim L_{\rm iso}/(4\pi R^2c\langle E\rangle)\)。一个高能光子穿过长度约 \(R\) 的辐射场，pair-production 光深约为 \(\tau_{\gamma\gamma}\sim n_\gamma\sigma_{\gamma\gamma}R\sim L_{\rm iso}\sigma_{\gamma\gamma}/(4\pi Rc\langle E\rangle)\)，其中 \(\sigma_{\gamma\gamma}\) 的峰值数量级可取 \(0.1\sigma_T\)。代入宇宙学 GRB 的 \(L_{\rm iso}\) 和毫秒—秒级 \(\delta t\)，\(\tau_{\gamma\gamma}\) 往往远大于 1，这与观测到的透明非热谱矛盾。

第六层基础是 \(\gamma\gamma\) 阈值。两个光子能否产生电子正电子对，不只取决于各自能量，还取决于夹角 \(\theta\)：\(E_1E_2(1-\cos\theta)\ge2(m_ec^2)^2\)。头碰头碰撞最有效，小角度同向传播则更难超过阈值。这个式子连接了 Band 高能尾和 compactness：给定最高能光子 \(E_{\max}\)，能与之湮灭的目标光子通常来自满足阈值的较高能尾部，而目标光子数由 \(\beta\) 和 \(E_{\rm peak}\) 附近的谱形控制。于是，谱拟合参数会直接影响 \(\Gamma_{\min}\)，而不是只影响辐射机制讨论。

第七层基础是相对论膨胀的三重修正。若发射物以 Lorentz factor \(\Gamma\) 向观测者运动，观测变时不再给出 \(R\simeq c\delta t\)，而是由角延迟或径向延迟得到 \(\delta t\sim(1+z)R/(2\Gamma^2c)\)，即 \(R\sim2\Gamma^2c\delta t/(1+z)\)。同时，观测能量在共动系降低为 \(E'\simeq E(1+z)/\Gamma\)；最高能光子的目标阈值在观测系近似变为 \(E_t\sim\Gamma^2(m_ec^2)^2/[E_{\max}(1+z)^2]\)。\(\Gamma\) 越大，发射半径越大、共动能量越低、目标阈值越高；若高能谱 \(N(E)\propto E^\beta\) 且 \(\beta<-1\)，阈值以上目标光子数迅速减少，常得到 \(\tau_{\gamma\gamma}\propto\Gamma^{-2(1-\beta)}\) 的陡峭下降。

第八层基础是观测闭合与模型边界。课堂上的透明性诊断最终写成 \(\tau_{\gamma\gamma}(E_{\max},\Gamma_{\min})=1\)，用来定义 \(\Gamma_{\min}\)；实际论文中则应通过 forward-folded Band 拟合获得 \(A,\alpha,\beta,E_{\rm peak}\) 的后验，再结合 \(z,\delta t,E_{\max}\) 传播到 \(\Gamma_{\min}\) 分布。读图时要看 \(\tau_{\gamma\gamma}(\Gamma)\) 曲线的 \(\tau=1\) 交点，也要看残差图是否支持所用谱模型。这个 \(\Gamma_{\min}\) 是单区、共时、共空间假设下的透明性下限，不是喷流 Lorentz factor 的精确测量。本讲到这里为止：它提供 \(\alpha,\beta,E_{\rm peak},L_{\rm iso},\delta t,\Gamma_{\min}\) 这些共同变量；下一讲才用这些变量检验同步辐射及其他 prompt 辐射机制是否自洽。

公式与推导

第一步：先固定本讲的观测谱语言。探测器经过响应折叠后约束的是入射光子数谱，而不是直接的物理发射率。定义

\[N(E)\equiv {dN\over dE\,dA\,dt}\quad [{\rm ph\,cm^{-2}\,s^{-1}\,keV^{-1}}],\qquad F_E=E\,N(E),\qquad \nu F_\nu=E^2N(E).\]

这里 \(E\) 是观测者系光子能量，\(F_E\) 是单位能量间隔的能量通量，\(E^2N(E)\) 则告诉我们每个对数能量间隔贡献多少能量通量。这一步的假设只是使用观测者系谱变量；它的物理作用是提醒学生：Band 的 \(\alpha,\beta\) 是 photon index，而 \(\nu F_\nu\) 的斜率要分别加 2。下一步才能在正确的谱空间中写 Band 经验函数。

第二步：写出 Band 低能段。假设在断裂能量以下，prompt 谱可由带指数截断的幂律经验描述，则

\[N(E)=A\left({E\over100\,{\rm keV}}\right)^\alpha\exp\left(-{E\over E_0}\right),\qquad E

其中 \(A\) 是归一化，\(\alpha\) 是低能光子指数，\(E_0\) 控制谱弯曲的能量尺度。这个公式不是辐射机制假设，只是经验坐标系；它为下一步求 \(\nu F_\nu\) 峰值能量提供函数形式。

第三步：由 \(\nu F_\nu\) 极值定义 \(E_{\rm peak}\)。在低能指数截断段内，令

\[{d\over dE}\left[E^2N(E)\right]=0,\]

代入上式得

\[E_{\rm peak}=(2+\alpha)E_0,\qquad \alpha>-2.\]

这里解出的物理量 \(E_{\rm peak}\) 是能量输出在对数能量空间最集中的观测尺度，而不是 Band 的连接能量。条件 \(\alpha>-2\) 表示低能侧 \(\nu F_\nu\) 在峰前上升；若峰落在探测器能段外，则后续用它估计光度和目标光子数都会带来额外不确定性。

第四步：补上高能尾和平滑连接。假设断裂以上由高能光子指数 \(β\) 的幂律描述，并与低能段在函数值和对数斜率上平滑连接，则连接能量为

\[E_b=(\alpha-\beta)E_0,\]

高能段写作

\[N(E)=A\left[{(\alpha-\beta)E_0\over100\,{\rm keV}}\right]^{\alpha-\beta}e^{\beta-\alpha}\left({E\over100\,{\rm keV}}\right)^\beta,\qquad E\ge E_b.\]

这里 \(\beta\) 决定高能尾中有多少光子可作为 \(\gamma\gamma\) 湮灭的目标光子，所以它不仅是谱拟合参数，也是 compactness 诊断的关键输入。下一步把谱积分成通量和光度。

第五步：从观测通量进入宇宙学能标。假设 GRB 位于红移 \(z\)，由时间分辨或时间积分谱得到观测能量通量 \(F\)，则各向同性等效光度为

\[L_{\rm iso}=4\pi d_L^2(z)\,F\,k.\]

其中 \(d_L\) 是 luminosity distance，\(k\) 是把观测能段换算到指定源静止系能段的修正。这个式子使用 BATSE 统计和后续红移观测所支持的宇宙学距离尺度；它解出的 \(L_{\rm iso}\) 不是喷流真实总功率，而是观测上最方便的能量尺度。下一步把巨大 \(L_{\rm iso}\) 与短变时结合，估计源内光子密度。

第六步：先做非相对论因果估计以暴露矛盾。若源静止或非相对论运动，最短显著变时 \(\delta t\) 给出发射区大小上限

\[R_{\rm nr}\lesssim {c\delta t\over1+z}.\]

在近似各向同性、半径为 \(R\) 的区域内，用平均观测光子能量 \(\langle E\rangle\) 做数量级估计，光子数密度为

\[n_\gamma\sim {L_{\rm iso}\over4\pi R^2c\langle E\rangle}.\]

这里解出的 \(n_\gamma\) 是 compactness 的核心中间量：短 \(\delta t\) 让 \(R\) 小，宇宙学距离让 \(L_{\rm iso}\) 大，二者共同产生极高的 MeV 光子密度。下一步把数密度转化为 pair-production 光深。

第七步：估算静止源的 \(\gamma\gamma\) 光深。假设高能光子穿过长度约 \(R\) 的目标光子场，并取 \(\gamma\gamma\) 截面峰值数量级 \(\sigma_{\gamma\gamma}\sim0.1\sigma_T\)，则

\[\tau_{\gamma\gamma}\sim n_\gamma\sigma_{\gamma\gamma}R\sim {L_{\rm iso}\sigma_{\gamma\gamma}\over4\pi Rc\langle E\rangle}.\]

代入非相对论半径 \(R_{\rm nr}\) 得

\[\tau_{\gamma\gamma,\rm nr}\sim {L_{\rm iso}\sigma_{\gamma\gamma}(1+z)\over4\pi c^2\delta t\langle E\rangle}\gg1.\]

典型 \(L_{\rm iso}\sim10^{51}-10^{53}\,{\rm erg\,s^{-1}}\)、\(\delta t\sim10^{-3}-1\,{\rm s}\)、\(\langle E\rangle\sim{\rm MeV}\) 会给出远大于 1 的光深。这个结果意味着静止紧致源应热化成 pair fireball，而不是让非热 Band 谱逃逸；因此下一步必须检查 pair-production 的能量阈值和相对论修正。

第八步：明确 \(\gamma\gamma\to e^+e^-\) 的阈值。两个光子能量为 \(E_1,E_2\)，夹角为 \(\theta\)，产生电子正电子对要求

\[E_1E_2(1-\cos\theta)\ge2(m_ec^2)^2.\]

这个关系说明，最高能光子并不与所有低能光子同等相互作用，而主要寻找满足阈值的目标光子；碰撞越接近头碰头，越容易超过阈值。它也提示相对论外流会通过减小共动系能量、改变角分布和增大发射半径同时降低光深。

第九步：引入相对论薄壳的时间延迟。假设发射区以 Lorentz factor \(\Gamma\) 指向观测者运动，观测变时主要由角延迟或径向延迟控制，则

\[\delta t\sim{(1+z)R\over2\Gamma^2c},\]

因此

\[R\sim{2\Gamma^2c\delta t\over1+z}.\]

这一步解出的是相对论发射半径：同样的短变时不再意味着 \(R\sim c\delta t\)，而允许半径增大 \(\Gamma^2\) 量级，从而降低源内光子密度。下一步还要处理光子能量在共动系中的降低。

第十步：把观测能量变换到共动系并求目标阈值。对朝向观测者的相对论外流，数量级上

\[E'\simeq {E(1+z)\over\Gamma}.\]

若观测到最高能光子 \(E_{\max}\)，忽略阶数为 1 的角因子后，与其发生 \(\gamma\gamma\) 湮灭所需的目标光子阈值在观测系可写为

\[E_t\sim{\Gamma^2(m_ec^2)^2\over E_{\max}(1+z)^2}.\]

这里 \(E_t\) 是从 Band 高能尾中抽取目标光子的下限。\(\Gamma\) 越大，阈值越高；若 \(\beta<0\)，阈值以上目标光子数迅速减少。下一步把这种减少写成 \(\tau_{\gamma\gamma}\) 对 \(\Gamma\) 的幂律依赖。

第十一步：用 Band 高能尾估计透明性对 \(\Gamma\) 的依赖。假设目标阈值以上的光子数谱近似为 \(N(E)\propto E^\beta\)，且 \(\beta<-1\)，则阈值以上目标光子数满足

\[N(>E_t)\propto\int_{E_t}^{\infty}E^\beta dE\propto E_t^{\beta+1}.\]

再结合 \(E_t\propto\Gamma^2\)、相对论半径 \(R\propto\Gamma^2\)，以及共动光子密度的 Lorentz 变换，单区数量级推导常写成

\[\tau_{\gamma\gamma}(E_{\max},\Gamma)\propto\Gamma^{-2(1-\beta)}.\]

当 \(\beta\simeq-2\) 时，\(\tau_{\gamma\gamma}\propto\Gamma^{-6}\)，所以 \(\Gamma\) 的小幅增加就能显著提高透明性。这一步把可观测的高能谱指数 \(\beta\) 与动力学下限直接连接起来。

第十二步：用透明性条件定义 Lorentz factor 下限。若最高能光子 \(E_{\max}\) 与低能 Band 光子来自同一发射区、同一时间段且共动系光子场近似各向同性，则透明性要求

\[\tau_{\gamma\gamma}(E_{\max},\Gamma_{{\min}})=1,\qquad \Gamma\ge\Gamma_{{\min}}.\]

更严格的光深不是一个简单幂律，而应写作

\[\tau_{\gamma\gamma}(E)=\int dl'\int d\Omega'\int d\epsilon'\,n'(\epsilon',\Omega')\,(1-\cos\theta')\,\sigma_{\gamma\gamma}(s).\]

这里带撇号的量在共动系中定义，\(n'\) 来自 forward-folded Band 谱参数后验，\(\sigma_{\gamma\gamma}(s)\) 使用双光子不变量 \(s\)。课堂中的 \(\Gamma^{-2(1-\beta)}\) 关系用于理解参数依赖；真实论文应数值积分并传播 \(A,\alpha,\beta,E_{\rm peak},z,\delta t,E_{\max}\) 的不确定性。最终得到的 \(\Gamma_{\min}\) 是单区透明性下限，不是喷流真实 \(\Gamma\) 的精确测量；若最高能光子来自外激波或更大半径区域，这个闭合关系就必须重新解释。

经典拟合 / 应用方法

第一，谱拟合要从“探测器看到什么”开始，而不是从已经画好的 unfolded 谱点开始。BATSE、GBM 或 LAT 的原始约束是不同能道中的计数 \(C_i\)，理论模型给出的是入射光子数谱 \(N(E;\theta)\)，其中 Band 模型的核心参数为 \(\theta=(A,\alpha,\beta,E_0)\) 或 \((A,\alpha,\beta,E_{\rm peak})\)。模型必须先经过探测器响应矩阵折叠，得到预测计数

\[\mu_i=\int R_i(E)N(E;\theta)dE+B_i,\]

其中 \(R_i(E)\) 包括有效面积、能量重分布和能道边界，\(B_i\) 是背景计数。读论文时第一件事就是确认作者拟合的是 folded count spectrum，而不是把响应修正后的视觉谱点当作独立高斯数据来拟合。

第二，似然函数决定了参数误差能否被正确解释。高计数能道中可以近似使用

\[\chi^2=\sum_i{(C_i-\mu_i)^2\over \sigma_i^2},\]

但 GRB 谱特别是高能尾常处于低计数区，更自然的主线是 Poisson likelihood 或 Cash statistic：

\[\ln\mathcal L=\sum_i\left[C_i\ln\mu_i-\mu_i-\ln(C_i!)\right].\]

如果背景来自触发前后时间段的多项式插值，背景不确定性不能简单忽略；它应进入联合似然、协方差矩阵或以 nuisance parameters 的形式传播。否则，高能端几个计数的过量就可能被误解释为真实的额外幂律或更硬的 \(\beta\)。

第三，拟合参数之间的退化必须作为课程重点讲清楚。Band 谱的 \(A\) 控制归一化，\(\alpha\) 控制低能 photon spectrum，\(E_{\rm peak}=(2+\alpha)E_0\) 控制 \(\nu F_\nu\) 峰值位置，\(\beta\) 控制高能尾。若探测器能段没有覆盖峰值两侧，\(E_{\rm peak}\) 会与 \(\alpha\) 或 \(\beta\) 强相关；若高能计数不足，Band 的 \(\beta\) 可能只是下限或被先验主导。读论文时不能只看 best-fit table，还要看置信区间、二维误差椭圆或 posterior corner plot，判断这些参数是否真的由数据约束。

第四，竞争模型比较不能只看统计量改善。常见模型包括单幂律、CPL/Comptonized、Band、黑体加幂律、多黑体、Band 加额外幂律等。CPL 少一个高能幂律参数，在高能尾不显著时可能更稳定；Band 可以描述高能尾，但若 \(\beta\) 未被数据约束，则把它用于 compactness 计算会引入很大的谱外推误差。模型选择时要同时检查 AIC/BIC、似然比或 Bayes factor、参数是否可辨识、残差是否系统，以及新增成分是否只是在吸收背景或校准误差。

第五，时间积分谱和时间分辨谱应分开处理。GRB prompt emission 常有 hard-to-soft 或 intensity-tracking 演化：\(E_{\rm peak}\)、\(\alpha\)、\(\beta\) 会在一个脉冲内部快速变化。若把多个时段叠加，窄谱可以被人工拓宽，看起来更像 Band 谱；这会影响辐射机制判断，也会影响 compactness 中目标光子场的估计。因此用于 \(\Gamma_{\min}\) 的谱，最好与最高能光子 \(E_{\max}\) 共时，并使用能反映最短结构的 \(\delta t\)，而不是机械采用全暴平均谱和全暴总 fluence。

第六，残差诊断是从经验拟合走向物理推断的关口。读 count spectrum 时要看上面板中 folded model 是否跟随计数谱，也要看下面板 residual 是否随机围绕零分布。低能端连续偏差可能来自响应矩阵、背景估计、能量阈值或额外热成分；\(E_{\rm peak}\) 附近的 S 形残差说明 Band 曲率不合适；高能端连续正残差可能提示额外 power-law、外激波贡献或背景处理问题。若残差已经显示模型不充分，就不能把 Band best-fit 参数直接代入 compactness 公式而不讨论系统误差。

第七，\(\Gamma_{\min}\) 应由谱参数后验传播，而不是由单组最佳拟合数值给出。实际流程可以是：从 \(A,\alpha,\beta,E_{\rm peak}\) 的联合 posterior 中抽样，同时抽样或设定 \(z,\delta t,E_{\max}\) 及最高能光子关联概率；对每组参数计算 \(\tau_{\gamma\gamma}(E_{\max},\Gamma)\)，再由 \(\tau_{\gamma\gamma}=1\) 求出一个 \(\Gamma_{\min}\)，最终得到 \(p(\Gamma_{\min}|{\rm data})\)。这个 posterior 的宽度不只是统计误差，还包括红移未知、时间 bin 选择、谱外推、探测器交叉归一化、LAT 光子背景概率和单区共时假设等系统项。

第八，应用时要把“透明性下限”和“动力学模型”区分开。compactness 拟合的产物是条件性结论：在最高能光子和低能 Band 光子同区、共时、共空间产生，并且共动系光子场近似各向同性的假设下，源必须满足 \(\Gamma\ge\Gamma_{\min}\)。这并不等于测得了唯一真实 \(\Gamma\)，也不自动证明内部激波、磁重联或光球模型。它的价值在于给所有 prompt emission 模型设置共同门槛：任何模型都必须解释 Band 参数、短变时、高能光子逃逸和相对论透明性。

重点章节 / 结果段落怎么读

读 BATSE 经典结果时，第一段要看样本定义和触发选择，而不是直接跳到“天空各向同性”结论。课程阅读应要求学生标出：触发能段是什么、峰值流量 \(P\) 在多长时间尺度上定义、全天曝光是否均匀、弱源完整性阈值在哪里。只有理解这些选择函数，才能判断天空分布不追踪银道面和 logN-logP 弱源端变平是否具有物理意义。

第二段要读 BATSE 的天空分布与 \(V/V_{\max}\)、logN-logP 的组合证据。各向同性本身不能唯一证明宇宙学起源，因为一个足够大的银河晕也能近似各向同性；关键是各向同性再加上相对均匀欧几里得预期 \(N(>P)\propto P^{-3/2}\) 的偏离。读这一部分时要训练学生问：强源端是否接近 -3/2？弱源端变平是否发生在完整性阈值之上？\(V/V_{\max}\) 是否显著小于均匀欧几里得样本的 1/2？这些问题共同构成“距离尺度被推向宇宙学”的观测动机。

读 Band 等人的经验谱工作时，重点不是机械记忆分段函数，而是理解它如何成为 prompt emission 的公共坐标系。应仔细阅读作者如何定义光子数谱 \(N(E)\)、低能指数 \(\alpha\)、高能指数 \(\beta\)、连接能量和 \(E_{\rm peak}\)。特别要检查论文中的图是 photon spectrum、energy flux spectrum 还是 \(\nu F_\nu\)；同一组 \(\alpha,\beta\) 在不同纵轴上给人的视觉直觉完全不同。课程中要反复强调：Band function 是经验函数，不是同步辐射、光球或 Compton 化的判决书。

读 compactness 经典论证时，应按一条推导链阅读。先找作者如何从通量和距离得到 \(L_{\rm iso}\)，再找如何用 \(\delta t\) 得到非相对论半径 \(R_{\rm nr}\lesssim c\delta t/(1+z)\)，接着看光子数密度和 \(\gamma\gamma\) 光深的数量级估计，最后看相对论 Lorentz factor 如何同时改变发射半径、共动系能量和碰撞角。若一篇论文只给出一个 \(\Gamma_{\min}\) 数值，却没有说明 \(E_{\max}\)、\(\beta\)、\(\delta t\)、红移和共时性假设，就应视为不完整的透明性诊断。

读 Fermi 时代的现代论文时，要把 GBM 与 LAT 作为同一推断链的两端。GBM 提供 keV–MeV 的 Band 目标光子场，LAT 提供 GeV 光子、到达时间和高能延迟。高能延迟是关键段落：若 GeV 光子明显晚于 MeV prompt 峰值，它可能来自更大半径或外激波区域，此时单区 \(\Gamma_{\min}\) 的解释需要修改。现代论文的重点往往不再是“是否需要相对论外流”，而是“哪个区域产生最高能光子，目标光子场是否与低能 Band 成分同区”。

最后，读任何 GRB prompt 论文都应把三个层次分开：第一，经验谱模型是否能描述 count data；第二，源是否满足 \(\tau_{\gamma\gamma}<1\) 的透明性约束；第三，微观辐射机制是否能解释 \(\alpha,\beta,E_{\rm peak}\) 和谱演化。Band 拟合成功不等于同步辐射成立；\(\Gamma_{\min}\) 很大不等于内部激波成立；发现热成分也不等于整个 prompt 谱都是光球。本讲只建立前两个层次的语言，为下一讲的辐射机制比较做准备。

建议重点查看的图表

  1. BATSE 全天分布图：通常用银河坐标投影显示 GRB 方向。应检查点源是否沿银道面集中，是否需要曝光修正、地球遮挡修正和触发阈值修正。重点是整体各向同性，而不是局部空洞或小团簇。

  2. logN-logP 图：横轴为峰值流量或峰值计数率 \(P\)，纵轴为累计数 \(N(>P)\)。应比较强源端与均匀欧几里得 \(-3/2\) 斜率、弱源端变平位置和样本完整性阈值。陷阱是把阈值不完整造成的变平直接解释为宇宙学效应。

  3. \(V/V_{\max}\) 统计图或表：应看样本平均 \(\langle V/V_{\max}\rangle\) 是否显著低于 1/2，以及不同子样本的选择效应。它与 logN-logP 是同一距离尺度问题的互补表达。

  4. Band 谱的 photon spectrum 与 \(\nu F_\nu\) 图：若纵轴是 \(N(E)\)，\(\alpha,\beta\) 是直接斜率；若纵轴是 \(E^2N(E)\)，低能和高能斜率变为 \(\alpha+2\)、\(\beta+2\)。应检查 \(E_{\rm peak}\) 是否在探测器有效能段内。

  5. \(\alpha,\beta,E_{\rm peak}\) 参数分布图：应看误差条、时间积分/时间分辨样本的区别、能段覆盖和被固定参数的比例。不要把无约束的 \(\beta\) 或落在能段边缘的 \(E_{\rm peak}\) 当作稳固物理结论。

  6. 光变曲线与时间分辨谱演化图：横轴为时间，面板中同时显示 flux、\(E_{\rm peak}\)、\(\alpha\)、\(\beta\)。应判断谱演化是 hard-to-soft 还是 intensity tracking，并检查用于 compactness 的时间 bin 是否与 \(E_{\max}\) 共时。

  7. \(\tau_{\gamma\gamma}(\Gamma)\) 或 \(\Gamma_{\min}\) 曲线：横轴为 Lorentz factor，纵轴为 pair opacity。应读出 \(\tau=1\) 的交点，并比较不同 \(\delta t,\beta,E_{\max},z\) 假设下曲线如何移动。陷阱是把交点当成真实 \(\Gamma\) 的精确测量。

  8. 谱拟合残差图：上面板看 folded model 与计数谱，下面板看 residual。应关注低能校准/背景残差、高能端额外正残差、\(E_{\rm peak}\) 附近 S 形结构。残差是否系统，往往比 best-fit 参数本身更能判断模型是否可靠。

关键图表逐图导读

图 1：BATSE 全天分布图。横轴和纵轴通常是银河经纬度的投影，点代表触发到的 GRB 方向。读图先找银道面：若 GRB 是简单银河盘源，点应沿银道面显著聚集；BATSE 的关键信息是点云近似各向同性。模型线通常不是重点，重点是与银河盘预期的空间模式比较。误读风险在于把局部空洞或局部密集区当作真实大尺度结构，而忽略曝光、地球遮挡、触发灵敏度和有限样本噪声。

图 2：logN-logP 累计亮度分布。横轴为峰值流量或峰值计数率 \(P\)，纵轴为大于该 \(P\) 的累计数 \(N(>P)\)，常用双对数坐标。均匀欧几里得空间中标准参考线斜率为 \(-3/2\)。读图时应看强源端是否接近参考线、弱源端是否变平、变平是否在完整性阈值以上。常见误读是只看到偏离 \(-3/2\) 就断言宇宙学起源；严格说还要同时考虑 luminosity function、源演化和触发选择函数。

图 3：\(V/V_{\max}\) 图或统计表。横轴可为样本子类、峰值流量区间或持续时间分类，纵轴为 \(\langle V/V_{\max}\rangle\)。均匀欧几里得无演化样本的期望约为 1/2。若平均值显著低于 1/2，说明弱源相对亏缺或样本接近尺度边界。读图时要看误差条和样本划分；短暴、长暴、软暴的触发效率不同，不能把所有子样本差异都直接解释为不同物理距离尺度。

图 4：Band 谱的 \(\nu F_\nu\) 表示。横轴为能量 \(E\)，纵轴为 \(E^2N(E)\)，模型线在 \(E_{\rm peak}\) 达到峰值。低能侧视觉斜率是 \(\alpha+2\)，高能侧视觉斜率是 \(\beta+2\)。数据点应与 folded model 对应，不能把 unfolded 点的误差当作独立观测。误读风险是把 photon index \(\alpha=-1\) 当成 \(\nu F_\nu\) 下降；实际上此时低能 \(\nu F_\nu\propto E^{1}\)，仍在上升。

图 5：Band photon spectrum 图。横轴仍为能量，但纵轴是 \(N(E)\) 或 photons \({\rm cm^{-2}s^{-1}keV^{-1}}\)。这时 \(\alpha\) 和 \(\beta\) 才是图上的直接幂律斜率，\(E_b=(\alpha-\beta)E_0\) 是连接能量，而 \(E_{\rm peak}=(2+\alpha)E_0\) 是 \(\nu F_\nu\) 峰值，不一定等于图上 photon spectrum 最显眼的弯折位置。常见陷阱是混淆 \(E_b\)、\(E_0\) 与 \(E_{\rm peak}\)。

图 6：时间分辨谱演化图。横轴为时间，上面板常为光变曲线，下面板给出 \(E_{\rm peak}\)、\(\alpha\)、\(\beta\) 的时间演化。读图时要把谱 bin 与脉冲结构对齐：若 \(E_{\rm peak}\) 跟随 flux 升降，是 intensity tracking；若随时间单调变软，是 hard-to-soft。残差和误差条同样重要，因为低信噪比 bin 中的 \(\beta\) 往往不可约束。误读风险是用全暴平均谱替代瞬时谱，从而掩盖真实演化。

图 7：\(\alpha\) 分布与理论参考线。横轴为低能 photon index \(\alpha\)，纵轴为样本数或概率密度，图上常标出 \(\alpha=-2/3\) 和 \(\alpha=-3/2\)。这些线属于下一讲同步辐射边界的预告：前者对应标准慢冷低能极限，后者常与快冷谱联系。读本讲时只需把它们作为经验参数分布的参考，不展开辐射机制判决。误读风险是忽略误差、时间积分偏置和热成分混合后，直接宣布某个机制被排除。

图 8：\(\tau_{\gamma\gamma}(\Gamma)\) 曲线。横轴为 \(\Gamma\)，纵轴为 \(\tau_{\gamma\gamma}\)，通常为对数坐标；模型线随 \(\Gamma\) 急剧下降，水平线 \(\tau=1\) 与曲线交点给出 \(\Gamma_{\min}\)。读图要检查曲线族对应哪些假设：更短 \(\delta t\)、更高 \(E_{\max}\)、更硬或更不确定的 \(\beta\)、不同红移都会移动交点。误读风险是把 \(\Gamma_{\min}\) 画成喷流 Lorentz factor 的精确测量，而它只是单区透明性下限。

图 9：谱拟合残差图。上面板显示计数谱与折叠后的模型计数，下面板显示 residual、pull 或 \(\Delta\chi\)。随机残差说明经验模型在统计意义上可接受；高能端连续正残差可能指向额外 power-law 或背景低估；低能端成片偏差可能来自响应校准或背景；\(E_{\rm peak}\) 附近 S 形残差说明模型曲率不匹配。读图结论应是“这个模型是否足以支撑后续 compactness 输入”，而不是只报告 \(\chi^2/{\rm dof}\) 和 best-fit 参数。

后续阅读路径

第一条路径是从 BATSE 统计到红移确认。建议先读 BATSE 全天分布、logN-logP 和 \(V/V_{\max}\) 的经典结果，理解为什么各向同性加非欧几里得亮度分布把 GRB 推向大尺度起源；再读 BeppoSAX 之后的 X 射线/光学余辉定位、宿主星系识别和红移测量。这样可以看清历史逻辑：compactness problem 并不是等红移时代才出现的附属问题，而是在宇宙学距离尺度逐渐被接受时已经形成的物理压力。

第二条路径是从 Band function 到 prompt 辐射机制。读完 Band 经验谱后，应继续读 CPL、Band 加黑体、Band 加额外幂律、多黑体和物理化 Comptonization 模型的比较。进入下一讲时，把每个模型的预言都翻译成 \(\alpha,\beta,E_{\rm peak}\)、谱宽、时间演化和残差结构。尤其要注意：同步辐射 line-of-death、快冷问题、边缘冷却、衰减磁场、jitter radiation 和小 pitch-angle 模型，都是在试图解释本讲建立的同一组经验参数。

第三条路径是从 compactness 到 Lorentz factor 与发射半径诊断。经典 fireball 论文说明相对论膨胀如何解决 \(\tau_{\gamma\gamma}\gg1\) 的矛盾；Fermi/LAT 时代的高能光子则把 \(\Gamma_{\min}\) 约束推到更高精度，但也引入高能延迟、多区发射和外激波 GeV 起源的问题。现代研究的核心不再是“GRB 是否相对论性”，而是 \(\Gamma\)、发射半径 \(R\)、磁化程度、辐射效率和多成分谱如何同时自洽。

第四条路径是方法论训练。应学习 XSPEC、RMFIT、3ML 或类似 forward-folding 框架，理解响应矩阵、背景建模、Poisson likelihood、posterior sampling 和多探测器联合拟合。GRB prompt 研究很容易被平滑的经验曲线误导；真正稳健的结论来自时间分辨分析、系统误差显式传播、残差诊断和物理量后验的完整链条。

第五条路径是从单区透明性走向多区 prompt-afterglow 连接。本讲的 \(\Gamma_{\min}\) 假设最高能光子与低能 Band 光子同区共时；后续可阅读外激波早期 GeV 辐射、内部耗散半径、光球半径和磁重联半径的比较。这样能理解为什么同一个 LAT 光子既可能加强 compactness 约束，也可能迫使我们修改单区解释。

下一讲建议

下一讲最自然的方向是本系列第 2 讲：“Band 谱的物理起源：同步辐射、快冷问题与 line-of-death”。它直接使用本讲建立的 \(\alpha,\beta,E_{\rm peak}\)、时间分辨谱和残差诊断语言，讨论 optically thin synchrotron 能否解释低能指数、谱峰和谱演化。本讲只把 \(\alpha=-2/3\) 与 \(\alpha=-3/2\) 作为边界提示；下一讲才系统推导这些谱指数并比较观测。

第二个方向是本系列第 3 讲：“光球、耗散与多成分 prompt 谱”。它承接本讲的 compactness 和透明性边界，但问题从“光子如何逃逸”转向“光子在喷流中如何形成谱形”。这一讲可讨论热光球为何会变宽、亚光球耗散如何产生非热尾、Band 加黑体残差如何读图，以及光球模型与同步辐射模型在 \(E_{\rm peak}\)、谱宽和时间演化上的差异。

若课程需要在三讲之外扩展，第三个可选方向是“Fermi LAT 高能延迟、\(\Gamma_{\min}\) 与多区发射半径诊断”。它会重新审视本讲的单区假设，把最高能光子、外激波起始时间、早期余辉和 prompt MeV 成分放在同一时间轴上。但在当前三讲规划中，这部分应作为第 3 讲后的扩展，而不应提前打断本讲从 BATSE、Band 到 compactness 的基础链条。
