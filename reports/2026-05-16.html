1. GRB prompt emission 经典课（一）：紧致性灾难、相对论火球与 prompt 光谱的 Band 函数

  • 课程系列：GRB prompt emission 经典课程（第 1/3 讲）
  • 本讲边界：本讲只讲 prompt emission 的两个基础支柱：第一，为什么短时标、高光度、MeV 光子会导致紧致性灾难，并如何由此推出超相对论 bulk Lorentz factor；第二，BATSE 时代 Band 函数如何把复杂 prompt 光谱压缩为可拟合的经验模型，并由低能谱指数、高能谱指数和峰值能量连接到物理判据。本讲不讲外激波 afterglow 的 Blandford-McKee 动力学和多波段余辉闭合关系；也不系统讲 internal shock、磁重联或光球辐射的详细竞争，这些留到本系列第 2、3 讲。
  • 上下文承接：这是一个新系列的第一讲。此前已经讲过 GRB 余辉标准模型，从 Blandford-McKee 自相似解到同步辐射多波段光变曲线；本讲有意避开外激波余辉，把时间窗口前移到触发后毫秒到数百秒的 prompt 阶段，关注紧致性、相对论运动和经验光谱拟合。
  • 主题：GRB prompt emission and afterglow classic work
  • 经典工作：BATSE 时代确立的 GRB prompt 非热谱经验规律，尤其是 Band 等人在 1993 年提出的双幂律经验谱函数；以及 Goodman、Paczynski、Shemi 与 Piran 关于紧致性问题和相对论火球的经典理论框架。
  • 为什么经典：GRB prompt emission 的经典性在于：观测上，BATSE 显示 GRB 在全天各向同性、短时标、非热 MeV 光谱和复杂光变方面形成了一组强约束；理论上，紧致性问题迫使我们接受 bulk Lorentz factor 通常达到 \(\Gamma\sim 10^2-10^3\)，这是现代 GRB 物理的入口条件；方法上，Band 函数虽然是经验式，却至今仍是 Fermi/GBM、Konus-Wind、Swift/BAT 与联合谱分析的基准模型。任何关于 prompt radiation mechanism 的物理模型，首先都必须通过三个门槛：不被 \(\gamma\gamma\) 吸收、给出正确的 \(E_{\rm pk}\) 分布、解释低能谱指数与光变时标。

经典工作详细讲解

第一段：科学问题。GRB prompt emission 最直接的问题不是“它从哪里来”，而是“它为什么能被我们看见”。典型 GRB 在数秒内释放的各向同性等效能量可达 \(10^{51}-10^{54}\,\mathrm{erg}\)，光变结构可短至毫秒到几十毫秒，谱峰落在几十 keV 到数 MeV。若源区静止且尺度由 \(R\lesssim c\delta t\) 给出，则光子数密度巨大，高能光子彼此发生 \(\gamma\gamma\to e^+e^-\)，光学深度远大于一，结果应是热化的黑体或至少严重截断的谱，而不是 BATSE 看到的非热宽谱。这就是紧致性灾难。

第二段：历史背景中的方法转折。早期 GRB 距离未知时，紧致性问题已经存在；一旦 1990 年代 BATSE 的各向同性分布和后来的 afterglow 红移确认宇宙学距离，问题变得不可回避。Goodman 和 Paczynski 等人的火球图像指出，一个初始辐射主导的高温等离子体若携带较少重子，可通过绝热膨胀把内能转化为 bulk kinetic energy，形成 \(\Gamma\gg1\) 的外流。相对论运动一方面放大可见区域尺度，另一方面在共动系降低光子能量和密度，极大减小成对产生不透明度。

第三段：关键观测结果。BATSE 不是只给出 GRB 数量，而是给出了 prompt 研究的坐标系：\(T_{90}\) 的双峰分布、各向同性天空分布、非欧几里得 \(\log N-\log P\)、复杂多脉冲光变、以及宽能段时间积分光谱。Band 函数把大部分明亮 burst 的能谱表示为平滑连接的 broken power law，参数为归一化 \(A\)、低能光子指数 \(\alpha\)、高能光子指数 \(\beta\)、特征能量 \(E_0\) 或 \(E_{\rm pk}\)。这个经验模型的威力在于它足够简单，可以做样本统计；又足够灵活，可以暴露物理模型的失败，例如同步辐射低能极限问题。

第四段：局限。Band 函数不是辐射机制。它没有告诉我们电子分布、磁场、光球、耗散半径或喷流组成；时间积分谱还会把谱演化混在一起，使 \(\alpha\) 变软、\(E_{\rm pk}\) 展宽。紧致性下限也不是单个精确数值，它依赖最高能光子、变源时标、谱指数、红移、几何和辐射区域假设。若高能 GeV 光子来自不同区域，用它反推 prompt MeV 区的 \(\Gamma\) 会有系统误差。

第五段：后续影响。现代 prompt 研究几乎都沿着本讲两个经典支柱展开：一类工作从 Fermi/LAT 高能光子和快速光变给 \(\Gamma\) 下限；另一类工作从 Band 参数分布、热成分搜索、谱滞后和时间分辨谱出发，判断辐射机制是 optically thin synchrotron、photosphere、Comptonization 还是 magnetic reconnection。即使在多信使时代，prompt 光谱仍是喷流物理的最密集信息源。

背景知识

第一段：GRB prompt emission 是指探测器触发时看到的强烈 keV-MeV 爆发阶段，通常持续 \(\sim 10^{-2}\) 到 \(10^3\) 秒，包含许多窄脉冲和强烈谱演化。它不同于 afterglow：prompt 通常被认为来自中心引擎驱动外流内部的耗散，afterglow 则多来自外流与星际或恒星风介质相互作用形成的外激波。

第二段：历史上，BATSE 的关键贡献是把 GRB 从少数奇异事件变成统计天体物理问题。全天各向同性排除了明显的银河盘分布；弱源亏缺提示宇宙学尺度或复杂选择效应；短暴和长暴的 \(T_{90}\) 双峰分布预示不同前身星通道。对 prompt 物理而言，更重要的是 BATSE 提供了大量宽能段光谱，使 Band 函数成为共同语言。

第三段：为什么 prompt emission 重要？因为它离中心引擎最近，携带关于喷流组成、磁化程度、能量耗散半径和粒子加速机制的信息。余辉能告诉我们总动能、环境密度和喷流角，但 prompt 才直接面对“能量如何从引擎输出转成辐射”这个问题。GRB 的辐射效率、谱峰能量和毫秒光变都对模型构成强约束。

第四段：常见误区之一是把 Band 函数当作同步辐射模型。实际上 Band 函数只是经验拟合式；其 \(\alpha\) 和 \(\beta\) 可以与同步辐射、康普顿化或光球模型比较，但不能直接等同于电子谱指数。误区之二是把 \(R\sim c\delta t\) 当作相对论源区真实半径；对指向我们的 relativistic shell，曲率效应给出的可变时标通常是 \(\delta t\sim R/(2\Gamma^2 c)\)，因此同样的毫秒结构可来自大得多的半径。

基础理论 / 方法脉络

第一段：核心物理图像是“高紧致度要求相对论运动”。若一个各向同性静止源在半径 \(R\) 内释放光度 \(L\)，光子密度约为 \(n_\gamma\sim L/(4\pi R^2 c\langle E\rangle)\)。成对产生截面在阈值附近为 Thomson 截面的一个量级分数，因此 \(\tau_{\gamma\gamma}\sim n_\gamma\sigma_{\gamma\gamma}R\) 很容易远大于 1。可观测非热谱意味着实际共动系中光子密度和能量必须被相对论效应降低。

第二段：相对论外流带来三种关键修正。首先，观测能量与共动能量关系为 \(E\simeq \mathcal{D}E'/(1+z)\)，其中 \(\mathcal{D}\approx 2\Gamma\) 对准视线。其次，变源时标对应的半径可为 \(R\sim 2\Gamma^2 c\delta t/(1+z)\)。第三，对给定观测高能光子，共动系中可与其湮灭的靶光子阈值升高，从而靶光子数减少。三者合在一起使 \(\tau_{\gamma\gamma}\) 对 \(\Gamma\) 呈强幂律下降。

第三段：Band 函数的观测逻辑是把 photon spectrum \(N(E)=dN/dE\) 写成低能指数 \(\alpha\) 和高能指数 \(\beta\) 的平滑拼接。通常报告的 \(\nu F_\nu\) 峰值能量满足 \(E_{\rm pk}=(2+\alpha)E_0\)，前提是 \(\beta<-2\) 且 \(\alpha>-2\)。这使不同 burst 可在 \((\alpha,\beta,E_{\rm pk})\) 空间比较，也可研究 hardness-intensity correlation 和 Amati/Yonetoku 相关等经验规律。

第四段：分析 prompt 的基本流程是：先做时间选择和背景建模，得到不同时间 bin 的计数谱；再用仪器响应矩阵把 photon model forward-fold 到 count space；然后比较 Band、cutoff power law、blackbody plus power law、Comptonized 等模型。真正被拟合的不是解卷积后的“光子点”，而是各能道中的 Poisson 计数或近似 Gaussian 计数。

公式与推导

从紧致性开始。若观测到各向同性等效光度 \(L_{\rm iso}\) 和最短变源时标 \(\delta t\)，非相对论估计给出源区尺度

\[R_0\lesssim c\delta t .\]

光子平均能量为 \(\langle E\rangle\) 时，源区光子数密度约为

\[n_\gamma\simeq {L_{\rm iso}\over 4\pi R_0^2 c\langle E\rangle} .\]

高能光子穿过该区域的成对产生光学深度可粗略写成

\[\tau_{\gamma\gamma}\simeq n_\gamma\sigma_{\gamma\gamma}R_0\simeq {L_{\rm iso}\sigma_{\gamma\gamma}\over 4\pi c^2\delta t\langle E\rangle} ,\]

其中 \(\sigma_{\gamma\gamma}\sim 0.1\sigma_T\)。代入 \(L_{\rm iso}\sim10^{52}\,\mathrm{erg\,s^{-1}}\)、\(\delta t\sim10^{-2}\,\mathrm{s}\)、\(\langle E\rangle\sim1\,\mathrm{MeV}\)，可得 \(\tau_{\gamma\gamma}\gg1\)，这与非热 MeV 谱矛盾。

成对产生的阈值来自四动量不变量。两光子能量为 \(E_1,E_2\)，夹角为 \(\theta\)，则

\[E_1E_2(1-\cos\theta)\ge 2(m_ec^2)^2 .\]

对观测到的高能光子 \(E_{\max}\)，若源静止，靶光子阈值为

\[E_{t}\gtrsim {2(m_ec^2)^2\over E_{\max}(1-\cos\theta)} .\]

相对论外流中，观测能量与共动能量满足

\[E={\mathcal D\over 1+z}E',\qquad \mathcal D=[\Gamma(1-\beta\cos\theta_{\rm obs})]^{-1}\approx 2\Gamma .\]

同时曲率时间给出辐射半径

\[R\simeq {2\Gamma^2 c\delta t\over 1+z} .\]

因此共动辐射能量密度近似为

\[u'_\gamma\equiv u'_\gamma\simeq {L_{\rm iso}\over 4\pi R^2 c\Gamma^2} ,\]

这里额外的 \(\Gamma^{-2}\) 表示从实验室系到共动系的能量密度变换。若高能端 photon spectrum 近似 \(N(E)\propto E^{\beta}\)，超过阈值的靶光子数还会随 \(\Gamma\) 下降，典型可写成

\[\tau_{\gamma\gamma}(E_{\max})\propto {L_{\rm iso}\over \delta t}\,E_{\max}^{-(1+\beta)}\,\Gamma^{-2(1-\beta)} .\]

令 \(\tau_{\gamma\gamma}<1\) 即得 \(\Gamma_{\min}\)，对 \(\beta\simeq-2\) 常出现 \(\Gamma_{\min}\sim10^2-10^3\)。这个推导的核心不是系数，而是 \(\tau_{\gamma\gamma}\) 对 \(\Gamma\) 的强依赖。

Band 函数定义为 photon flux

\[N(E)=A\left({E\over 100\,\mathrm{keV}}\right)^\alpha \exp\left(-{E\over E_0}\right),\quad E
\[N(E)=A\left[{(\alpha-\beta)E_0\over100\,\mathrm{keV}}\right]^{\alpha-\beta}e^{\beta-\alpha}\left({E\over100\,\mathrm{keV}}\right)^\beta,\quad E\ge E_b,\]

其中拼接能量

\[E_b=(\alpha-\beta)E_0 .\]

能量谱常看 \(\nu F_\nu\propto E^2N(E)\)。对低能段求极值：

\[{d\over dE}\left[E^{\alpha+2}e^{-E/E_0}\right]=0\]

得到

\[E_{\rm pk}=(\alpha+2)E_0 .\]

这条公式解释了为什么 \(E_0\) 不是通常讨论的谱峰能量；当 \(\alpha\le -2\) 时低能段 \(\nu F_\nu\) 不再上升，Band 参数的物理解释也会变得危险。

经典拟合 / 应用方法

经典 prompt 谱拟合必须在计数空间完成。设第 \(i\) 个能道的预测计数为 \(\mu_i(\boldsymbol\theta)=b_i+t\int R_i(E)N(E;\boldsymbol\theta)dE\)，其中 \(b_i\) 是背景计数，\(t\) 是曝光时间，\(R_i(E)\) 是仪器响应，\(\boldsymbol\theta=(A,\alpha,\beta,E_0)\) 或 \((A,\alpha,\beta,E_{\rm pk})\)。若计数较低，常用 Poisson likelihood 或 Cash statistic：\(\ln\mathcal L=\sum_i[n_i\ln\mu_i-\mu_i-\ln(n_i!)]\)。若背景由 off-source 时间段拟合并带不确定度，应使用 PG-stat 或把背景参数一并纳入似然。

常见模型包括：power law，适合窄能段或弱 burst 但不能描述谱峰；cutoff power law/Comptonized model，参数为 \(A,\alpha,E_{\rm cut}\)，常用于高能端不足以约束 \(\beta\) 的情形；Band model，适合宽能段明亮 burst；blackbody 或 multicolor blackbody plus power law，用于搜索光球热成分；Band plus blackbody，用于检验是否存在窄热峰叠加在非热连续谱上。模型选择可用 likelihood ratio、AIC/BIC、posterior predictive check，但需注意边界参数和非嵌套模型时简单 \(\Delta\chi^2\) 判据可能失效。

参数退化是 prompt 拟合的核心问题。若探测器上限不够高，\(\beta\) 会与背景斜率混淆；若 \(E_{\rm pk}\) 接近能段边缘，\(\alpha\) 与 \(E_{\rm pk}\) 强相关；时间积分会把 hard-to-soft 演化叠加成较宽谱形，使 Band 看起来优于物理上更窄的瞬时谱。严肃分析应报告参数协方差、posterior corner plot、能道残差、不同时间 bin 的一致性，以及换背景区间和响应矩阵后的系统变化。

残差诊断要看三类结构：第一，在低能端是否存在系统正残差，可能提示额外软成分或吸收/校准问题；第二，在 \(E_{\rm pk}\) 附近是否有窄峰，可能提示 quasi-thermal photosphere；第三，在高能端是否有持续正残差，可能需要额外 power-law 成分并可能涉及 LAT 延迟高能辐射。拟合应用上，Band 参数可用于 population study、\(k\)-correction、\(E_{\rm iso}\) 估计、谱硬度演化和与 \(\Gamma_{\min}\) 约束结合。

重点章节 / 结果段落怎么读

读经典 Band 谱论文和 BATSE 结果时，第一步不要先看物理解释，而要看样本选择、能段和背景处理。prompt 光谱的经验结论高度依赖探测器能段，BATSE 对几十 keV 到约 MeV 最敏感，因此特别适合定义 \(E_{\rm pk}\)，但对更高能的额外成分并不总有判别力。

第二步读模型定义。要确认作者的谱指数是 photon index 还是 energy index，确认 \(E_0\)、break energy 和 \(E_{\rm pk}\) 的关系。许多误读来自把 \(E_b=(\alpha-\beta)E_0\) 当成 \(\nu F_\nu\) 峰，或把 \(\alpha\) 与电子谱指数直接对应。

第三步读统计段落。注意作者使用的是解卷积光子谱展示，还是 forward-folded count fitting；注意误差是单参数 \(1\sigma\) 还是联合置信区间；注意样本中有多少 burst 的 \(\beta\) 实际被约束。经典论文的表格往往比摘要更重要，因为那里显示了参数分布的真实离散度。

第四步读讨论段。关键不是“Band 函数拟合得好”，而是“它给理论制造了哪些麻烦”：低能 \(\alpha\) 有时硬于 optically thin synchrotron 的 \(-2/3\) 极限，高能 \(\beta\) 控制能量是否收敛，\(E_{\rm pk}\) 分布窄于某些简单火球预期。这些张力引出了后续几十年的 prompt radiation mechanism 研究。

建议重点查看的图表

  1. \(T_{90}\) 分布图：看短暴和长暴双峰，以及分类并非绝对物理边界。
  2. 全天分布图：看各向同性，理解为什么银河盘模型受到挑战。
  3. 典型 prompt 光变曲线：看毫秒到秒级脉冲、非周期结构和 hard-to-soft 演化。
  4. \(\nu F_\nu\) 光谱与 Band 拟合图：看谱峰位置、低能斜率、高能尾以及模型残差。
  5. \(\alpha\) 分布图：看有多少事件越过同步辐射 line of death \(\alpha=-2/3\)，并区分时间积分和时间分辨结果。
  6. \(E_{\rm pk}\) 分布图：看探测器选择效应、亮 burst 与暗 burst 的差异。
  7. 高能光子到达时间与 \(\Gamma_{\min}\) 图：看最高能光子是否与 MeV prompt 同区产生。
  8. 参数相关图，如 \(E_{\rm pk}\)-fluence 或 hardness-intensity：看相关是否由选择效应驱动。

关键图表逐图导读

图 1：典型 prompt 光变曲线。横轴通常是触发后时间，纵轴是不同能段计数率。读图时先找最窄显著脉冲，由它估计 \(δt\)，但要注意 binning 会抹平结构。再比较软硬能段：若高能峰更早、更窄，说明存在谱演化；这种演化会影响时间积分 Band 参数。图上的背景线若不是水平的，要检查是否有轨道或粒子背景变化。

图 2：Band 函数的 \(\nu F_\nu\) 拟合图。横轴为能量，纵轴常为 \(E^2N(E)\) 或 unfolded energy flux。低能段斜率由 \(\alpha+2\) 决定，高能段斜率由 \(\beta+2\) 决定，峰值为 \(E_{\rm pk}\)。若模型线在峰附近平滑转折而残差无结构，Band 经验拟合成立；若低能端持续正残差，可能需要热成分；若高能端上翘，可能存在额外 power-law。必须记住 unfolded 点依赖假设模型，判断优劣应看 count residuals。

图 3：\(\alpha\) 与 \(E_{\rm pk}\) 的样本散点或直方图。横轴可能是 \(E_{\rm pk}\)，纵轴是低能 photon index \(\alpha\)。课堂上应先画出 \(\alpha=-2/3\) 的 optically thin synchrotron 慢冷却硬极限，以及 \(\alpha=-3/2\) 的快速冷却典型值。若大量时间分辨谱位于 \(-2/3\) 以上，单区 optically thin synchrotron 受压；若时间积分谱较软，说明谱演化混合显著。看图时还要注意误差椭圆方向，因为 \(\alpha\) 与 \(E_{\rm pk}\) 往往强相关。

图 4：紧致性约束示意图。横轴可为 \(\Gamma\)，纵轴为 \(\tau_{\gamma\gamma}\) 或允许的 \(R-\Gamma\) 区域。模型线快速下降，穿过 \(\tau=1\) 的位置给出 \(\Gamma_{\min}\)。若图中标出最高能光子能量、到达时间和变源时标，应分别检查：该高能光子是否属于同一时间 bin；是否采用同一 Band 外推作为靶光子场；是否考虑红移。主要结论不是某个精确 \(\Gamma\)，而是非相对论源被排除，且 prompt 区域必须 relativistically expanding。

图 5：时间分辨谱演化图。横轴是时间，纵轴给出 \(E_{\rm pk}\)、\(\alpha\) 或 flux。常见模式有 hard-to-soft 和 intensity-tracking。若 \(E_{\rm pk}\) 随 flux 同步变化，可能指向局地耗散强度或磁场变化；若单调变软，可能反映冷却、曲率效应或中心引擎演化。判断物理前必须确认每个 time bin 的信噪比足够，否则参数跳动可能只是拟合退化。

后续阅读路径

第一条阅读路径是从经验谱到物理谱。先熟悉 BATSE 和 Fermi/GBM 的 Band 参数统计，再读 optically thin synchrotron 的低能谱极限、快速冷却问题和边冷却边加热模型。这样可以理解为什么简单同步辐射既有吸引力又不断遇到 \(\alpha\) 难题。

第二条路径是从紧致性到喷流动力学。读 Goodman、Paczynski、Piran 的火球模型，再接 Fermi/LAT 对高能光子的 \(\Gamma_{\min}\) 约束。重点比较不同推导对 \(R\)、\(\delta t\)、高能光子同区性和靶光子谱外推的假设，而不是只记住 \(\Gamma\sim300\)。

第三条路径是从 Band 函数走向多成分 prompt 谱。现代 Fermi 数据显示某些 burst 需要热成分或额外 power-law，典型问题包括 photospheric emission、subphotospheric dissipation、magnetic reconnection 和 hadronic cascades。读这些工作时，应始终把它们放回本讲的两条基准线上：能否避免 \(\gamma\gamma\) 不透明，能否在 forward-folded 拟合中显著优于 Band 或 cutoff power law。

下一讲建议

本系列下一讲最自然的方向有三个：1. 第 2 讲讲 internal shock 与 optically thin synchrotron：从壳层碰撞半径、效率问题到 synchrotron line of death；2. 第 3 讲讲 photospheric emission 与磁化喷流：黑体展宽、亚光球耗散和 Band-like spectrum 的形成；3. 也可在第 2 讲加入 Fermi/LAT GeV 延迟高能成分，但需明确区分 prompt 同区辐射与早期外激波。
