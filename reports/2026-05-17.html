1. GRB prompt emission 经典课（二）：内激波、薄同步辐射与 synchrotron line of death

  • 课程系列：GRB prompt emission 经典课程（第 2/3 讲）
  • 本讲边界：本讲承接第 1 讲的紧致性灾难与 Band 函数，但不再重复由 \(\gamma\gamma\) 不透明度推出大 \(\Gamma\) 的基本论证，也不重复 Band 函数的经验定义。本讲只讲三个相互连接的问题：相对论喷流中为什么会自然产生半径 \(R_{\rm is}\sim 2\Gamma^{2}c\delta t\) 的内激波；内激波如何把壳层动能转化为非热电子、磁场与薄同步辐射；为什么最简单的快冷同步辐射预言 \(\alpha=-3/2\) 而许多 BATSE/Fermi 光谱给出更硬的低能谱指数，从而形成所谓 synchrotron line of death。本讲有意不系统讲光球辐射、亚光球 Compton 化、磁重联喷流与热成分展宽；这些留给第 3 讲。

经典工作详细讲解

第一，本讲的科学问题从第 1 讲自然延伸而来。上一讲已经说明，若 GRB prompt emission 在静止而紧致的源中产生，MeV 光子会通过 \(\gamma\gamma\) 相互作用造成严重不透明，因此必须有超相对论 bulk motion；上一讲也把观测谱压缩为 Band 函数的 \(\alpha\)、\(\beta\) 与 \(E_{\rm pk}\)。但这些结论还没有回答一个更物理的问题：既然喷流已经以大 \(\Gamma\) 向外运动，能量究竟在哪里、以什么方式耗散？本讲的核心就是把“透明性要求的大 \(\Gamma\)”推进到“内耗散半径、壳层碰撞、非热电子与同步辐射谱形”的完整链条。

第二，历史动机来自 prompt 光变的快速、多脉冲和高度不规则。外激波当然可以把相对论动能转化为辐射，但在半径 \(R\) 处，角时间尺度通常为 \(t_{\rm ang}\sim R/(2\Gamma^2c)\)，会把小尺度结构平滑掉。BATSE 所揭示的毫秒到秒级尖锐脉冲更像是在记录中心引擎本身的非稳态活动。Rees 与 Mészáros 的内激波图像因此成为经典：中心引擎发射一串具有不同 \(\Gamma_i\) 的相对论壳层，后发快壳追上先发慢壳，在喷流内部形成碰撞和激波，从而把中心引擎的时间结构映射为辐射脉冲。

第三，方法上，本讲把复杂喷流简化为最小可计算单元：两个冷壳层的相对论追赶与非弹性合并。若慢壳在 \(t=0\) 发出，快壳在 \(\delta t\) 后发出，碰撞前近似匀速，则追赶条件为 \(\beta_{\rm f}c(t-\delta t)=\beta_{\rm s}ct\)。在 \(\Gamma\gg1\) 下使用 \(\beta\simeq1-1/(2\Gamma^2)\)，可得 \(\(R_{\rm is}\simeq\) 2\Gamma_{
m s}^2c\delta t/(1-\Gamma_{
m s}^2/\Gamma_{
m f}^2)\(\sim\) 2\Gamma^2c\delta t)。这个公式是本讲的时空骨架：观测到的最短变光时间 \(\delta t\) 不直接等于源区大小，而是通过 \(\Gamma^2\) 放大为远离中心引擎的内耗散半径。

第四，动力学部分解决“能释放多少能量”。碰撞前总能量为 \(E_i=m_{\rm f}\Gamma_{\rm f}c^2+m_{\rm s}\Gamma_{\rm s}c^2\)，总动量为 \(P_i=m_{\rm f}\Gamma_{\rm f}\beta_{\rm f}c+m_{\rm s}\Gamma_{\rm s}\beta_{\rm s}c\)。合并后系统的 invariant mass 满足 \(M_{\rm inv}^2c^4=E_i^2-P_i^2c^2\)，共同 Lorentz factor 为 \(\Gamma_{\rm m}=E_i/(M_{\rm inv}c^2)\)。由此定义耗散效率 \(\epsilon_{\rm diss}=1-[(m_{\rm f}+m_{\rm s})\Gamma_{\rm m}]/[m_{\rm f}\Gamma_{\rm f}+m_{\rm s}\Gamma_{\rm s}]\)。这一步告诉我们，内激波不是自动高效的；若相邻壳层的 \(\Gamma\) 对比不大，可耗散能量往往有限。

第五，辐射部分把宏观耗散转为可观测谱。给定各向同性等效光度 \(L\)，碰撞区共动能量密度量级为 \(e'\simeq L/(4\pi R_{\rm is}^2\Gamma^2c)\)。若比例 \(\epsilon_B\) 的内能进入磁场，则 \(B'^2/(8\pi)=\epsilon_Be'\)；若比例 \(\epsilon_e\) 的内能进入电子，且电子注入为 \({\rm d}N_e/{\rm d}\gamma_e\propto\gamma_e^{-p}\)，则 \(\gamma_m\simeq \epsilon_e[(p-2)/(p-1)](m_p/m_e)(\Gamma_{\rm rel}-1)\)。同步特征频率为 \(\nu_m\simeq [\Gamma/(1+z)](3q_eB'/(4\pi m_ec))\gamma_m^2\)，并与观测 \(E_{\rm pk}\sim h\nu_m\) 相连。因此，内激波同步辐射确实能把 \(\delta t\)、\(L\)、\(\Gamma\)、\(\epsilon_e\)、\(\epsilon_B\) 与 MeV 谱峰放在同一套框架中。

第六，关键结果也是关键困难来自冷却。同步冷却时间为 \(t'_{\rm syn}=6\pi m_ec/(\sigma_TB'^2\gamma_e)\)，动力学时间为 \(t'_{\rm dyn}\simeq R_{\rm is}/(\Gamma c)\)，令二者相等得 \(\gamma_c=6\pi m_ec/(\sigma_TB'^2t'_{\rm dyn})\)。若 \(\gamma_c<\gamma_m\)，电子快冷，标准 optically thin synchrotron 给出中间段 \(F_\nu\propto\nu^{-1/2}\)，由于 \(F_\nu\propto\nu^{\alpha+1}\)，对应 \(\alpha=-3/2\)。若考虑单电子低频渐近，则 \(F_\nu\propto\nu^{1/3}\)，对应 \(\alpha=-2/3\)。这两条线含义不同：\(\alpha=-3/2\) 是标准快冷预言，\(\alpha=-2/3\) 是 optically thin synchrotron 的低频硬度极限。

第七，观测锚点是 BATSE 与 Fermi 的时间分辨谱。许多谱的低能 photon index 并不集中在 \(\alpha=-3/2\)，而是常见于 \(\alpha\sim -1\)，甚至有一部分达到 \(\alpha>-2/3\)。前者构成 fast-cooling problem：电子若真快冷，谱应更软；后者构成 synchrotron line of death：最简单的 optically thin synchrotron 难以产生这么硬的低能谱。高能端则可用 \(\beta=-p/2-1\) 反推电子指数 \(p\)，例如 \(\beta\sim-2.2\) 对应 \(p\sim2.4\)，这通常并非最尖锐的矛盾；真正的压力集中在低能谱斜率、冷却状态与效率之间。

第八，局限性必须与模型价值一起理解。最朴素的 baryonic internal shock plus fast-cooling synchrotron 模型同时面临低效率、低能谱过软、\(E_{\rm pk}\) 对 \(B'\)、\(\Gamma\) 与 \(\delta t\) 过度敏感，以及不同脉冲谱演化难以统一等问题。但它的经典价值并未因此消失：它提供了 prompt emission 文献中最常用的时空语言，即 \(R_{\rm is}\sim2\Gamma^2c\delta t\)、壳层碰撞、\(\epsilon_e\)、\(\epsilon_B\)、\(p\)、\(\gamma_c\) 与 Band 参数之间的接口。后来的磁场衰减、持续加热、Klein-Nishina 抑制、小尺度磁场和 jitter radiation 等修正，乃至第 3 讲要系统讨论的 photospheric emission 与 magnetic reconnection，都是在回应这套基准模型暴露出的张力。

背景知识

GRB prompt emission 的历史背景可以从一个简单事实说起：这些事件在短时间内释放巨大各向同性等效能量，并产生非热宽谱。第 1 讲已经说明，若光源静止且紧致，高能光子之间会产生严重 \(\gamma\gamma\) 吸收，只有引入大 \(\Gamma\) 才能使源区透明。同时，Band 函数把复杂谱形经验化为 \(\alpha\)、\(\beta\) 与 \(E_{\rm pk}\)，为物理模型提供了可检验接口。本讲不再重复紧致性灾难和 Band 函数定义，而是追问下一层问题：超相对论喷流中的能量为何会在 prompt 阶段转化为光子？

早期一个自然想法是外激波。若一个相对论火球撞击外部介质，它会形成 forward shock，并通过加速电子产生同步辐射；这个图像后来成为 afterglow 标准模型的基础。但 prompt emission 的快速变光对外激波提出挑战。外激波在大半径处的角时间尺度 \(t_{\rm ang}\sim R/(2\Gamma^2c)\) 会平滑来自不同角度的辐射，难以自然保留毫秒到秒级的多脉冲结构。换言之，prompt light curve 看起来不像单次外部碰撞，而像中心引擎在持续“打拍子”。

内激波模型正是在这个历史动机下兴起的。它假设中心引擎在持续时间 \(T\) 内非稳态地发射多个冷壳层，每个壳层有自己的质量 \(m_i\)、Lorentz factor \(\Gamma_i\)、宽度 \(\Delta_i\) 与发射时刻 \(t_i\)。如果后发壳层更快，就会追上先发慢壳。碰撞前，能量主要储存在有序 bulk kinetic energy 中；碰撞后，部分有序动能转化为随机内能，再分配给磁场和非热电子，最终通过辐射形成 prompt \(\gamma\)-ray。

这个模型的重要性在于它第一次把三个观测事实放到一条因果链中：中心引擎变光给出 \(\delta t\)，相对论追赶给出 \(R_{\rm is}\sim2\Gamma^2c\delta t\)，激波耗散和粒子加速给出非热宽谱。它不是只为解释某一个谱峰位置而提出的参数化模型，而是一个连接 light curve、碰撞半径、耗散效率、电子分布和谱斜率的动力学框架。因此，哪怕最简版本后来遇到困难，内激波仍是阅读 prompt emission 文献时必须掌握的共同语言。

需要避免的第一个误区是把“内激波同步辐射有严重张力”理解为“内激波概念已经完全无用”。更准确的说法是，最简单的低磁化重子喷流、壳层非弹性碰撞、均匀磁场、瞬时电子注入、快冷 optically thin synchrotron 版本与观测存在系统冲突；但多壳层内耗散作为时空骨架仍被许多模型继承。现代模型可能改变耗散机制、磁化程度、粒子加热方式或辐射过程，却常常仍需面对 \(\delta t\)、\(R\)、\(\Gamma\) 与透明性之间的同一组约束。

第二个误区是混淆 \(F_\nu\) 斜率与 photon index。观测 Band 函数的低能指数 \(\alpha\) 通常定义在 photon number spectrum 中，而同步辐射理论常先给出 \(F_\nu\) 的频率依赖。二者关系是 \(F_\nu\propto\nu^{\alpha+1}\)。因此 \(F_\nu\propto\nu^{1/3}\) 对应 \(\alpha=-2/3\)，而 \(F_\nu\propto\nu^{-1/2}\) 对应 \(\alpha=-3/2\)。若把这两个指数直接相等，会把 line-of-death 的物理含义完全读错。

第三个误区是把 \(\alpha=-2/3\) 与 \(\alpha=-3/2\) 当成两条任意经验线。\(\alpha=-2/3\) 来自单电子同步辐射在低频处的 \(F_\nu\propto\nu^{1/3}\) 渐近，是 optically thin synchrotron 在常规假设下能达到的最硬低能极限；\(\alpha=-3/2\) 则来自标准快冷电子分布在 \(\gamma_c<\gamma_e<\gamma_m\) 的 \(N(\gamma_e)\propto\gamma_e^{-2}\)。前者挑战辐射机制的低频极限，后者挑战冷却图像。读论文时必须先问作者讨论的是哪一条线。

第四个背景点是 prompt 与 afterglow 的相似和不同。两者都常使用 \(\epsilon_e\)、\(\epsilon_B\)、\(p\)、同步冷却频率和特征频率这些语言；但 afterglow 外激波的动力学由外介质密度和 Blandford-McKee 演化控制，而 prompt 内激波的动力学由中心引擎壳层序列、\(\delta t\)、\(\Gamma_i\) 分布和内部碰撞控制。不能把 afterglow 的闭合关系直接搬到 prompt 内激波中，也不能因为两个领域都使用同步辐射就认为它们的参数约束等价。

最后，本讲还要明确与第 3 讲的边界。本讲只把内激波、薄同步辐射、冷却谱形和 line of death 作为主线；photospheric emission、sub-photospheric Comptonization、magnetic reconnection、ICMART、prompt 偏振和 GeV 延迟诊断只作为模型局限和后续入口出现。第 3 讲才会系统讨论光球和磁化喷流如何回应内激波同步辐射的效率与谱斜率张力。

基础理论 / 方法脉络

第一，内激波图像的起点是非稳态相对论喷流。中心引擎在持续时间 \(T\) 内发射多个壳层，而不是一股完全平滑、单一 \(\Gamma\) 的风。每个壳层可用 \(m_i\)、\(\Gamma_i\)、\(\Delta_i\) 与 \(t_i\) 描述。若后发快壳满足 \(\Gamma_{\rm f}>\Gamma_{\rm s}\)，它会在向外运动中追上先发慢壳。这个图像的关键不是碰撞本身多复杂，而是它把中心引擎发射间隔 \(\delta t\) 转化为喷流内部的耗散半径。

第四，碰撞的流体结构通常包含 forward shock、reverse shock 与接触间断。forward shock 加热慢壳物质，reverse shock 加热快壳物质，接触间断两侧压强近似匹配。由于相邻壳层的相对 Lorentz factor 往往只是 mildly relativistic，内激波虽然发生在整体 \(\Gamma\gg1\) 的喷流中，但局地激波强度并不一定极端。这一点直接影响耗散效率和电子最小 Lorentz factor \(\gamma_m\)。

公式与推导

第一步：从两壳层追赶条件开始。假设上一讲已经给出喷流必须有大 \(\Gamma\)，本讲不再推导紧致性下限，而是问能量在何处耗散。令慢壳在 lab frame 的 \(t=0\) 发出，Lorentz factor 为 \(\Gamma_{\rm s}\)；快壳在 \(\delta t\) 后发出，Lorentz factor 为 \(\Gamma_{\rm f}>\Gamma_{\rm s}\)。碰撞前忽略壳层宽度和加速度，则要求两壳层到达同一半径：

\[\beta_{\rm f}c(t-\delta t)=\beta_{\rm s}ct .\]

这里要求解的是碰撞时间 \(t\)，它把中心引擎的发射间隔 \(\delta t\) 转换为喷流内部的耗散位置；下一步用超相对论速度展开得到可观测的半径标度。

第二步：推出内激波半径。对 \(\Gamma\gg 1\) 使用 \(\beta\simeq 1-1/(2\Gamma^{2})\)，并取碰撞半径为慢壳位置 \(R_{\rm is}\simeq \beta_{\rm s}ct\)，可得

\[R_{\rm is}\simeq \frac{2\Gamma_{\rm s}^{2}c\delta t}{1-\Gamma_{\rm s}^{2}/\Gamma_{\rm f}^{2}}\sim 2\Gamma^{2}c\delta t .\]

这个结果说明，观测到的最短变光时间 \(\delta t\) 并不直接等于源区半径，而是经由 \(\Gamma^{2}\) 放大到远离中心引擎的内耗散半径；下一步要检查该半径是否位于可见非热辐射窗口。

第三步：加入透明性和动力学窗口。最简薄同步辐射内激波要求碰撞发生在光球之外，同时早于外激波显著减速，因此需要

\[R_{\rm ph}

若 \(R_{\rm is}R_{\rm dec}\)，prompt 内耗散会与早期 afterglow 动力学混合。通过这一步，\(\delta t\)、\(\Gamma\)、\(L\) 与可见非热辐射的允许区联系起来；下一步计算碰撞能释放多少内能。

第四步：写出碰撞前四动量。假设两个壳层在碰撞前为冷壳层，静质量分别为 \(m_{\rm f}\) 与 \(m_{\rm s}\)，内能可忽略，则初始总能量和总动量为

\[E_{\rm i}=m_{\rm f}\Gamma_{\rm f}c^{2}+m_{\rm s}\Gamma_{\rm s}c^{2},\qquad P_{\rm i}=m_{\rm f}\Gamma_{\rm f}\beta_{\rm f}c+m_{\rm s}\Gamma_{\rm s}\beta_{\rm s}c .\]

这一步明确可耗散能量来自有序 bulk kinetic energy 的差异，而不是凭空产生的热能；下一步用不变量求合并后系统的总质量和共同 Lorentz factor。

第五步：由四动量不变量求合并后状态。把碰撞近似为完全非弹性合并，合并后系统的 invariant mass \(M_{\rm inv}\) 包含静质量和随机内能，满足

\[M_{\rm inv}^{2}c^{4}=E_{\rm i}^{2}-P_{\rm i}^{2}c^{2},\qquad \Gamma_{\rm m}=\frac{E_{\rm i}}{M_{\rm inv}c^{2}} .\]

这里求出的 \(\Gamma_{\rm m}\) 是合并后整体 bulk motion 的 Lorentz factor；\(M_{\rm inv}\) 相比 \(m_{\rm f}+m_{\rm s}\) 的增加就是内能的体现。下一步用它定义耗散效率。

第六步：定义内激波耗散效率。辐射前可用内能等于初始 bulk energy 与合并后仍保留的 bulk energy 之差，因此量级效率为

\[\epsilon_{\rm diss}=1-\frac{(m_{\rm f}+m_{\rm s})\Gamma_{\rm m}}{m_{\rm f}\Gamma_{\rm f}+m_{\rm s}\Gamma_{\rm s}} .\]

若 \(\Gamma_{\rm f}/\Gamma_{\rm s}\) 不大，\(\epsilon_{\rm diss}\) 通常偏低；若速度对比很大，又会给光深、峰值能量和中心引擎壳层分布带来压力。下一步把宏观耗散转成局地共动能量密度。

第七步：估计碰撞区共动能量密度。用各向同性等效光度 \(L\) 描述喷流，在 \(R_{\rm is}\) 处近似球对称等效流动，则共动能量密度为

\[e'\simeq \frac{L}{4\pi R_{\rm is}^{2}\Gamma^{2}c} .\]

这里的 \(e'\) 是磁场放大和非热电子加速的能量库。因为 \(R_{\rm is}\sim 2\Gamma^{2}c\delta t\)，所以 \(e'\) 对 \(\Gamma\) 与 \(\delta t\) 极敏感；下一步用 \(\epsilon_B\) 求磁场。

第八步：由 \(\epsilon_B\) 得到共动磁场。假设耗散内能中比例 \(\epsilon_B\) 进入等效共动磁场，则

\[\frac{B'^{2}}{8\pi}=\epsilon_B e',\qquad B'\simeq \left(\frac{2\epsilon_B L}{R_{\rm is}^{2}\Gamma^{2}c}\right)^{1/2} .\]

这个式子说明，\(B'\) 不只是微物理参数，也受碰撞半径控制；因此 \(B'\) 会把 light curve 的 \(\delta t\) 与谱峰 \(E_{\rm pk}\) 联系起来。下一步求电子注入 Lorentz factor。

第九步：由 \(\epsilon_e\) 和激波相对运动估计电子最小 Lorentz factor。若电子获得比例 \(\epsilon_e\) 的耗散能量，并注入为 \({\rm d}N_e/{\rm d}\gamma_e\propto \gamma_e^{-p}\) 的幂律分布，从 \(\gamma_m\) 起始，则

\[\gamma_m\simeq \epsilon_e\frac{p-2}{p-1}\frac{m_p}{m_e}(\Gamma_{\rm rel}-1) .\]

这里 \(\Gamma_{\rm rel}\) 是内激波两侧流体的相对 Lorentz factor。由于相邻壳层碰撞常只是 mildly relativistic，\(\gamma_m\) 会受到速度对比和 \(\epsilon_e\) 的强限制；下一步把 \(B'\) 与 \(\gamma_m\) 转换为同步峰值频率。

第十步：连接同步特征频率与观测谱峰。假设辐射区 optically thin，电子在 \(B'\) 中同步辐射，观测频率受到 bulk boosting 和红移修正，则

\[\nu_m\simeq \frac{\Gamma}{1+z}\frac{3q_eB'}{4\pi m_ec}\gamma_m^{2},\qquad E_{\rm pk}\sim h\nu_m .\]

这一步给出 \(E_{\rm pk}\) 与 \(L\)、\(\Gamma\)、\(\delta t\)、\(\epsilon_B\)、\(\epsilon_e\) 和 \(p\) 的接口。能调到 MeV 只是必要条件；模型还必须解释低能斜率、效率和时间演化。下一步判断电子是否快冷。

第十一步：定义同步冷却 Lorentz factor。暂时忽略逆 Compton、Klein-Nishina 抑制、持续加热和磁场衰减，只考虑同步冷却，则

\[t'_{\rm syn}(\gamma_e)=\frac{6\pi m_ec}{\sigma_TB'^{2}\gamma_e},\qquad t'_{\rm dyn}\simeq \frac{R_{\rm is}}{\Gamma c},\qquad \gamma_c=\frac{6\pi m_ec}{\sigma_TB'^{2}t'_{\rm dyn}} .\]

若 \(\gamma_c<\gamma_m\)，电子在动力学时间内显著冷却，为快冷；若 \(\gamma_c>\gamma_m\)，则为慢冷。这个判据决定低能谱形，并直接连接到 Band 低能 photon index \(\alpha\)。

第十二步：给出快冷同步分段谱与两条低能判据。在均匀磁场、各向同性 pitch angle、瞬时注入、optically thin 且无自吸收主导的标准快冷模型中，谱形为

\[F_\nu\propto \begin{cases}\nu^{1/3},& \nu<\nu_c<\nu_m,\\ \nu^{-1/2},& \nu_c<\nu<\nu_m,\\ \nu^{-p/2},& \nu>\nu_m .\end{cases}\]

观测 Band 低能指数定义在 photon number spectrum 中，因此要用 \(F_\nu\propto \nu^{\alpha+1}\) 转换。于是 \(F_\nu\propto \nu^{1/3}\) 对应 \(\alpha=-2/3\)，这是 optically thin synchrotron 的单电子低频硬度极限；\(F_\nu\propto \nu^{-1/2}\) 对应 \(\alpha=-3/2\)，这是标准快冷中间段预言。下一步用高能端检查电子指数。

第十三步：用高能 photon index 反推电子幂律指数。快冷高频段由冷却后的幂律电子控制，Band 高能 photon index \(\beta\) 与电子指数 \(p\) 的近似关系为

\[\beta=-\frac{p}{2}-1 .\]

例如 \(\beta\sim -2.2\) 对应 \(p\sim 2.4\)，通常与 relativistic shock acceleration 的常见范围并不严重冲突。因此许多 prompt 光谱对最简模型的尖锐压力集中在低能 \(\alpha\) 和冷却状态，而不是高能 \(\beta\)。最后一步把谱斜率张力与总效率张力合并。

第十四步：形成观测闭合诊断。prompt \(\gamma\)-ray 辐射效率可分解为动力学耗散、电子能量份额和电子实际辐射效率：

\[\epsilon_\gamma\simeq \epsilon_{\rm diss}\epsilon_e\eta_{\rm rad},\qquad \eta_{\rm rad}\simeq \begin{cases}1,& \gamma_c<\gamma_m,\\ (\gamma_m/\gamma_c)^{p-2},& \gamma_c>\gamma_m .\end{cases}\]

这给出最简内激波同步辐射的核心诊断：快冷可使 \(\eta_{\rm rad}\simeq 1\)，但标准预言 \(\alpha=-3/2\) 往往过软；慢冷或边缘冷却可使谱更硬，却常降低 \(\eta_{\rm rad}\)。因此读时间分辨谱时应联合检查 \(\alpha(t)\)、\(\beta(t)\)、\(E_{\rm pk}(t)\)、flux、\(\delta t\)、\(\Gamma\) 下限以及 \(R_{\rm ph}

经典拟合 / 应用方法

第一，拟合训练要从经验谱开始，但不能停在经验谱。对每一个时间分辨区间，先拟合 Band 函数、cutoff power law 或 smoothly broken power law，得到归一化、低能 photon index \(\alpha\)、高能 photon index \(\beta\) 与峰值能量 \(E_{\rm pk}\)。这里的课程重点是：这些参数不是物理模型本身，而是观测数据与内激波同步辐射之间的接口。时间分辨谱优先于时间积分谱，因为时间积分会混合多个脉冲、多个 \(E_{\rm pk}(t)\) 和不同冷却状态，从而把真实谱演化误读为单一宽谱曲率。读论文时首先检查作者如何选取 time bin：是固定信噪比、固定时间宽度，还是按脉冲结构切分；不同切分会系统影响 \(\alpha\) 与 \(E_{\rm pk}\)。

第二，物理拟合要明确参数层级。最小的一区薄同步辐射模型通常需要 \(\Gamma\)、\(L\)、\(\delta t\)、\(\epsilon_e\)、\(\epsilon_B\)、\(p\) 与 \(z\)，并通过 \(R_{\rm is}\sim 2\Gamma^{2}c\delta t\)、\(e'\simeq L/(4\pi R_{\rm is}^{2}\Gamma^{2}c)\)、\(B'^{2}/(8\pi)=\epsilon_Be'\)、\(\nu_m\simeq \Gamma(3q_eB'/(4\pi m_ec))\gamma_m^{2}/(1+z)\) 连接到 \(E_{\rm pk}\)。扩展模型还可加入壳层 Lorentz factor 分布、电子注入持续时间、磁场衰减指数、同步自吸收、synchrotron self-Compton 或 Klein-Nishina 修正。课程中要强调：加入额外自由度可以改善拟合，但必须说明新增参数解决的是低能谱斜率、峰值位置、效率，还是残差结构，不能只把复杂性当作成功。

第三，严格比较模型和数据应在 count space 中完成。给定 photon model \(M(E;\theta)\)，必须通过探测器响应矩阵折叠，得到第 \(i\) 个能道的预测计数

\[\lambda_i(\theta)=\int R_i(E)M(E;\theta){\rm d}E+b_i,\]

其中 \(R_i(E)\) 是响应，\(b_i\) 是背景贡献。若观测计数为 \(n_i\)，Poisson likelihood 可写为

\[{\cal L}(D|\theta)=\prod_i \frac{e^{-\lambda_i(\theta)}\lambda_i(\theta)^{n_i}}{n_i!} .\]

等价地，常用 Cash statistic 进行最大似然拟合。不要把 unfolded spectrum 当作独立高斯数据点直接拟合，因为 unfolded 点之间包含响应矩阵引入的相关性，会夸大模型的视觉匹配质量。

第四，贝叶斯分析的后验写作

\[p(\theta|D)\propto {\cal L}(D|\theta)p(\theta) .\]

先验不是装饰，而是物理可行性的一部分。应要求 \(0<\epsilon_e<1\)、\(0<\epsilon_B<1\)，\(p\) 落在相对论激波加速可接受范围附近，\(\Gamma\) 满足第 1 讲由 \(\gamma\gamma\) 透明性得到的下限，并且 \(R_{\rm is}\) 满足 \(R_{\rm ph}

第五，后验解释要看边缘分布、参数相关和可信区间，而不仅是 best fit。常见退化包括 \(\alpha\) 与 \(E_{\rm pk}\) 的相关、\(\Gamma\)、\(\delta t\) 与 \(B'\) 的退化、\(\epsilon_B\) 与 \(E_{\rm pk}\) 的退化，以及 \(p\) 与 \(\beta\) 之间由 \(\beta=-p/2-1\) 给出的联系。实际读论文时要检查作者是否报告了二维置信区域或 corner plot；若只给出单个最佳拟合值和形式误差，通常不足以判断内激波同步辐射是否真的通过了检验。

第六，残差诊断是本讲拟合部分的核心训练。低能端连续残差直接关系到 synchrotron line of death：如果物理同步模型在低能端持续低估 counts，说明数据要求比 \(\alpha=-3/2\) 甚至 \(\alpha=-2/3\) 更硬的谱。高能端正残差可能提示额外 power-law、Fermi/LAT 成分、synchrotron self-Compton 或早期外激波。谱峰附近交替残差则可能意味着单一平滑 break 不足、背景估计偏差、响应矩阵问题或真实谱曲率不同于 Band 函数。课程中应让学生先看残差，再读作者对 \(\chi^2\)、Cash statistic 或 Bayesian evidence 的总结。

第七，系统误差必须与物理失败分开讨论。背景区间选择、低能阈值、能段覆盖、dead time、pile-up、探测器响应、跨探测器交叉归一化和时间 bin 混合都会影响 \(\alpha\) 与 \(E_{\rm pk}\)。尤其当 \(E_{\rm pk}\) 靠近仪器能段边界时，\(\alpha\) 可能被强烈牵引；当 burst 很亮时，响应非线性和 dead time 也会改变谱形。读 BATSE 或 Fermi/GBM 论文时，要检查作者是否改变背景模型、能段范围和探测器组合来验证 \(\alpha\) 分布的稳健性。

第八，最终应用可组织成三层判据。第一层看经验拟合是否给出 \(\alpha>-2/3\)；若是，最简 optically thin synchrotron 的单电子低频极限受到直接挑战。第二层看 \(-3/2<\alpha<-2/3\) 的谱是否可由边缘快冷、慢冷、磁场衰减、持续加热或时间积分效应解释。第三层联合 \(E_{\rm pk}\)、flux、\(\delta t\)、\(\Gamma\)、\(R_{\rm is}\) 与效率 \(\epsilon_\gamma\simeq \epsilon_{\rm diss}\epsilon_e\eta_{\rm rad}\)，判断所需参数是否自然。只有同时通过谱斜率、残差、半径窗口和效率约束，内激波同步辐射才算在物理层面被支持。

重点章节 / 结果段落怎么读

读 Rees 与 Mészáros 内激波经典图像时，第一段应先读动机，而不是先读公式。作者试图解决的是 prompt emission 的快速、多脉冲、不规则光变如何在大 \(\Gamma\) 喷流中保留下来。上一讲已经说明大 \(\Gamma\) 是透明性要求，本讲要看经典论文如何进一步提出“非稳态中心引擎发射不同 \(\Gamma_i\) 壳层，快壳追慢壳并在内部耗散”。读这一部分时要标出壳层发射间隔、壳层宽度、Lorentz factor 分布和观测脉冲之间的对应关系。

第二个重点段落是碰撞半径推导。论文中若出现 \(R_{\rm is}\sim 2\Gamma^{2}c\delta t\)，要回到两壳层 worldline：慢壳先发，快壳延迟 \(\delta t\) 后发，追赶条件来自 \(\beta_{\rm f}c(t-\delta t)=\beta_{\rm s}ct\)。读者要问三个问题：这里的 \(\delta t\) 是中心引擎发射间隔、观测最短变光，还是脉冲宽度；这里的 \(\Gamma\) 是慢壳、快壳还是合并后流体的代表值；公式是否默认 \(\Gamma_{\rm f}\) 明显大于 \(\Gamma_{\rm s}\)。这些细节会影响 \(R_{\rm ph}

第三个重点段落是耗散效率。两壳层完全非弹性合并模型通常用初始能量 \(E_i=m_{\rm f}\Gamma_{\rm f}c^2+m_{\rm s}\Gamma_{\rm s}c^2\)、初始动量 \(P_i=m_{\rm f}\Gamma_{\rm f}\beta_{\rm f}c+m_{\rm s}\Gamma_{\rm s}\beta_{\rm s}c\) 和不变量 \(M_{\rm inv}^{2}c^{4}=E_i^{2}-P_i^{2}c^{2}\) 求 \(\Gamma_{\rm m}\)。读到效率 \(\epsilon_{\rm diss}\) 时，不要只看数值大小，还要确认作者采用等质量壳层还是等能量壳层。等质量假设通常使大 \(\Gamma_{\rm f}/\Gamma_{\rm s}\) 更容易提高效率，等能量假设则会改变可耗散能量分布。

第四个重点段落是辐射微物理。经典内激波论文往往引入 \(\epsilon_e\)、\(\epsilon_B\) 与 \(p\)，把耗散内能分给电子和磁场，再用同步辐射解释 \(E_{\rm pk}\)。读这一部分时要检查作者是否假设电子瞬时注入、磁场恒定、pitch angle 各向同性、optically thin、无持续加热、无强逆 Compton 冷却。若这些假设成立，标准快冷谱预言 \(F_\nu\propto \nu^{-1/2}\)，对应 \(\alpha=-3/2\)；单电子低频极限 \(F_\nu\propto \nu^{1/3}\)，对应 \(\alpha=-2/3\)。这两条线的物理来源必须分开读。

第五个重点段落是与 BATSE 或 Fermi 光谱的比较。Preece 等关于低能谱指数分布的工作要按样本、时间分辨方式、能段范围和误差处理来读。若论文说某些谱违反 synchrotron line of death，应确认它指的是 \(\alpha>-2/3\)，而不是仅仅比 \(\alpha=-3/2\) 更硬。若论文讨论 fast-cooling problem，则重点是大量谱落在 \(-3/2<\alpha<-2/3\) 或接近 \(\alpha\sim -1\)，说明标准快冷同步辐射过软，但未必直接越过单电子同步硬度极限。

第六个重点段落是讨论与局限。优秀论文会把问题拆成 low efficiency problem、fast-cooling problem、line-of-death problem 和 \(E_{\rm pk}\) 调参敏感性，而不是笼统说“同步辐射失败”。课程阅读时可以把每篇论文整理成四列表格：核心假设、主要预言、观测冲突、作者提出的修正。这样才能看出本讲边界：本讲只系统处理内激波与薄同步辐射，第 3 讲才展开 photospheric emission、sub-photospheric Comptonization 和 magnetic reconnection。

  7. count-space 残差图：横轴为能道或能量，纵轴为标准化残差。要比较 Band、cutoff power law 与物理 synchrotron 模型折叠探测器响应后的残差。低能系统残差对应 line-of-death 张力，高能残差可能提示额外成分，谱峰附近交替残差可能来自响应、背景或谱曲率假设。

后续阅读路径

第一条路径从内激波动力学走向 prompt 光变模拟。先读 Rees 与 Mészáros 关于相对论壳层内部碰撞的经典思想，再读多壳层模型如何用随机 \(\Gamma_i\)、壳层质量 \(m_i\)、发射间隔 \(t_i\) 生成复杂 light curve。阅读重点不是记住某个模拟光变，而是理解 \(\delta t\)、\(R_{\rm is}\)、\(\epsilon_{\rm diss}\) 与脉冲宽度之间的因果链。沿这条线继续读，可以进入壳层分布、中心引擎活动和 prompt variability 统计的现代研究。

第二条路径从同步辐射谱形走向 synchrotron line of death。读 Katz、Tavani、Sari、Piran、Narayan 和 Preece 等相关工作时，要始终区分 \(\alpha=-2/3\) 与 \(\alpha=-3/2\)：前者是 optically thin synchrotron 的单电子低频硬度极限，后者是标准快冷电子分布给出的中间段预言。随后可读关于边缘快冷、磁场衰减、持续电子加热、Klein-Nishina 抑制和时间积分效应的论文，理解为什么现代问题不再是简单问“同步辐射是否死亡”，而是问哪些同步辐射变体能同时通过谱斜率、效率和时间演化检验。

第三条路径从经验 Band 拟合走向物理谱拟合。现代研究越来越强调在 count space 中折叠响应矩阵、使用 Poisson likelihood 或 Cash statistic，并用后验分布评估 \(\Gamma\)、\(L\)、\(\epsilon_e\)、\(\epsilon_B\)、\(p\) 与 \(\delta t\) 的自然性。沿这条路径读文献时，应优先关注残差图、posterior predictive check、模型比较和系统误差，而不是只比较 \(\alpha\)、\(eta\) 与 \(E_{\rm pk}\) 的表格。

第四条路径自然通向本系列第 3 讲。当最简内激波同步辐射同时面对低效率、快冷谱过软和部分谱超过 \(\alpha=-2/3\) 的压力时，后续文献转向 photospheric emission、sub-photospheric Comptonization 与 magnetic reconnection。读这些模型时，应把本讲建立的诊断量作为对照：它们如何解释 \(E_{\rm pk}\)、低能谱指数、热成分缺失或存在、偏振、GeV 延迟和高能截断。这样才能看出第 3 讲不是替换名词，而是在回应本讲暴露的具体张力。

下一讲建议

第 3 讲最自然的方向是“光球辐射、亚光球耗散与磁化喷流”。本讲已经把最简内激波同步辐射的时空骨架、效率问题和低能谱张力讲清楚；下一讲应从 \(R_{\rm ph}\) 出发，说明为什么光球附近的热辐射会被展宽，sub-photospheric Comptonization 如何产生 Band-like spectrum，以及这些模型如何回应 \(\alpha>-2/3\) 与 \(\alpha\sim -1\) 的观测事实。

第二个可衔接方向是 magnetic reconnection 与高磁化 prompt emission。它与本讲的边界在于：本讲的耗散来自低磁化重子壳层的相对运动和内激波，下一讲则可讨论 Poynting-flux dominated jet 中磁能如何通过 reconnection 转化为粒子能量与辐射。比较重点应放在效率 \(\epsilon_\gamma\)、谱峰 \(E_{\rm pk}\)、低能谱指数 \(\alpha\)、偏振和 GeV 延迟，而不是重复两壳层碰撞效率推导。

第三个可作为第 3 讲观测诊断模块的方向是 prompt 偏振、高能延迟与额外高能成分。它们可以帮助区分 optically thin synchrotron、photospheric emission、magnetic reconnection 与早期外激波贡献。但这些内容应作为光球和磁化喷流模型的诊断工具出现，不应把本讲已经完成的 \(R_{\rm is}\sim 2\Gamma^{2}c\delta t\)、快冷谱和 synchrotron line of death 主线重新展开。
